    Page Component Visibility

    Toggle visibility of default interface components: Adjust layout settings to suit your workspace needs.



    1. Remove Extra Padding: Removes outer spacing around embedded components.


    2. Hide Section Titles: Suppress default section headers for cleaner presentation.


    3. Hide Side Navigation: Removes side panel navigation from workspace pages.


    4. Hide Top Toolbar: Conceals command bar for a distraction-free interface.


    5. Hide Page Header: Allows removal of the page header to simplify layout.


    6. Hide Comments Section: Disable comments display area on workspace pages.


    7. Hide Search Bar: Remove built-in search functionality from the interface.

Default Currency

By default, the system currency is set to USD. Administrators can update this setting by selecting a different currency from the dropdown menu and clicking Save. Once saved, the selected currency will be applied as the new system default.



Once configured, the selected currency will automatically appear in the following places:

  • Requisition: When adding a requisition, the chosen currency will be visible for the Minimum Annual Budget and Maximum Annual Budget fields. Users can also change the currency manually if needed.


  • Job Posting: The selected currency will appear when posting a new job.


  • Candidate Management: While adding a candidate to a job, the default currency will be displayed.


Currency Separator

Currency Separator Options: Administrators can choose how numbers are displayed system-wide by selecting one of the following options:




  1. International: Uses a comma to separate every three digits from the right. For example, 1,000,000 for one million.




  2. India: Follows the Indian numbering system, where the first three digits are grouped, and subsequent digits are grouped in twos. For example, 10,00,000 for ten lakh.




  3. No Separator: Displays numbers without any thousand separators. For example, 1000000 for one million.
