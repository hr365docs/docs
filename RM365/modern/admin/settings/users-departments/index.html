Users and Departments


You can add departments and users under this setting.



    User

    Click the +Add Button to add a new user.



    To add a user, fill out the form and click the "Add" button to save it.



    This button allows the user to upgrade from M365.



    If an admin wants to mark users as inactive within the application, they can select multiple users. The admin will see a 'Bulk Inactive' option button available, which they can click to mark the selected users as inactive.



    After that, the users who were marked as inactive will be displayed under the Inactive tab.



    If the admin wants to activate a user in the application, they can check the checkbox next to the user and mark them as active.



    Then, the user that the admin marked as active will reappear in the Active section.



    When the admin is adding a user to the application, they will have the option to select Active or Inactive. If the admin selects Active, the user will remain active. If the admin selects Inactive, the user will be displayed in the Inactive section.



    Departments

    Click the +Add Button to add Department.



    After entering the department name and remarks, click the "Save" button



    If the user selects multiple departments, an action button labeled 'Set Bulk Inactive' becomes available.



    In the Inactive section, the departments that the user selected and marked as inactive will be displayed.



    If a department is marked inactive by mistake, the user can check the checkbox and mark it as active again.



    The department that the user checked will be moved back to the Active section and displayed there.



    When a user or admin adds a department, they see an Active/Inactive toggle. By default, the toggle is set to 'On.' If it is turned 'Off' while adding the department, that department will be added to the Inactive section."
