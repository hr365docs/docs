        • The Performance Improvement Plan (PIP) feature in the system enables managers to create, track, and complete structured performance plans for team members. This ensures accountability, measurable outcomes, and timely progress monitoring.

        In-Progress

        Managers can initiate a new PIP and configure key details while the plan is active.

        1. Click on the + Add button to start creating a new PIP.
        2. Select the team member to be placed under the PIP, specify the reason, and define the start and end dates. Add planned activities to guide the improvement process.
        3. Assign specific tasks with start and end dates to measure performance improvements.
        4. Use the + icon to add additional tasks as needed.
        5. Once all details are entered, click the Save button to finalize the PIP.

          6. Once a PIP is created, the manager can access the 3-dot menu for additional actions. From here, the manager can:

          • Edit
          • Make decision
          • View tasks

        Completed

        After the PIP duration ends, managers can review the progress and outcomes. A PIP marked as Completed provides a summary of tasks achieved, areas improved, and overall performance results.


      PIP

      If the manager places the user under a Performance Improvement Plan, the user will be notified, and the PIP details will appear here.
