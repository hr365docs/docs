Contracts

A contract is an agreement between two parties that creates an obligation to perform (or not perform) a particular duty.

New Contract Type: Click on the new contract type tab. A panel will open named Add Contract Type. Fill out all the details then click on Save button.


Once the Save button is clicked a new page will open. From here click on the field names which you want to add in the document and now click on the Save button.


A new contract type will be added as shown below. You can edit or delete the contract type.


Click on the gear icon to configure conditional field.


Click on the Condition tab in the Configure Conditional Field panel. A panel will open named Add New Condition.


Select option from dropdown and set the condition according to your requirement. Now click on the Save button
