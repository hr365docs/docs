    General Settings


      This section allowing administrators to configure various general preferences such as date formats, themes, languages Currency etc.


      1. Theme: In this section, you can switch the theme to dark, light, or a Site theme color.


      2. Date Format: In the Date Format section, you can customize the date format to options such as MM-DD-YY, YY-MM-DD, or any other format you prefer.


      3. Currency: In this section, you can configure your organizationâ€™s currency preferences. This includes setting a default currency, allowing selection of available currencies, and defining how numbers should be displayed with separators.


        1. Default Currency: Set the primary currency for your application (e.g., USD, EUR, INR). All new expenses will use this currency by default.


        2. Currency Separator: Define how large numbers are formatted in reports and expense forms. Options include:
          International: 1,000.00
          India: 1,00,000.00
          No Separator: 100000.00


        3. Currency Selection: From here, you can choose which currencies should be available in the dropdown for users when submitting expenses. For example, you may enable USD, EUR, and INR while hiding others.


      4. Language: With this feature, you can change the language selection if the browser's default language is not detected. You can select the desired language from a dropdown menu, and that language will be applied to your application. For example, if you select French, the application language will change to French.


      5. Display Project Name: To enable the project name, simply turn on the toggle. The project will then be displayed in both reports and the Administration section.



      6. Display Location: Toggle on to display location details in expense reports.



      7. Expense Tracking Number: In the "Expense Tracking ID" field, you can enter the expense ID, such as request or ET, which will be added as a prefix.



      8. Reimbursable: Enable or disable tracking of expenses that are reimbursable. When enabled, the Reimbursable option will appear in the Expense Form.



      9. Reimbursed Badge: When the Reimbursable option is enabled, this toggle controls whether a Reimbursed badge is shown in Expense Details.


      10. Sample Dashboard Data: Load sample data to view dashboard in demo mode.



      11. Sample Expense Data: Populate the application with sample expense data for testing/demo.



      12. Paid Expenses : When enabled, the Paid Pivot will be displayed on the home page, and the Paid icon will appear in Expense Details.




      13. Expense Attachments: Enable this toggle to make attachments mandatory while submitting an expense, time, and distance.



      14. Expense Type Code: Enable this to addcode for your expense types.



      15. Multiple Cost Center: When enabled, a separate cost center entry will be created for each row in the Multiple Expense.


      16. Draft Expense ID: When enabled, an expense ID will be generated for draft expenses as well.



      17. Self-Declaration:When enabled, a checkbox will appear while creating expenses, asking if you agree to submit the expenses with or without uploading a receipt.



      18. Expense Time: When enabled, you can specify the start and end time while creating an expense.



      19. Clone Multiple Expense: When enabled, you can clone expense rows in multiple expense.

          Brand logo

          Here you can add your company logo, which will be displayed at the top of the navigation bar. From here, the admin can add or update the company logo. The updated logo will appear at the top of the navigation bar. The recommended size for the logo is 150 X 60, with the same aspect ratio, and a transparent or white background is recommended for better visibility and user experience.


          Click on the 'Upload Logo' button, then select the logo file. After selecting, the user will have the option to crop the logo, allowing them to adjust the image as needed.


          After cropping, the user can click on the 'Save Logo' button. Once saved, the logo will be visible on the top bar.



          If the user wants to remove the logo they added and upload a new one, they can click the 'Reset Logo' button.
