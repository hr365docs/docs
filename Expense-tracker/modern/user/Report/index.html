Expense Tracker 365
  • User Guide

    Report

    The User Report page provides users with detailed access to expense submissions. Regular users can view only their own reports, while administrators have complete visibility across all departments. Each section displays relevant expense details with options to view records by status such as All, Pending, Approved, and Rejected.


    Expenses

    The Expenses tab displays all expense records, including pending, approved, and rejected reports. Users can review key information such as Expense ID, Title, Expense Type, Dates, Requested Amount, and Approval Status.


    Expense Type

    The Expense Type tab categorizes expenses based on their nature, such as travel, meals, or office-related costs. It includes views for All and Monthly expense reports for better tracking and analysis.


    Cost Center

    The Cost Center tab organizes expenses by specific cost centers or budget allocations. Users can switch between All and Monthly views to monitor financial activity within each cost center.


    Department

    The Department tab displays expenses grouped by departments, providing insights into departmental spending patterns. Both All and Monthly reports can be accessed for detailed review.


    Manager

    The Manager tab lists expense reports submitted under each manager. It helps track approval responsibilities and view spending reports in All or Monthly formats.


    Employee

    The Employee tab allows viewing expense records submitted by individual employees. It supports both All and Monthly views, helping to analyze expense patterns for each employee.
