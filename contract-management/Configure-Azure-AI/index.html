• Configure Azure AI

    1. Go to portal.azure.com.


    2. Access Azure's OpenAI service.


    3. Click on Create.


    4. Fill out the required configuration fields and click Next.


    5. On the Review + Submit page, click Create.


    6. Open Azure AI Foundary Portal.


    7. Select Create New Deployment.



    8. Choose the model, version, deployment type, and enter a name. Copy the deployment name.


    9. Go to Resource Management and select Keys and Endpoints.


    10. Copy the key and endpoint details. These values will be required inside your Contract Management application for Azure AI integration.
