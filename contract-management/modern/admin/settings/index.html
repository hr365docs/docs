    Settings

  • Settings

      Setting options, as mentioned below, are available for Admin, HR Manager Admin, and HR Head admin. All the general settings like adding users to the application, notification settings and features setup or setup the appraisal cycle, etc. could be accomplished via setting.

      Under the settings tab, subsets are provided for general, users, notifications, and feature settings.



Users, Roles and Permissions


    From here, admin can assign roles to users and teams.

  • Add/Edit Users: API configuration help in working delta sync, excluding unlicensed and shared mailbox users on theuser's page Click on the link


  • Advance Permissions: Admin could select the desired date format (Reflects the entire application like date columns etc.)


Integrations




  • Phone: API configuration help in working delta sync, excluding unlicensed and shared mailbox users on theuser's page Click on the link


  • CLM365 Mailbox: Admin could select the desired date format (Reflects the entire application like date columns etc.)

Contract Fields


    Notification settings or email templates users could change the body of the email (changing the text in the email body or adding placeholders).

    The notification has three categories: Self Review, appraisers review, and HR review.


  • Add Custom Fields: API configuration help in working delta sync, excluding unlicensed and shared mailbox users on theuser's page Click on the link


Features

    Contract Management 365 features could be customized.

  • Remove Custom Webpart CSS: If it is enabled, the space outside of webpart would be removed.


  • Hide webPart title in sharePoint page : Hide webpart title in the SharePoint page where app is running.


  • Hide Side Navigation Panel In SharePoint Page: Hide side navigation panel in SharePoint page where app is running.


General Settings

    From here you can update date format, logo, theme etc.

  • Date Format: From here admin can change the date format. The date format settings apply everywhere in the application.


  • Side Navigation Theme : From here, the admin can change the theme of the helpdesk application.


  • Brand Logo : From here admin can add or update the company logo. The updated logo is displayed at the top of the left-side navigation panel. The recommended size of the logo is 150 X 60 with the same aspect ratio along with a transparent or white background is recommended for a better visibility and experience.
