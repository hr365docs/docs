    Integrations


    Seamlessly connect CLM365 with third-party applications to enhance functionality and streamline your workflows.


      CLM 365 Mailbox


      Integration feature enabling the synchronization of emails with CLM365. It manages emails related to contracts within the system, centralizing communication for better contract tracking.


      E-Signature


      E-Signature: CLM365 offers integration with leading e-signature platforms, allowing users to digitally sign contracts with ease. Toggle between available services to configure your preferred e-signature provider.


      • DocuSign: Enable the DocuSign toggle to integrate with DocuSign, allowing users to electronically sign contracts directly within CLM365. This integration streamlines the signing process, enhances efficiency, and eliminates the need for manual signatures.


      • Adobe Acrobat Sign: When the Adobe Acrobat Sign toggle is enabled, users can utilize Adobe’s secure digital signature capabilities to sign documents within CLM365, ensuring compliance and a seamless signing workflow.


        • AI Integration


        CLM365 now supports integration with modern AI tools to enhance user experience, streamline workflows, and offer intelligent suggestions.



        • Azure OpenAI Integration: Integration with Azure OpenAI enables advanced language model capabilities within CLM365 for smarter data interaction.

        • Azure OpenAI — Toggle to activate or deactivate AI integration.

        • Azure AI Endpoint — REST API or service endpoint hosted on Microsoft Azure.
        • Azure AI API Key — Authentication token to securely connect with Azure AI services.
        • Azure AI Deployment Name — Identifier of the deployed model within your Azure resource.

          • To learn how to generate and retrieve your Endpoint, API Key, and Deployment Name refer this Guide: Configure Azure AI.

        Once configured, click Save to apply the integration settings.


      • Copilot Integration: Allows enabling or disabling the Copilot assistant feature. To activate, enter the configured Power Automate (PA) URL and click Save. Once activated, Copilot provides intelligent automation and contextual assistance.

        • To integrate Copilot with CLM 365, begin by creating your Copilot agent in Copilot Studio.

        Ensure that you have a Premium Power Automate license to proceed with the configuration.

        In the left-hand sidebar, select Agents.


        If you don’t already have an agent, click on + New Agent to create one, then configure it as required.


        If you already have an existing agent, simply click on that agent to open it.


        In the Knowledge section, add your Knowledge Base.


        Then go to Settings. Under Security, select Web Channel Security and copy the Secret 1 key.




        Next, in Power Automate, go to My Flows and import the Copilot server solution.


        Open the flow and click Edit.


        Disable the New Designer toggle and paste the previously copied Secret 1 value into the Headers field.


        Save the flow and copy the HTTP GET URL generated under the When an HTTP request is received trigger.


        Return to the application, enable the Copilot integration toggle, and paste this URL into the configuration field.


        AI Summary

        The AI Summary feature becomes available once AI Integration is configured with either Azure OpenAI or Copilot. It automatically generates a concise summary of the contract document. For lengthy contracts (100–200+ pages), the AI highlights key elements such as an overview, major sections, critical clauses, risks, compliance factors, actionable insights, and key takeaways — offering a clear and structured understanding of the document.



        AI Agent

        The AI Agent becomes available once AI Integration is configured with either Azure OpenAI or Copilot. It opens a right-side panel that responds to contract-specific queries such as summaries, clauses, timelines, obligations, and risks.

        Unlike general chatbots, it only responds to contract-related queries, ensuring accurate, secure, and context-aware assistance within the selected contract.


        The AI Agent Interface in CLM 365 has been enhanced with a collapsible sidebar feature. This allows users to maximize screen space while still having quick access to AI-powered contract insights. By collapsing or expanding the sidebar, users can choose between a distraction-free view of the contract or instant access to AI guidance.


        Move Icon : Allows users to freely reposition the AI Agent panel anywhere on the screen for convenience.



        Expand/Collapse Icon : Toggles the panel between expanded and collapsed views, letting users choose between a full detailed view or a compact sidebar.


        AI Obligation

        The AI Obligation module simplifies compliance by automatically extracting and managing obligations from uploaded contract documents. Users can attach a contract PDF, and the AI engine identifies and organizes obligations for review and action.

        When creating a contract, upload the contract template in PDF format.


        If AI integration is enabled, click Next to view all obligations automatically fetched from the uploaded PDF.


        The obligations appear in a structured list for easy validation.


        For each obligation, the system auto-fetches the title and description, while you define the category, assign an owner, and set the priority.


        Select whether the task is one-time or recurring.


        If recurring, define the start date, frequency (monthly, quarterly, etc.), and number of occurrences.


        In advanced scheduling, the On option allows you to choose either a specific date (1–31) or select a weekday with its corresponding week number (e.g., 2nd Monday).


        Set the obligation’s end date, either aligned with the contract end date or defined as a custom date.


        When ready, click Add to include the obligation, or deselect it if not required.


        You can deselect any obligation that is not required.


        Publish the contract to finalize and enable obligation tracking.


        All obligations are available in the Obligations section for quick access.


        Clicking on an obligation name opens its detailed view with all associated tasks.

        If the task type is set to one-time, the obligation appears as a single instance.


        If set to recurring, it follows the defined recurrence schedule.


        If you are the obligation owner, open the task to approve or reject it, ensuring accountability and proper compliance tracking.


        AI Clause

        The AI Clause module enhances contract management by allowing users to quickly search, replace, or generate alternative clauses within contract templates. This ensures consistency, accuracy, and flexibility when managing legal language.

        Navigate to Administration, select Contract Type, and choose any contract type template. Click the edit icon to open it.


        The selected contract template opens for editing.


        On the right-side panel, select Search Alternative Clause(s).


        Highlight any text in the template to find alternatives from the existing clause library.


        If AI integration is enabled, click Get from AI to generate intelligent clause suggestions.


        The AI engine provides four alternative options for the selected text, and you can choose the most suitable one.


        After selecting an alternative, click Select to confirm your choice.


        You can replace the clause either within the contract only or in both the contract and the clause library.


        Finally, click Update to apply the changes to the contract template.


        The same option is also available when you open any contract or request. Navigate to the Draft stage and click on the document to edit.


        Within the draft document, you can access the clause replacement and AI-powered suggestions in the same way.



        AI Risk

        The AI Risk feature automatically analyzes contracts and highlights potential risks directly from the document. These risks can be reviewed, modified, or replaced with AI-powered alternatives to strengthen contract compliance and negotiation.

        When you click on Risk in any contract, the system automatically fetches potential risks from the document using AI.



        Click the Edit icon for any risk you want to modify.


        Click the Eye icon next to the clause to view its context.


        The contract template will open, and the selected risk will be highlighted for reference.


        Select Search Alternative Clauses to get AI-generated suggestions.


        AI will provide four alternative options. Choose one and click Update in Content.


        You can either Accept or Reject the changes.


        Only when you accept the changes will the updated text be applied in the contract template.
