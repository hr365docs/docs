      General Settings

      General Settings allow you to configure core preferences such as theme, language, date format, sample data, branding, and system behavior for a personalized CLM365 experience.


        Date Format

        Customize how dates are displayed across the system (e.g., MM-DD-YYYY, DD-MM-YYYY).



        Theme

        Select between light mode, dark mode, or custom site themes.



        Sample Dashboard Data

        Preload sample data for dashboard views to simulate contract workflows and metrics.




        Sample Contract/Request Data

        Insert mock contracts and request records to preview system behavior.




        Home Dashboard Theme

        Apply a theme specifically to the home dashboard for a tailored visual experience.




        CLM365 Favicon

        Upload and enable your favicon to appear in the browser tab and other applicable areas.




        GCC Tenant

        Enable configuration for Government Community Cloud (GCC) tenant settings.



        Expiring Contract

        Set alert preferences and notification periods for contracts nearing expiration.



        Enable Logs Collection

        Toggle this option to activate log collection. Collected logs support issue diagnosis and troubleshooting.

        Logs Types

        Once log collection is enabled, choose a log type from the list below based on the level of detail needed.


        • Debug Logs: Captures detailed system activity. Suitable for advanced troubleshooting. High storage usage.

        • Information Logs: Tracks general application activity. Balanced storage use with broad insights.

        • Warning Logs: Highlights potential issues. Helps with early detection. Low storage impact.

        • Error Logs: Records critical failures affecting functionality. Low storage usage and essential for urgent fixes.


        When 'Debug' is selected and saved, a download option becomes available via the Help page.


        Navigate to the Help page, enter your username, and click 'View' to access the debug logs.




        Use the 'Share' icon to securely send encrypted logs to Cubic for technical assistance.


        Click the 'Download' button to save logs locally.

        Languages


        Use Browser Default Language: Enable this option to automatically apply your browser's default language to the application interface.



        Preferred Language (Fallback): Select a default language to be used when the browser’s language setting cannot be detected.



        Support for Hebrew and Arabic languages has been added to enhance multilingual accessibility within the application.

        When Hebrew is selected as the preferred language, the application interface will automatically update to Hebrew.



        When Arabic is selected as the preferred language, the application interface will automatically update to Arabic.


        Brand Logo

        You can upload logo by clicking on Upload logo. You can crop the logo and finally click on Save Logo button.
