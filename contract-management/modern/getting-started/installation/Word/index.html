  • Installation of applications from Word

    Prerequisites:

    1. The application can only be installed by a Global Admin or SharePoint Admin by following the steps outlined below.
    2. The application’s features rely on APIs, which can be enabled by a Global Admin with the necessary permissions.

    1. Open your browser and navigate to your SharePoint tenant site.
      Example: megazap.sharepoint.com/#/

    2. Click on the App launcher (grid icon) at the top-left corner of the page.

    3. From the list of Microsoft 365 apps, scroll down if needed and click on the Admin icon (highlighted with a gear and 'A').



    4. After clicking on the Admin icon, you will be redirected to the Microsoft 365 Admin Center.

    5. In the left-hand navigation panel: Click on Settings to expand its options. Then click on Integrated apps from the submenu.


    6. On the Integrated apps page, you will see a list of deployed apps for your tenant users.
      To add a new custom app, click on the Upload custom apps button located near the top of the page.


    7. On the Upload Apps to deploy page:

      • • The App type is automatically set to Office Add-in (grayed out).
      • • Select Upload manifest file (.xml) from device.
      • • Click Choose File and browse to select the manifest XML file (e.g., manifest 9.xml).


    8. Select users to assign the app (Just me, Entire organization, or Specific users/groups) and click Next.


      9. The app CLM365 - Contract Management (Word) has been successfully deployed and is now listed under Integrated apps in the Microsoft 365 admin center with a status of OK for Word.


    9. The Word Add-In feature in CLM365 allows users to integrate Microsoft Word functionalities with contract lifecycle management features such as clause management, terms, and contract structure validation.


    10. The CLM365 Word Add-in enhances contract document management in Word by introducing features such as Terms, Alternative Clauses, and Missing Clauses. You will be provided with a draft type for the contract, and upon selecting it, you will be redirected to a new page where the Word document for the request opens, as shown below.


    11. The system displays a loading popup with the message “Redirecting to Word. Please wait…” as the draft document opens in Word.



    12. To open the CLM365 Word Add-in, go to "Add-ins" under the three-dot menu in the Word ribbon and select "CLM365 for Word" from My Add-ins.


    13. Click "Sign in with Microsoft" in the CLM365 for Word pane to start using the contract management features inside Word.


    14. After clicking "Show Task Pane," you'll see the request details along with options for Terms, Alternative Clauses, and Missing Clauses. Additionally, you can select the specific request you want to work on, as shown below.


    15. Clicking "Terms" displays all the terms related to the request, counterparty, and organization linked to the selected request.


    16. Click on a term to insert it directly into the document.


    17. After this, you can save the changes by clicking the "Save" icon, which will replace the term with its actual value and save the document.


    18. AI Assistant for Clauses: Select a clause from the contract and use the AI to generate an alternative version of that clause.


    19. i. Get from AI: After clicking the "Get from AI" button, the system will analyze the selected text and display relevant suggestions in a side panel. You can browse through these options and click the "Select" button on your preferred suggestion. This action will render a text field containing the selected option along with two actionable buttons. Meanwhile, users can also switch the pivot to access suggestions from the Clause Library instead of the AI-generated content, offering flexibility in clause selection. (As shown in the image below.)


    20. ii. Replace in Contract: Once the desired suggestion is selected, click on the "Replace in Contract" button. This will replace the selected clause or text in the document with the content chosen from the AI-generated suggestions or Clause Library.







    21. Once the user clicks the Create button, the AI-suggested content is saved to the Clause Library as a new clause, using the user-provided Clause Type and Tags. Simultaneously, the selected text in the contract is replaced with the AI-generated clause.

    22. Get from Clause Library: In addition to the AI-generated suggestions, users will have the option to retrieve clauses directly from the Clause Library. This allows users to select and insert predefined clauses into the contract as an alternative to the AI-generated content.


    23. Clicking the Get from Clause Library button opens a modal window displaying options to filter clauses by Clause Type and Tags. Users can use these filters to quickly find and select relevant clauses to insert into the contract.

    24. After the user selects the desired Clause Type and Tags and submits the form, the system displays default clause options from the Clause Library that match the selected criteria. Additionally, a "Show/Hide Alternatives" link will be available, allowing users to view alternative clause variations related to the same type and tags.


    25. Once the user clicks the Show Alternatives link, the alternative clauses will appear as additional options below the default clause.


    26. When the user selects one of the clause options, a Select button will appear. Upon clicking it, a text field containing the selected clause will be displayed along with a Replace in Contract button. Clicking this button will replace the selected text in the document with the chosen clause.



    27. AI Assistant for Missing Clauses: On clicking AI Assistant for Missing Clauses, the system will analyze the document and display a list of all missing clauses identified by the AI.



    28. AI Assistant for Clause Summary: Users will have the option to access the AI Assistant for Clause Summary, which provides concise summaries of the clauses within the document.


    29. On clicking AI Assistant for Clause Summary, you will be shown summaries of all clauses in the document. Clicking on a summary will insert it into the document.


    30. Contract Template: When adding or updating a contract type template, clicking the Update button will redirect you to a new page where the Word document for the contract template opens, as shown below.
