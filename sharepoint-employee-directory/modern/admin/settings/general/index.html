    General

    In this, we can hide the search filter on the home page, grant the admin role to the user, do collaboration settings, upload company logo, language selection, update user's department, update users location, update users job title, and add custom links.


    Auto Load Configuration

    Automatically load additional records either by scrolling or by clicking the "Load More" button.


    If the 'Auto Load' toggle is turned off, the 'Load More' button will be displayed.



    If the toggle is on, the 'Load More' button will not be shown, and records will load automatically.




    Birthday and anniversary templates

    Birthday Template: Customize the email template for employee birthdays. You can include personalized messages, company branding, and specific details to make the email more engaging. Schedule the emails to be sent automatically on the employee's birthday.




    Anniversary Template: Customize the email template for employee work anniversaries. You can include congratulatory messages, milestones achieved, and company branding to recognize the employee's contributions. Schedule the emails to be sent automatically on the employee's anniversary date.

    GCC Tenant

    If your Microsoft 365 tenant is GCC (US Government Community Cloud), please ensure that the specific setting is enabled in order to ensure compatibility within the GCC environment.


    Extend Mobile Number and Phone Number Search Capability

    Please activate this feature if you want to search by 'starts with', 'ends with', or 'contains', but only if the number of employees is under 3000.



    Extend Custom Fields Search Capability

    If you want to utilize the 'starts with', 'ends with', or 'contains' search functionalities, please enable this feature. Additionally, limit its use to cases where the number of employees is under 3000.



    Filter Attributes

    You can choose from three filtering attributes:
  • Update Department
  • Update Job Title
  • Update Location



    1. Update department: This configuration is employed for modifying the user's department.


    2. The user can sync all departments from Azure, and within a short time, all the departments will appear.


    3. Click on Import CSV button


    4. Here, the user can both download and upload files. After that, by clicking the Save button, the file will be uploaded.


      5. Enter Department and then Save button click your department will save.


    5. Update job title: This configuration is utilized for updating the user's job title.


    6. The user can sync all job title from Azure, and within a short time, all the job title will appear.

    8. Here, the user can both download and upload files. After that, by clicking the Save button, the file will be uploaded.


      9. Enter Job title and then Save button click your Job title will save.


    9. Update location: This setting is employed to modify the user's location..


    10. The user can sync all location from Azure, and within a short time, all the location will appear.


    11. Click on Import CSV button.

      13. Enter Location and then Save button click your Location will save.

    Language selection as browser default language

    By default, this feature is enabled. You can disable it from the toggle switch. If this is enabled, then the language will auto change to the language used in the browser. Employee Directory can work in Czech, Dutch, English, French, German, Hungarian, Italian, Japanese, Polish, Portuguese, Russian, and Spanish as per your browser language.

    Language selection

    From here you can change the language manually. Here the selected language will reflect for all users, even if the browser language is different. To enable Language selection, make sure that the Language selection as browser default language must be in disable condition.


    Support for Hebrew and Arabic languages has been added to enhance multilingual accessibility within the application.


    When Arabic is selected as the preferred language, the application interface will automatically update to Arabic.




    When Hebrew is selected as the preferred language, the application interface will automatically update to Hebrew.



    Organization name

    Please include your organization's name here. This will ensure accurate QR code generation and proper titling when printing the employee directory as a PDF.

    Records to load

    Load the number of profiles when the page refreshes or for the initial page load. Consider load times carefully when choosing higher numbers.

    Role and Permissions.

    Select Role and Permissions from the menu.

    Click Add to assign admin rights.


    The admin will have the rights to configure the application.

    Sync Job Title, Location, and Department

    This feature allows automatic synchronization of Job Title, Location, and Department details from the Office 365. When enabled, it ensures that employee records are updated with the latest organizational structure information.



    Validate Work Email with SharePoint Account Name

    Enable this setting to validate the work email with the SharePoint account name. Note that enabling this setting may increase the page load time of the application, so use it judiciously.
