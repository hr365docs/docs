• SPFx Permissions

    1. Start by opening the Microsoft Teams admin center – https://portal.azure.com/


    2. From here select App registrations.


    3. Search for SharePoint and click on “SharePoint Online Client Extensibility Web Application Principal”.




    4. Click on “API permissions” under Manage (left side) you will see the application related permissions.


    5. Move these permissions to Configured permissions and grant admin consent.


    6. After all these changes please go to Employee Directory and check.
