How to add new users in the Employee Directory Modern?
You can add users from the O365 admin page or azure and users sync or appears in Employee Directory Modern after 24 hours automatically.
Can I add non-office365 users In the Employee Directory?
Yes, we can add non-office365 users. To add non-office365 follow below steps:
  • Go to the Settings.
  • In Advance, click on the import button, the panel open and download the sample file.
  • Fill non-O365 user details as per the specified format in the file. Choose the file, and click on the Save button.
    • How to edit the user profiles in Employee Directory Modern?
  • Admin can edit the user’s details from the O365 or Azure. The user’s details like birth date, joining date, skills, hobbies, projects, and about me can be updated from the SharePoint User profile. SharePoint admin can visit the SharePoint user profile from the below link URL format.

    • https://Your tenant name-admin.sharepoint.com/_layouts/15/TenantProfileAdmin/ProfMngr.aspx?

  • An individual user can edit their profile details like birthday, joining date, skills hobbies, projects, mobile number, and about me from delve profile. Users can visit the below URLs format to update the details.

    https://Your tenant name -my.sharepoint.com/person.aspx

    • Note: Update reflects in Employee Directory Modern after some time.
    What are the locations from where profile pictures can be synced into the Employee Directory Modern?
    We can sync pictures from SharePoint Profile, outlook, SharePoint Library (custom pictures)
    Why does the user outlook picture do not appear or update in the Employee Directory Modern?
    Due to CORS outlook Pictures don’t appear.
    What is CORS?
    A request for a resource (like a picture) outside of the origin is known as a cross-origin request. CORS (cross-origin resource sharing) manages cross-origin requests i.e., EDM works from https://tenant-appid.sharepoint.com & if Pictures are being called from https://outlook.office.com, it treats as cross-origin resource sharing. Please click on the CORS tab to allow cross-origin resource sharing from the help in Employee Directory Modern. This activity is one time until you clear cache/cookies from your browser Note: We recommend you to choose either SPO or ED (Employee Directory SharePoint library).
    How to configure SharePoint API?
    Go to SharePoint admin center then from left panel under advanced click on API access option under pending request, there are 3 requests (Presence.Read.All, User.Read.All, and MailboxSettings.Read) select and click on approve one by one for all the 3 requests.
    How to add custom fields?
  • First Configure SharePoint API
  • In settings in advance click on the action button of custom fields.
  • Select the properties from the AD properties dropdown list, type the custom fields name, and click on the Save button.
  • Custom fields value is displayed at bottom of grid views of the home page.
    • In which case we can use extension attributes to add the custom fields?
    If you are not using the active directory and need to show extra users details in the Employee Directory Modern you can use extension attributes.
    In O365, where we can update the users’ details in the extension attribute?
  • Exchange admin can update the user’s details in the extension attribute.
  • Go to the Exchange admin center.
  • Click on the recipient in the side navigation panel.
  • Double click on the display name, edit user mailbox popup window appears, click on more options then click on the edit icon of custom attributes, 15 text boxes appear where you can update the details and syncs in Employee Directory Modern using extension attributes custom fields.
    • How to use custom user profile Pictures or add Pictures in Employee Directory Modern?
    You can upload the Pictures in the SharePoint library
  • Go to settings in advanced click on click here link under “Sync employee profile Pictures from”, redirects to SharePoint library of Employee Directory Modern. upload Pictures in the library, the format of the picture is - username@domain.com.jpg.
  • Finally, select the ED option in settings in advance and click on the Save button.
    • How to exclude shared mailbox users?
    The global admin can exclude shared mailbox users Please ensure SharePoint API configured
  • Go to settings then click on exclude option.
  • At the bottom, click on the click here link, permission page gets opened in another tab then select the check box and click on accept button you will redirect to the Employee Directory tool.
  • Put below format URL with a forward slash in Shared mailbox URL box and click on run button Wait for some time till process complete https://Your Tenant name.sharepoint.com/.
  • After the completion of the process, you will find the shared mailbox users deleted reports at the bottom of the same page
  • Now in Employee Directory Modern settings in Exclude option Enable toggle for Exclude Shared mailbox
    • Note: If the report does give deleted users results or the report could not produce after the process completion, please try the same process after some time

    From where do filter options come (department, location, job title)?

    How we can exclude inactive users of the Active Directory in the Employee Directory Modern?
    You can do block sign-in for inactive users from O365 and those users can be excluded by enabling “Exclude O365 sing in blocked users” in Employee Directory Modern settings.
    How to sync mobile no from azure?
    After configuring SharePoint API enable the toggle for “Sync and display mobile number from azure” in settings in advance.
    Can we export all the users into a CSV sheet?

    Hidden GAL users appear in Employee Directory Morden thou exclude users hidden in address list is enabled in the setting?

    How to enable user presence status (active, busy, appear away, etc)?

    Why does the grey picture icon appear instead of the name initial in Employee Directory Modern?
    In SharePoint user photo’s folder, if grey picture icon exists for users, shows same in Employee Directory else shows name initial. SP admin can visit the folder from the below URL format https://Your tenant name-my.sharepoint.com/User%20Photos

    What is the difference between employee directory classic and employee directory modern?

    Employee Directory Modern

  • It's a SharePoint web part that works on SharePoint modern portals.
  • It is virtually for unlimited users.
  • Performance is better as the latest technology and scripting language React are used.
  • The online presence of users is visible.
  • Its works on SharePoint Modern Portal.

    • Emoloyee Directory Classic

  • It's a SharePoint hosted Add-in.
  • It has a limitation of 3000 users.
  • Performance is good.
  • The online presence of a user is not visible.
  • It works on both SharePoint classic portal and Modern Portal.
