Home Page

    The home page is the first page that is displayed to you in Employee Directory 365 application. The home page has filters, employee's profiles, Settings, views (Grid View,List View, Tile View) and organization chart etc.


Search

    • Users alphabetical filter: You can sort the users by first name.


    • Search filter: Search filter allows you to quickly define what it is you want to see. In employee directory 365 you can filter by name search, free search, title, departments, office location, project and skill.

    System Administrators can enable/disable filters by clicking on check-box within the setting panel.
