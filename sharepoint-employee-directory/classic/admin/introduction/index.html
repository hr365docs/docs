    Introduction

    Description

    Office 365 SharePoint employee directory app displays information about employees in the organization in a visual manner and allows you to find any user with various filters. This employee directory software helps in retrieving important information. With a QR code, you can scan and save it on your mobile phone.

    Features

    • Search users
    • Display User Presence status
    • Microsoft Teams and Delve Integration
    • Exclude User
    • Sync Data from Microsoft 365

    Home page

    Use the Application home page to search the users, edit the profile, download the details, view the profiles.


    • Users Alphabetical Filter:You can search different fileds.
    • Download: You can download the details in excel.
    • Settings: You can edit the fields.
    • Help: You can contact and support.
    • Filters: You can refresh the page.
    • Grid View:You can view.
    • List View: You can view in list.
    • Tile View: You can view the profile in tile.
    • Org chart: You can view organization chart.
