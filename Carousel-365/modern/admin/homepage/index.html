  • Home Page

      • The Carousel 365 Home Page allows you to add and manage slides for your SharePoint site. These slides can include images, titles, and links to create an engaging and interactive experience for users.


      Add Slide: Click the + Add Slide button to begin creating a new slide.


      When adding a slide, you can choose an image source from the following options:

      • Recent — Select from recently used images.

      • Site — Browse and select images stored within the SharePoint site.

      • Upload — Upload a new image directly from your device.

      • From a link — Insert an image by pasting its URL (hosted online).

      Upload Image: Choose an image from your device, then click Insert to add it to the slide.


      After inserting an image, the Slide Details panel will open. Here you can configure:

      • Alternative Text — Provide descriptive text for accessibility.

      • Title — Enter the slide’s display title.

      • Show Title in Layout — Toggle on to display the title within the slide layout.

      • Background Gradient — Optionally apply a gradient background effect to the slide.

      Add Link: Provide a URL that the slide should link to. When a link is added, a toggle appears to Open in New Tab.


      Save Slide: Once all details are entered, click Save to publish the slide to Carousel 365.


      Navigate Slides: Use the left and right arrow icons to browse through all the available slides in the carousel.



      Edit Slide: If you need to update a slide, click the Edit option to modify its image, title, link, or layout settings.




      Settings: The Carousel 365 Settings option allows administrators to configure layout, content order, and display preferences such as indicators and autoplay. These settings help control how slides are presented on the home page.



      Help: The Help section provides quick access to license details, product version, subscription validity, and links to resources such as Support, Privacy Policy, FAQ, Release Notes, and Product Activation.
