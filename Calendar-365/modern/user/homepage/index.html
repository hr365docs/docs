  • Home Page

      • The Calendar 365 Home Page provides users with an intuitive interface for viewing and managing calendar events within SharePoint. Users can see all events they have access to, create new events if permitted, and collaborate across teams.


      Key features available for users:

      View Events: Click on any scheduled event in the calendar to open a detailed view, including event name, description, and timings. Users can see all shared events, but personal events of other users are not visible.



      Create Event: Users can create new events only if the administrator has granted the permission. Click on any date to enter event details such as title, description, start & end times, and select an available source.



      Edit/Delete Events: Users can update or remove events only if the administrator has given permission. Otherwise, these options are disabled.



      Calendar Views: Switch between different calendar formats – Month, Week, Day, or Agenda – to view events in your preferred layout.



      Help: Access the Help panel to view license details, version information, subscription validity, and quick links to Support, Privacy Policy, FAQ, Release Notes, Product Activation, and License Validation.
