Settings

The Calendar 365 Settings panel allows administrators to configure global options, including visual themes, user permissions, and calendar sources. These settings help customize the web part’s appearance and control access effectively.

Theme

Select a visual theme for Calendar 365:

  • Light: Bright, clean interface.
  • Dark: Optimized for low-light environments.
  • Site: Inherits the theme of the SharePoint site.

Hide SharePoint Settings

The Hide SharePoint Settings option allows administrators to disable the default SharePoint settings menu within the Calendar 365 web part. This helps prevent unauthorized modifications and keeps the interface focused only on poll-related configurations.



Calendar Sources

Administrators can add and manage calendar sources.


To add a new source, click the + Add button. Sources can be created from either a SharePoint list or a personal calendar, with options to configure the title, permissions, event columns, colors, and read-only settings.


Add SharePoint List

  • Enter a Title for the calendar source.
  • Enable Read Only toggle if required.
  • Select the Site, then the List and List View.


    Click on + Add to create a new list. Enter the List Name and then click Save to add it to Calendar 365.



  • Map columns to calendar fields: Title, Description (click + to add multiple descriptions), Start , End.

  • Specify which users can Add/Edit/Delete events and select a color for the events.




  • Click Add to save the source.



Add Personal Calendar

  • Enter a Title for the personal calendar.
  • Enable Read Only toggle if required.
  • Select a Color for the calendar events.
  • Click Save to add the personal calendar source.

Note: Personal calendar events are visible only to the creator and cannot be accessed by other users.

Once a source is added, you can make it active by clicking the checkbox. You can also edit or delete the source as needed.


Add Additional Admins

The Additional Admins section allows administrators to manage permissions in Calendar 365. The user management grid provides the following functionality:


  • User Grid: Displays Name, Email, and Role (Admin/User) for each entry.


  • Role Assignment: Administrators can assign or update roles (Admin/User) via the dropdown.


  • Search & Pagination: Search by Name, Email and adjust the number of entries displayed (10, 25, or 50 per page).
