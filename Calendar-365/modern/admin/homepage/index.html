  • Home Page

      • The Calendar 365 Home Page provides an intuitive interface for scheduling and managing events within SharePoint. It allows users to create events, view them in multiple formats, and collaborate seamlessly across teams.


      Key features include:

      Create Event: Click on any date to create a new event by entering a title, description, start & end dates, and selecting a source.



      View Event: Click on any scheduled event in the calendar to open a detailed view, including event name, description, and timings. Options to Edit or Delete are also available.



      Manage Events: Update existing events by editing details or remove them permanently using the delete option. Color-coded categories make it easier to differentiate events.



      Calendar Views: Switch between different calendar formats – Month, Week, Day, or Agenda – to see events in your preferred layout.




      Settings: Administrators can configure options such as theme (Light, Dark, Site), event sources (SharePoint Lists or Personal), and user permissions for adding, editing, or deleting events.


      Help: The Help panel provides license details, version information, subscription validity, and quick links to Support, Privacy Policy, FAQ, Release Notes, Product Activation, and License Validation.
