Copilot Integration

To integrate Copilot with Helpdesk 365, begin by creating your Copilot agent in Copilot Studio.

Ensure that you have a Premium Power Automate license to proceed.

In the left-hand sidebar, select Agents.


If you don’t already have an agent, click + New Agent to create one.


If you already have an agent, simply click on it to open.


In the Knowledge section, add your Knowledge Base.


Go to Settings → Security → Web Channel Security and copy Secret 1.




In Power Automate, go to My Flows and import the Copilot server solution.


Open the flow and click Edit.


Disable New Designer and paste Secret 1 into the Headers field.


Save the flow and copy the HTTP GET URL from the trigger.


This is the URL you need to provide to Helpdesk 365 Copilot.
