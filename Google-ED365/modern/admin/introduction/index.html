    Introduction

    Overview

    Google Workspace Employee Directory 365 provides a visually intuitive interface to browse and search employee information within the organization. With various filters, it enables quick user lookup and supports QR code scanning to save contact details directly to a mobile device.

    Home Page

    The home page includes key features such as User Alphabetical Filter, Dashboard, Download, Settings, and Help.


    User Alphabetical Filter: Easily search for employees by first name, last name, or display name. This can be customized from the Views section in Settings. Selecting a letter from the top navigation will filter employees accordingly.


    Dashboard: View key organizational insights and employee statistics in a consolidated format. This section provides quick access to important information at a glance.



    Export to CSV: Export the user directory to CSV format for reporting or record-keeping. Admins can control this functionality in General Settings.



    Click the cake icon on the tab bar to view upcoming birthdays and work anniversaries.



    Print to PDF: Export team member details displayed on the home page to PDF. This option can be managed by the admin in General Settings. While exporting, users can also choose between Portrait or Landscape orientation.



    Organization Chart: Signed-in users can view their reporting or relationship hierarchy. Admins can enable or disable the Organization Chart via Advanced Settings.


    Settings: Customize, enable, or disable features from the Settings section.



    Help: Access contact and support information from the Help section.



    View Icon: Toggle between Grid, List, and Tile views. Admins can control the visibility of the View option from the Views section under Settings.



    Grid View: Displays items in a grid layout for quick scanning.



    List View: Shows items in a vertical list with details.



    Tile View: Displays items as large tiles with emphasis on visuals.



    Admins can also manage which user details are shown or hidden by configuring the Views settings.
