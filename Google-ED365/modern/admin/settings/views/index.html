Google Workspace Employee Directory 365
  • Admin Guide

    Views

    In Google Workspace Employee Directory 365 you will be presented with various views. These views are structured to present you with key information.


      Align employee basic information

      In this, you have the option to align basic information to the left, right, or center.

      When the user selects left alignment, the user details will be displayed on the left side.



      When the user selects center alignment, the user details will be displayed on the Center.



      When the user selects right alignment, the user details will be displayed on the right side.


      Dashboard feature

        There are two features in the dashboard:

      1. Available for All Users: This indicates that the dashboard is accessible to all users
      2. Available for Selected Users: This means the dashboard will only be accessible to specific users who are added to the include user list.



      Grid Width

      In this location, you can adjust the grid width in the grid view by expanding or reducing it. Additionally, there are two buttons: Reset to Default and Save. Clicking the Reset to Default button will restore the width to its original default setting.


      When the user sets the width to 300px and clicks Save, the width will be applied accordingly.


      As a result, the card will be displayed with a width of 300px, based on the specified value.


      Hide manager of specific user

      In this section, you can hide the manager of specific user.


      If you want to hide a user's manager, enter their name, and the manager will not be visible in the user's profile card.



      if you want to revert to the previous view, select the user in the delete section and delete it. Then, it will show as it was before.



      Hide mobile number of specific user

      Utilize this configuration to conceal the mobile number of specific users within the application. Simply select the user and click the Add button.





      if you want to revert to the previous view, select the user in the delete section and delete it. Then, it will show as it was before.


      Image profile tag

      The image profile tag is utilized to showcase user properties on the profile card and offers two features:
      1. Customizing the background color of the tag.
      2. Modifying the font color of the tag.

      If the user wants to add a tag to the image inside the user card, they should select a name from 'Extension Attribute', choose the tag's background and font colors, click Submit, and the image will display the name with the selected styles.



      it can also be removed if needed.

    Labels

    Within this option, you can update the labels to any text of your choice. For example, you might set them as Executive to Executive(s).


    If you want to change a label, such as changing 'About Me' to 'About For Me', type the new label in the field under 'About Me', click Save, and the Profile Card will reflect the updated label; this also applies to Executive labels.


    Show or Hide modules

    Within this panel, there are many features, each accessible via a toggle switch:


    Report Incorrect Info : Enable this toggle to allow users to send an email to admins to report incorrect user information in the directory.


    Email ID(s) to Report Incorrect Information: Enter the email address where users can report incorrect information.


    When you click on the Help icon in your profile page, you can report incorrect information.


    Show Joyful Animations : When it is an employee's birthday or work anniversary, a celebratory animation along with a message such as "Happy Birthday" or "Happy Work Anniversary" will be displayed on their profile.



    When the Show extended fields in list view toggle is turned on, only a vertical scroll (up and down) will be visible within the list, and the horizontal scroll bar will be hidden.



    If the Show filters as dropdown toggle is turned on, a dropdown menu will appear at the top of the home page, allowing users to easily select filters.



    If the Show Groups toggle is on, then you can select the group you need.


    After turning on the toggle, a dropdown will appear labeled as Group. You can use this dropdown to search and select the desired group.


    If the Enable WFH toggle is turned on, an edit icon will appear next to the user's name on the profile card.



    Click on the icon to open a popup where you can input the WFH (Work From Home) details, after which the profile card will display the WFH icon.



    If the Organizational Chart toggle is turned on, the organizational chart icon will appear within the home tab.


    If you click on it, a chart will open in front of you.


    When the Show Member Of toggle is turned on, you will be able to view the groups the user belongs to.



    If the Show Praise toggle is turned on, the Praise icon will become visible on the home screen.



    To send a praise, fill in the required fields including the recipient’s name, select the praise type, choose the specific praise, and then click Submit to complete the process.


    The user will receive the praise, and it will be added to their profile as a part of their recognition history.

    When the Display Weekly Work Status on Profile Card toggle is turned on, the profile card will show the user's WFO or WFH details along with the date and day for the entire week.



    If you want to edit, click on the edit icon.


    Select Default Always Work From Home. If you choose this option, then click on save.


    Select Default Always Work From Offer. If you choose this option, then click on save.


    If you want to change the status for the current week, you can do so. After making the changes, click on the save button to save them.


    Similarly, if you want to make changes for the next week, you can do so as well.


    A Next icon is provided; clicking on it will display the weekly report for the next week.


    Turn on the toggle to enable the Show text to speech functionality.


    The Text-to-Speech icon (🔊) will be visible in the Grid View on the user's Profile Card, positioned next to their name. By clicking the speaker icon, the system will use built-in text-to-speech functionality to pronounce the user's name aloud.


    Enable the Allow users to update their pronouns option so that users can update their pronouns (they/them, she/her, he/him) and save the changes.



    If the Show Dashboard toggle is turned on, the dashboard icon will appear within the home tab.



    you can view the dashboard data by clicking on the icon at the top.


    When the Allow users to export directory toggle is turned on, users can click the "CSV file" icon on the home page to export the directory.



    Then, a popup will appear, and in the popup, you will have the option to download. If you want to download the displayed records, click on the download button to save them.


    When the Allow users to print PDF toggle is turned on, users can click the "Print file" icon on the home page.



    After that, a popup will appear with two buttons: "Portrait" and "Landscape." You can choose either option based on the type of file you wish to download.


    Upcoming Birthdays and Anniversaries

    Enable this toggle to display icons for upcoming birthdays and work anniversaries in the command bar.


    If the Favicon toggle is turned off, the default browser favicon will be shown. Enable this toggle to display a custom favorite icon.


    If the favicon toggle is turned on, an option will appear at the bottom, allowing you to upload and set a custom favicon.



    Then, your custom favicon will be displayed.


    Sort by

    You can sort employee cards using two options available under Advanced Settings: First Name and Last Name.


    If First Name is selected, the cards will be sorted alphabetically by first name.


    If Last Name is selected, the cards will be sorted alphabetically by last name.


    Views


    In the View section, these options will be displayed. You can select the options you want, and then click Save.


    According to these icons, you can view the directory in different formats: List, Grid, Tile.


    By default, the desktop view will be in Grid format.You will also have the options for List, Tile views.


    List Views


    By default, the Mobile view will be in List format.


      User properties in grid view

      User properties in grid view.


      You can update the user's properties, and their details will be displayed according to the options you selected.



      User properties in list view

      You can update the user's properties, and their details will be displayed according to the options you selected.


    User properties in Profile Card


    You can update the user's properties directly within the profile card.

    User properties in hover card

    The User Properties in Hover Card feature enables admins to customize the details shown when hovering over a user's profile in the directory. This allows key user information to be displayed at a glance, enhancing visibility and quick access to relevant data.


    You can update the user properties in the hover card. Once configured, the selected user properties will appear in the hover card view across the Employee Directory application, ensuring consistent and quick access to key user information.



    Theme

    From here you can select the theme for the application. There are three themes namely dark mode, light mode, and site theme color.


    If you select the Dark Theme option, the theme of your application will change to dark.


    If you select the Light Theme option, the theme of your application will change to light color.


    If you select the site Theme option, the theme of your application will change to site color.
