    General

    This section allows us to hide the search filter on the home page, assign the admin role to users, configure collaboration settings, upload the company logo, select the preferred language, update user departments, update user locations, modify user job titles, and add custom links.


    Birthday and anniversary templates

    Birthday Template: Customize the email template for employee birthdays by incorporating personalized messages, company branding, and relevant details to enhance engagement. You can also schedule the emails to be sent automatically on the employee's birthday.




    Anniversary Template: Customize the email template for employee work anniversaries by including congratulatory messages, milestones achieved, and company branding to celebrate the employee's contributions. You can schedule the emails to be sent automatically on the employee's anniversary date.



    Brand logo

    This functionality allows users to hide the titles of web parts and optionally display a brand logo on SharePoint pages where applications are active.

    Note: For optimal display, the recommended maximum width is 200px and the maximum height is 120px.


    Click on Upload Logo to add a brand logo.


    The brand logo you added will be visible at the top right of the user profile card.


    If you wish to remove it, simply click on the remove button.


    After removing it, it will no longer be visible on the user profile card.

Collaboration

If your organization uses a softphone, please choose either the SIP or Tel protocol. This enables direct calling functionality within the employee directory.



On work phones, selecting the Tel protocol allows numbers to be dialed directly through the device's phone app.



On mobile devices, selecting the Tel app prompts the user to choose from available calling apps.



If the Chat app is selected for a user, clicking the chat icon will redirect to the configured chat application.


The chat icon will be available inside the user card for quick access.


Modern View: Gmail and Chat

When using the Modern layout, the action icon reveals two options: Gmail and Chat.


These communication options (Gmail and Chat) will be displayed as icons at the bottom of each user card.


Classic View: Phone, Mail, and Chat

In the Classic layout, users can choose between Phone, Mail, and Chat communication options.


Corresponding icons will appear in the user card, depending on selected options for each user.


    Filter Attributes

    You can choose from three filtering attributes:
    1. Update Department
    2. Update Job Title
    3. Update Location



    1. Update department: This setting allows you to modify a user’s department.


    2. The user can sync all departments from Azure, and within a short time, all the departments will appear.


    3. Enter the department name in the Enter Department field, then click the Add button to save it manually.

    4. Update job title: Select the user and modify the job title in the Update Job Title section, then click the Save button to apply the changes.


    5. Click the Sync from Azure button to import all job titles automatically; they will appear in the list within a few moments.


    6. Enter Job title and then Save button click your Job title will save.

    7. Update location: This setting is employed to modify the user's location..


    8. The user can sync all location from Azure, and within a short time, all the location will appear.


      9. Enter your location and click the Save button to save the changes.

    Home page custom url

    In the custom URL section of the homepage, you can input the URL of your favorite website and navigate there with a single click.


    Here, the user can enter their custom URL, such as https://apps365.com, and also add an icon. After that, they can click the Save button."


    And then, the user will see this icon on their home screen. By clicking on the icon, they will be redirected to the site they provided in the custom link.


    Employee info page

    You can define specific url in Azure AD extension attribute column, this setting will enable employee info page. Please refer here on how to setup up extension attributes


    In the Employee Info section, first, select AD Properties, followed by the Property Name, provide a name for the icon in the field Then finally, click Submit.


    Languages

    By default, this feature is enabled. You can disable it using the toggle switch. When enabled, the language will automatically change to the language set in your browser. The Employee Directory supports Czech, Dutch, English, French, German, Hungarian, Italian, Japanese, Polish, Portuguese, Russian, and Spanish, depending on your browser's language settings.



    Language selection

    From here, you can manually change the language. The selected language will be applied for all users, even if the browser language is different. To enable Language Selection, ensure that the Language selection as browser default language option is disabled.



Organization Chart Type

If your organization has 2000 employees or fewer, a complete organizational chart is readily accessible. However, large-scale org charts may hit data limits depending on your platform or backend configuration. For viewing org charts with over 2000 employees, please use the dynamic option labeled 'Org Chart – Three Level.'
