  • Release Note

    25.0712.0015.0000        Released 14-July-2025


    🔧 Bug Fixes

    1. Improved overall application performance for a smoother and faster user experience.
    2. Introduced multiple user interface (UI) enhancements to improve usability and visual consistency.
    3. Fixed an issue in the User module where updating Date of Birth (DOB) and Date of Joining (DOJ) was not functioning as expected.
    4. Resolved a problem in the KRA module where changes to section names, such as Achievements and Self Reflections, were not being properly reflected.

  • Release Note

    25.509.1237.000        Released 15-May-2025

    🚀 What's New

    1. In the My Team section for Managers, employees are now categorized as Direct and Indirect Reportees for improved clarity and hierarchy visibility.
    2. HR can now reject reviews—both Employee and Manager reviews—directly from the HR Review page, streamlining the review management process.

    🔧 Bug Fixes

    1. Resolved an issue where Department and Manager names were not displaying in the PIP In-Progress list.
    2. Fixed a bug where the "Request Refill" option did not appear on the Manager Review page after submission.
    3. Addressed an issue with searching and selecting users in the "Add Employee" section of the Appraisal Cycle.

  • Release Note

    25.0301.1104.0000        Released 5-Mar-2025

    🚀 What's New

      1. In the "Select Appraisal Cycle" dropdown for self-review, the most recent appraisal cycle now appears at the top and is sorted in descending order by the review start date.
      2. Appraisal cycles in the Appraisal Cycle tab now appear in descending order based on the review start date.
      3. Badge titles now appear with their corresponding icons in the badge list.
      4. HR can now view praises for all users.

    🔧 Bug Fixes

    1. Resolved an issue where users were not being selected through the search functionality in the Appraisal and KPI cycle.
    2. Resolved the issue where selecting "All Categories" was not being applied in the appraisal.
    3. Fixed the sorting issue on the Review Report page.
    4. Implemented UI fixes on the "Give 360 Feedback" page.
    5. Implemented UI fixes on the "Edit PIP" page.
    6. Implemented UI fixes on the "View Tasks in PIP" page.

  • Release Note

    Version 24.1212.1343.0        Released 17-Dec-2024

    🚀 What's New

    1. In the column settings of the self-review in all pivots, the "Appraiser Rating" and "Appraiser Remarks" fields have been removed.

      🔧 Bug Fixes

    1. On the "Add and Edit Goals and Competency" page, when selecting "All" in the category dropdown, not all categories were being selected, as seen with the KRA categories. This issue has been resolved.
    2. The issue with the "Competency and Skills" page, where the category was not appearing in the exported file, has been fixed.
    3. On the "KPI" page, the sequence of pivots has been rearranged to: KPI, MY Team, and All Teams. Additionally, the "My HR Team" has been renamed to "All Teams".
    4. The renaming of module names (KRA, Goals, Competency & Skills) in the column settings was not being reflected. This issue has been resolved.
    5. On the dark theme of the SharePoint site, after the submission of both the self-review and manager review, text in all summary boxes was not visible. This has been fixed.

  • Release Note

    Version 23.1213.245.0        Released 14-Dec-2024

    🚀 What's New

    1. Capability to upload a PDF file in feature settings. The uploaded PDF, containing information, will be displayed in the self-review.
    2. The "Assign to template" option under administration has been relocated to the KRA template page.
    3. Addition of the ability to add an external manager to a user on the "Add User" page. The external manager can review the employee.
    4. Introduction of self-review delegation in Feature settings. Managers can perform a self-review of their employees by enabling the delegation option.
    5. Employees can now view their previous appraisal cycle performance by selecting the appraisal cycle in self-review.
    6. Removal of the brand logo.
    7. Download PDF feature in the Manager's dashboard. Once the Manager completes the appraiser review, they can download the review in PDF format by clicking on the PDF icon located in the action column.

    🔧 Bug Fixes

    1. Alignment fixes for the user's name.
    2. Fixes for empty duration filters in the Competency and Skill page.
    3. Resolution of validation popup issues in self-review.Ṭ

    Version 23.1112.9.0        Released 16-Nov-2024

    🚀 What's New

    1. Can add KRAs, goals, and competency & skill with different weightage and targets from the administration settings
    2. Can create a template by adding KRAs, goals, and competency & skills modules along with weightage from the template page from the administration settings
    3. Can Assign the template to the users from the administration settings

    7. Can create an appraisal cycle with or without a rating from the add appraisal cycle page by enabling or disabling the rating toggle.
    8. "Direct Rating Release by Manager" toggle setting in add appraisal cycle (If this is enabled manager can release the review rating without any HR approval by clicking on the release icon in the action column after the completion of their review)
