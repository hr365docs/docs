    General Settings

      Here you can set the date format, and theme color, configure SharePoint API permissions, hide the side navigation panel, hide the web part title, remove the custom web part CSS, final rating, brand logo, etc.

    • API Configuration: API configuration help in working delta sync, excluding unlicensed and shared mailbox users on theuser's page Click on the link.



    • Date Format: Admin could select the desired date format (Reflects the entire application like date columns etc.)



    • Hide Sharepoint Page's Default Components: This component is an integral part of SharePoint by default.



        Select Configure to process further.


      1. Hide Side Navigation Panel : Hide side navigation panel in sharepoint page where app is running


      2. Hide WebPart Title : GCC (US Government Community Cloud), if your Microsoft 365 tenant is GCC, please enable this setting to make it work


      3. Remove Custom Webpart CSS : Enable this toggle will remove space outside of webpart


      4. Hide Top Command Bar on Sharepoint Page : Instructions for hiding the Top Command Bar on a SharePoint page


      5. Hide Top Site Header: Instructions for hiding the Top Command Bar on a SharePoint page


    • Theme: A theme could be selected for the application.



    • GCC tenant: GCC (US Government Community Cloud), if your Microsoft 365 tenant is GCC, please enable this setting to make it work



        • Languages

        Performance Management 365 supports multiple languages, including Czech, Dutch, English, French, German, Hungarian, Italian, Japanese, Polish, Portuguese, Russian, Spanish, Chinese, and Korean as per the browser’s default language. Administrators can enable the Language Selection toggle to allow users to manually choose their preferred language.

        Support for Hebrew and Arabic has been added to enhance multilingual accessibility within PM365.


        When Hebrew is selected as the preferred language, the PM365 interface will automatically update to Hebrew.



        When Arabic is selected as the preferred language, the PM365 interface will automatically update to Arabic.


    • Overall Rating Method: Three methods of calculation are available. Upon completion of the selection review, you must select one of them. then a review Calculation will show accordingly.



    • Acknowledgement: For adding the Acknowledgement click on (+) button.



After adding Acknowledgement click on save button.




  • Brand Logo: Clients/ User could update their company logo which would be displayed on the top left corner of the navigation panel.
