  • Users

      Admin can Add Users , Sync user from M365 and Delta sync from M365.

      From here you can able to add Users.


      To add the Users, follow the below steps:

      1. Click on the +Add User,this displays Add User page.


      2. Enter the Name, Job Title , Department , Location , Date of joining , User's Email ID , Manager's Email ID , Select Role. Click on the Save button.




      Edit: Click the Edit icon , this display Edit page. Update the entries and click on Update button.


      Delete:Click the delete icon, this displays a dialog box. Click Yes button.

  • Categories

      From here you can able to add categories.


      To add the categories, follow the below steps:

      1. Click on the +Categories,this displays Add Category page.


      2. Enter the Category, Weightage and Remarks . Click on the Save button.




      Edit: Click the Edit icon button, this display Edit page. Update the entries and click on Update button.

      Delete:Click the delete button, this displays a dialog box. Click Yes button.

  • KRA (Key Result Area)

      From here you can able to add KRA.


      To add the KRA, follow the below steps:

      1. Click on the +Add KRA.

      2. Enter the Category, Department, KRA, Reference and Location.

      3. Finally click on the Save button.

      4. To add KRA in bulk, click on Bulk KRA. This displays Import KRAs page.


      5. The KRA can be edit or delete:


      Edit: Click the Edit button, this display Edit KRA page. Update the entries and click on Update button.

      Delete:Click the delete icon, this displays a dialog box. Click Yes button.

  • Role Template

      From here you can able to add Role Template.


      To add the Role Template, follow the below steps:

      1. Click on the +Role Template.

      2. Enter the Role Template Name, Description, Job Title.

      3. Select KRA for different categories.

      4. Finally click on the Save button.

      5. The Role Template can be edit or delete :


      Edit: Click the Edit icon, this display Edit Role Template page. Update the entries and click on Update button.

      Delete:Click the delete icon, this displays a dialog box. Click Yes button.
