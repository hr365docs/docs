Must-Read Before Proceeding

Here’s the information on how an app in Microsoft Teams interlinks with SharePoint.

There are two locations in Microsoft Teams where the app can be installed:

  1. Microsoft Teams Personal (Side Navigation)
  2. Microsoft Teams Channel
1. Microsoft Teams Personal App

The Microsoft Teams Personal app is located in the side navigation within Teams, as shown below.


The data for the Microsoft Teams personal app is stored in the root site of SharePoint. If you would like to use the app within SharePoint as well, you’ll need to add it to a web part on the root site page.


For example:

  • • Root Site URL: https://YourTenantName.sharepoint.com
  • • Example Root Site: If your tenant name is "Megazap" your root site URL will be https://megazap.sharepoint.com.



The app is installed on the root site by adding it as a web part on the site page.




If you’ve already installed the app on a SharePoint site other than the root site and want to install it in Microsoft Teams as a Personal app, you can proceed with the installation. However, please note that the data from the SharePoint app will not be reflected in the Microsoft Teams Personal app. This will be a fresh installation, requiring all settings to be reconfigured.


2. MS teams channel app.

If you have already installed the app in a SharePoint site collection other than the root site, completed all configurations, and populated it with data, and you want to use the same setup and data in Microsoft Teams, you can install the app as a Teams channel app. For every Teams SharePoint site, there will be a corresponding channel in Microsoft Teams.


Example: Suppose your Teams SharePoint site is named ICP-365. You can install the app by adding it as a web part on the site page at:

https://megazap.sharepoint.com/SitePages/ICP-365.


To use the same app with the existing configuration and data from the SharePoint site in Microsoft Teams, follow these steps:

  1. Navigate to the Teams workspace that is linked to the SharePoint site (with the same name as the SharePoint site).
  2. In that team’s channel, click the "+" icon at the top to add a new tab.
  3. Select the app from the list, add it to the channel, and save it.

This will enable the app in the Teams channel, utilizing the same configuration and data from the linked SharePoint site.



The Teams app is now installed in the Teams channel. The data for the Teams channel app is stored in its corresponding SharePoint site.




Conclusion:

  • • The data for the Teams Personal app is stored in the SharePoint root site.
  • • The data for the Teams channel app is stored in the corresponding SharePoint site linked to that channel.
  • • An app installed on a SharePoint site other than the root site, with its own settings and data, will not reflect in the Teams Personal app, as it requires a fresh setup.
