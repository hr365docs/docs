  • Home Page

      • The ICP 365 Home Page provides a clean and intuitive interface for administrators to manage job postings. It allows admins to add, edit, delete jobs, assign additional admins, and configure portal settings.


      Key features include:

      Job Dashboard: View all job postings with details such as Title, Description, Start Date, End Date, Years of Experience, Job Type, City, and Active Status.



      Admin Actions: From the dashboard, administrators can:


      • Edit job details in a pre-filled panel

      • Delete postings permanently


        • Add Job: Click the + Add Job button to open a panel and create a new job posting with the following fields:



        • Title: Enter the job title.

        • Footer: Enter the footer text.

        • Description: Provide detailed job information.

        • Portal Logo: Enter the logo URL.

        • Banner Image: Upload a banner image.

        • Theme & Text Color: Select portal theme and text color.

        • Privacy Policy Link: Enter the URL.

        • Privacy Policy Link: Enter the URL.

        • Company Name: Enter the company name.

        • Home Page Link: Provide the URL.

        • Portal Logo Link: Enter the link.

        • Start Date: Select the start date.

        • End Date: Select the end date.

        • Years of Experience: Define required experience.

        • Job Type: Enter or select job type.

        • City: Enter job location.

        • Hide Refer Candidate: Toggle to enable/disable referrals.

        • isActive: Activate/deactivate the posting.

        • Resume Size Limit: Select max resume size.

        Click the Save button to complete the job posting. Once saved, it will appear to employees based on start date, end date, and active status.


        Settings: Configure portal options including theme, SharePoint UI visibility, and user permissions.


        Help: Provides license info, version, subscription details, and support links.
