  • Home Page

      • The ICP 365 Home Page provides a clean and intuitive interface for creating and managing polls within SharePoint. It allows users to collect feedback, run quick surveys, and view responses in real time.


      Key features include:

      Create Poll: Click the + New Poll button to start creating a poll by adding a question, multiple choice options, and selecting an expiration date.



      Configure Poll Settings: After entering the poll details, configure additional settings using the toggles:


      • Allow multiple answers: Lets participants select more than one option when voting.



      • Allow to see results before vote: Enables users to preview poll results before casting their vote.



      • Allow to change vote: Allows participants to update or modify their response after submission.



      • Enable For All Users: Makes the poll available to every user within the site.


      Once all settings are configured, click the Save button to add the new poll.




      Manage Polls: View all existing polls with details such as title, creation date, and end date. You can also perform actions like editing, deleting, or copying polls.



      Copy Poll Link: Generate and copy a unique link to share the poll within the organization. Anyone with access to the link (and who is part of the organization) can vote on the poll.




      Edit Poll: Update the poll title, question, answer options, or expiration date when modifications are required.




      Delete Poll: Remove a poll permanently when it is no longer needed.



      View Poll: Click the eye icon to open and review the poll before voting.



      Submit Response: After reviewing the question, select your preferred option(s) and click Vote to submit your response.




      You can also perform actions like copying, editing or deleting polls.


      Settings: Administrators can configure options such as theme (Light, Dark, Site), response visibility (anonymous or public), and permissions for who can create or vote in polls.


      Help: The Help panel provides license details, version information, subscription validity, and quick links to Support, Privacy Policy, FAQ, Release Notes, Product Activation, and License Validation.
