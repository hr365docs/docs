My Assets

Here users can check the status of all assets assigned by the admin, by clicking the Home tab as shown below image.


By clicking on Assets ID, the user can check all the details of assigning assets by admin as shown in the below image.


QR code for all assets generate automatically in Asset Management, to check or take print of QR code, click on assets, display asset panel will open as shown in below. You can print QR code from any printer and can fix on assets
You can scan QR code from your mobile device using QR scanner app which gives you asset details. You can also setup fields that is to display when we can the QR code. Click on setting gear icon in display asset panel, QR code display fields panel will open select the select box and click on save button. You can select maximum 1 field.
