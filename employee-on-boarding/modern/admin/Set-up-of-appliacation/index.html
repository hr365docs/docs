How to setup an Employee Onboarding application?


    Before moving to start the onboarding or offboarding process of the employee, please set up the Onboarding application.

    1. In System Settings, click on Users and add users from side navigation panel.


    2. In System Settings click on Advance Settings and setup the Date Format, add HR, IT and Admin teams members from side navigation panel.



    3. Add catagories, departments, activity type, standard task, task templates, positions, employee types, location, reason for offboard and notice period as per your requirement under Administration from side navigation.
