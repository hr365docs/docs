Settings

The Visitors Log 365 Settings panel allows administrators to configure global options such as themes and user permissions. These settings help customize the web part’s appearance and control access effectively.

Theme

Select a visual theme for Visitors Log 365:

  • Light: Applies a bright, clean interface.


  • Dark: Optimized for low-light environments.


  • Site: Inherits the theme of the SharePoint site.

Hide SharePoint Settings

The Hide SharePoint Settings option allows administrators to disable the default SharePoint settings menu within the Visitors Log 365 web part. This helps prevent unauthorized modifications and keeps the interface focused only on poll-related configurations.



Users

The Users section allows administrators to manage permissions in Visitors Log 365. The user management grid provides the following functionality:


  • User Grid: Displays Name, Email, and Role (Admin/User) for each entry.


  • + Add: Click the + Add button to register a new user. Administrators can enter the user’s name and assign appropriate permissions or roles.

  • Role Assignment: Administrators can assign or update roles (Admin/User) via the dropdown.


  • Edit: Click the Edit icon to modify an existing user’s details such as name or assigned role. This allows administrators to keep user information accurate and up to date.


  • Delete: Click the Delete icon to permanently remove a user from the Visitors Log 365 user list. This helps maintain a clean and relevant user database.

  • Search & Pagination: Search by Name, Email and adjust the number of entries displayed (10, 25, or 50 per page).
