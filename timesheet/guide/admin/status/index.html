Approval

Project managers can approve the timesheet from Approval tab, select the week then employee, click on item of the table, timesheet table with approve and reject button appears from where you can take the action accordingly

Log Weekly Timesheet As.

By this feature admin and program manager can able to submit timesheet on behalf of the users and by default this feature is disabled, to enable it, go to the settings from side panel under settings, under advance settings tab enable “Time tracking delegation by Admin / PM” row toggle switch.

First click on weekly timesheet tab, log weekly timesheet dropdown appears, select the user from it and select the week number, fill the details in timesheets, and finally click on submit button shown in below image.

Note: same step should follow to submit daily timesheet on behalf of the users.

Task Status.

Project manager can check task status of their employees from Task Status Tab of home page and notify users in case of task deadline date is overdue as shown image.

Here you can filter task status from filters and can exports filtered data in to excel file, just click on export link at right top of the table.

To notify the users about their task status, select the items in action column, click on notify button, popup will display and add the comments and click on submit button, users will get notified through mail

8. Self-Assign Task (or Home page task tab).

Users can create their tasks by their own from task tab as shown in below image, by default it is disabled, admin can enable it from settings from side panel, under settings, under feature tab, enable toggle switch for “Self-Assign Task” row.

Allow self-assign task billable - By default this feature is disabled or billable is No in Add task page as shown in above image, when it is enable users are allowed to create billable tasks, admin can enable it from settings from side panel, under settings, under feature tab, enable toggle switch for “Allow self-assign task billable” row.

Pending weekly timesheet

By default it is disable, Admin can enable it from settings under general tab in “Show pending weekly timesheet from” row. select on from dropdown list and click on save settings button. Then you can able to see it in side panel under timesheet.

Select the filters based on your requirement and click on apply filters then all pending timesheets will be filtered out as shown in below image. If you click on remind button, a reminder mail will be sent to employee.
