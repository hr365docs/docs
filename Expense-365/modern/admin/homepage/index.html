Expense 365
  • Admin Guide

  • Home Page

    • The Expense 365 Home Page provides an overview of all expense-related activities. It consists of two main tabs: All Expenses and Approve Expenses. From here, users can add new expenses, review previously submitted reports, and approve or reject expenses based on their role.



    Key Sections:

    • All Expenses: Displays a list of all submitted expenses with details such as Expense ID, Title, Reported By, Expense Type, Code, Dates, and Amount.


    • Approve Expenses: Provides managers and approvers with pending requests for review and action.


    Add New Expense: To create a new expense record, click on the + Add New Expense button.



    When adding a new expense, users must provide:

    • Title: A clear title for the expense (e.g., Travel, Client Meeting, Equipment).

    • Expense Reported By: Automatically populated with the logged-in user’s name. However, administrators also have the option to submit expenses on behalf of other users.


    • Expense Type & Code: Select the appropriate expense type (e.g., Travel, Food, Office Supplies). The corresponding expense code will be generated automatically.


    • Date: The date of the expense.

    • Amount & Currency: Enter the expense amount and select the appropriate currency.

    • Description: Provide additional details for better clarity.

    • Attachments: Upload receipts or supporting documents by dragging and dropping files.


    Finally, click Save to submit the expense or Save as Draft to store it for later submission.



    View Expense: Click on the Expense ID to view the complete details of the applied expense, including title, type, amount, description, and attachments.



    View Attachments: Inside the expense details page, click on the eye icon to open and review the uploaded documents such as receipts or bills.




    Edit/Delete Expense: Click on the three-dot menu (⋮) next to an expense id to edit the details or delete the record permanently.



    Approve Expenses: Managers or approvers can review submitted expenses under the Approve Expenses tab. For each expense, they can see all details — including Title, Expense Type, Amount, Currency, Dates, and Attachments.

    Available actions:

    • Approve: Accepts the expense, moving it to the approved list.

    • Reject: Declines the expense, with the option to add comments explaining the reason.


    Approval Status: Once an expense is approved or rejected by the approver, the system automatically updates the Status field along with the Approver’s Name. This provides clear tracking of who reviewed the expense and its current state.



    Settings: Administrators can configure themes, manage users, define expense types, and assign approver roles in Expense 365.


    Help: The Help panel provides license details, version information, subscription validity, and quick links to Support, Privacy Policy, FAQ, Release Notes, Product Activation, and License Validation.
