• How to make Custom User profile property searchable ?


    1. Navigate to SharePoint admin center using the below link

      https://tenant.admin.sharepoint.com/_layouts/15/online/AdminHome.aspx


    3. Go to SharePoint Admin > User Profiles > Manage User Properties.



    4. Click on New Property to create new property.





    5. Update profile for one user.


    6. Go to SharePoint Admin > Search > Manage search schema > Crawled Properties.


      7. Please wait until the new property shows up. Have patience as this may take a little while (anything from 15 mins to four hours).

    7. Go to SharePoint Admin > Search > Manage Search Schema > Manage Properties. Create a new managed Properties with the following settings:

      The type of information in this property:Text. Select the check box Searchable, Queryable, Retrievable .


    8. Click Advanced Searchable Settings and select PeopleIdx in the popup window.



    9. Click the blue button Add a Mapping and find the crawled property we just created.
