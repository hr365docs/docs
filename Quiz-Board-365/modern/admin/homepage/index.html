  • Home Page

      • The Quiz 365 Home Page provides a clean and intuitive interface for creating, managing, and participating in quizzes within SharePoint. It enables users to engage with knowledge checks, assessments, and fun quizzes, while administrators can configure and manage quiz settings seamlessly.


      Key features include:

      Create Quiz: Click the + New Quiz button to start creating a new quiz.


      Enter the Title, provide a Description, and add the Questions with multiple-choice options.



      Select the Correct Answer for each question, set the Display Date, and text color.



      Configure additional settings with toggles:

      • Allow users to change their submitted answers: Enables participants to update their responses after submission.

      • Allow users to see quiz responses: Displays aggregated responses for participants.

      • Active: Activates the quiz for participation.

      • Background Attachment: Optionally attach a background for the quiz.

      Once all details are completed, click Save to add the new question to the quiz.



      Manage Quizzes: View all existing quizzes with details such as title, creation date, and status. You can also edit, delete, or duplicate a quiz.



      Edit Quiz: Update quiz details, modify questions, or adjust settings as needed.



      Delete Quiz: Permanently remove a quiz that is no longer needed.



      Card View: Provides a clean, card-based layout of quizzes, showing the title, question, options, and action buttons.




      Attempt Quiz: Users can answer the quiz questions, select their responses, and submit answers for scoring.



      Real-Time Results: Quiz results can be viewed instantly, with charts and insights into performance. If “Allow users to see the responses of the quiz” is enabled, users can view live results (count and percentage) immediately after submitting their answer. If “Allow users to change the submitted answer” is enabled, participants can resubmit their answer after the first submission.



      If this option is disabled, the Submit button is hidden and the ChoiceGroup options are locked once the answer is submitted.


      Settings: Administrators can configure options such as theme (Light, Dark, Site), user permissions (Admin/User roles), and quiz response visibility (public or anonymous).


      Help: The Help panel provides license details, version information, subscription validity, and quick links to Support, Privacy Policy, FAQ, Release Notes, Product Activation, and License Validation.
