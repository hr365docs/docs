  • Home Page

      • The FAQ 365 Home Page provides a clean and intuitive interface for exploring and managing FAQs. Users can view categorized questions, search for specific topics, add new entries, and track engagement through view counts and timestamps. All core features are accessible, except administrative settings, which are reserved for site administrators.


    Edit Categories: Click Edit Categories to create, rename, or organize FAQ categories. This helps group related questions for easier navigation and filtering.



    Add Question: If no questions exist, the interface displays a prompt—“Nothing to see here!”—with a link to Add your first question. Clicking this opens the question editor panel.



    Question Editor: Enter the question, assign it to a category, and format the answer using the rich text editor. Options include bold, italic, lists, alignment, and more. Click Save to publish or Discard to cancel.


    Question List: Once added, questions appear under the All tab with options to re-order, edit, or manage them. Each entry displays the question text and last updated timestamp.
