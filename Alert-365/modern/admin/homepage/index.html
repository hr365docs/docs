  • Home Page

    • The Alert 365 Home Page serves as a centralized dashboard where all created alerts are displayed. Administrators can view alerts at a glance and manage them efficiently from this single interface.


    Create Alert

    Click the + Alert button to create a new alert. A configuration panel opens, allowing administrators to define the alert type, message content, display duration, and activation status.


    Configure Alert Details

    While creating an alert, administrators can configure the message using a rich text editor, set start and end dates to control visibility, and enable or disable the alert using the Active toggle.


    Manage Alerts

    All created alerts are displayed in a structured list, allowing administrators to monitor active and inactive alerts and take appropriate actions when required.


    7. Edit Alert

    Administrators can update existing alerts by selecting the Edit option. The alert configuration panel opens with prefilled values, making it easy to modify alert details.


    Delete Alert

    Alerts that are no longer required can be permanently removed using the Delete option. Once deleted, alerts cannot be restored.


    Alert Display – End User View

    Active alerts are displayed as banner notifications on the SharePoint page. Alerts appear only within the configured date range and when the Active status is enabled.


    Settings

    The Settings section allows administrators to configure global options such as theme selection, hiding SharePoint elements for a clean view, and assigning additional admins or users.


    Help

    The Help section provides access to license details, version information, subscription validity, and quick links to support resources and documentation.
