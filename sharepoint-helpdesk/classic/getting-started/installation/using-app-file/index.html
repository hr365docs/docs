Installation of applications from app catalog

  1. Go to the apps catalog page from the below URL format and click on classic experience options.
    https://YourTenantName.sharepoint.com/sites/apps/_layouts/15/tenantAppCatalog.aspx/manageApps


  2. Click on App for SharePoint as shown in below.


  3. This displays the Apps for SharePoint page.

  4. Click on upload, choose your file and upload the file.



    5. Installation of App file


  5. First open your site collection where you need to install your app, click on site content and then click on Settings gear icon.


  6. You can either click on Add an app or App under New.


  7. This displays the SharePoint apps page.

  8. You will navigate to the page where ADD is visible as shown in below, click on it.


  9. And finally click on Trust it, application starts installing.


  10. Once application got installed you can check in Site contents.
