Installation of applications from SharePoint Store

Prerequisites:

  1. The application can only be installed by a Global Admin or SharePoint Admin by following the steps outlined below.
  2. The application’s features rely on APIs, which can be enabled by a Global Admin with the necessary permissions.

  1. Go to Location (i.e., Site collection or subsite) where you want to install the Applications.

  2. On the home page, click on New and select App.



  3. Click on SharePoint Store and type Apps365 in search box. All applications from Apps365 will show.



  4. Click on application which you want to install.



  5. Click on Switch to the classic SharePoint Store.


  6. Click on ADD IT button



    7. After clicking of Add IT button, if you find Continue button click it, click on purchase button (this will be a 14 days trial. You don’t need to pay or provide payment details.) and return to site contents.



  7. Click on Trust It


  8. Once you will click on Trust It, application will start installing. Please wait for couple of minutes to complete the installation.


  9. Once it gets completes you will able to click on it.
