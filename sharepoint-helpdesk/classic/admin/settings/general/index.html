Helpdesk Classic
  • Admin-Guide

    General

    From here you can update date format, logo, theme etc.


    Date Format

    From here admin can change the date format. The date format settings apply everywhere in the application.

    Theme

    From here, the admin can change the theme of the helpdesk application.


    Teams Option on New Ticket Form

    If you select the dropdown checkbox under Teams Option on New Ticket Form from General, you can view the teams option dropdown on the home page.


    If you select the button checkbox under Teams Option on New Ticket Form from General, you can view the teams option as a button on the home page.

    Home Page Table Sorting


    If you select the Ticket ID checkbox under Home Page Table Sorting from General, you can sort the data according to Ticket ID on the home page.


    If you select the created checkbox under Home Page Table Sorting from General, you can sort the data according to Created Date on the home page.


    If you select the created checkbox under Home Page Table Sorting from General, you can sort the data according to Created Date on the home page.

    Display Resolved Tab on Home Page

    If you enable Display Resolved Tabs in Home Page from General, you can view the resolved tabs on the home page.

    Display Help Page

    If the admin enables this from the general settings, users can see the help option on the side navigation panel.

    Allow All to Comment on Tickets

    In case it is turned on, team members, CCs, consulted individuals, and reviewers are allowed to comment/reply to the tickets.


    SharePoint Site Collection Home Link

    If the admin enables this from the general settings, users can see the sharepoint link option on the side navigation panel.


    Open Tickets in New Tab

    While clicking on ticket links/drill down from data tables, this toggle (if turned on) will allow to open the ticket details in a new tab of the browser.

    Naming Convention

    From here, you can rename the display title, company name, and application title. Click the configure button of the Naming Convention, this displays the Naming Convention page.


    Enter the name and click on save icon.


    Use Teams Code as Ticket Suffix : As a suffix, you may use team code.With the use of a toggle, you may turn this on or off . The team code will be added at the end of the Ticket ID.



Teams Name in New Ticket Page :Enable this to show the teams' name instead of the teams' code on the New Ticket panel or ticket form.



    Recycle Bin

    The deleted tickets will display in recycle bin, the admin can restore them or delete them permanently.


    Brand Logo

    From here admin can add or update the company logo. The updated logo is displayed at the top of the left-side navigation panel. The recommended size of the logo is 150 X 60 with the same aspect ratio and uses a transparent or white background for a better viewing experience.

    Language

    From here admin can change the language of the application.


    Helpdesk can work in Czech, Dutch, English, French, German, Hungarian, Italian, Japanese, Polish,Portuguese, Russian and Spanish as per your browser language. Admin can enable the toggle Language selection as browser default language.
