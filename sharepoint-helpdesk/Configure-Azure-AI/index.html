• Configure-Azure-AI

    1. Go to portal.azure.com


    2. Access Azure's OpenAI service by clicking.


    3. Click on Create button .


    4. Configure Azure's OpenAI service by filling out all the necessary fields, then proceed by clicking on the "Next" button.


    5. Once you've reached the "Review + Submit" window, proceed by clicking on the "Create" button.


    6. Go to Azure OpenAi Studio.


    7. Select "Create New Deployment" to proceed.



    8. First, opt for the "Helpdesk365" model, pick the desired model version, specify the deployment type, and enter a name for the deployment. Afterward, copy the deployment name, and finally, click on the "Create" button.


    9. Access the Resource Management dropdown menu.


    10. Select the key and endpoints. Duplicate the key and opt for the location.


    11. Please navigate to the Helpdesk365 application and access the settings menu.


    12. Please paste the AI endpoint, API key, and deployment name into the designated fields, then click on the "Save" button.
