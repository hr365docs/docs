Multiple Ticket Mailbox Power Automate configuration

    Prerequisites:

  1. The logged-in user must have "Send As" delegate permissions granted for the shared mailbox.
  2. The logged-in user should have SharePoint admin access.

    Resources:

    Shared Mailbox - MultipleMailboxFlow.zip

    Steps to Configure Power Automate Flow:

  1. Navigate to Power Automate and select "My Flows" from the left-hand navigation panel.


  2. Click "Import Package" and upload the "shared mailbox" package from the resources section.



  3. In the "Review Package Content" section, click on the action icon for " HR365 - HD MultipleSharedMailbox."


  4. Choose "Create as New" in the setup dropdown.


  5. In the "Related Resources" section, click the action icon for the first item and select "Create New."


  6. On the redirected page, select the option to create a new connection by clicking on "+ New Connection."


  7. Use the search tool to find the specific resource type, create the connection, and sign in with the logged-in user.


  8. Return to Power Automate, refresh the list, and select the newly established connection.


  9. Repeat steps 5-8 for the remaining resource types and click "Import" when completed.


  10. Select "Open Flow."


  11. Please switch to the old designer by selecting it from the top-right corner.


  12. Choose your team's shared mailbox ID from the dropdown. Note : If the desired mailbox ID is not visible in the dropdown, select the 'Custom Value' option and provide the necessary mailbox ID manually.


  13. Remove any special characters if present.

  14. Use the file icon to select the inbox folder.



  15. For the second step, input your team's shared mailbox ID.


  16. Copy your website URL up to the "SitePages" section and paste it into the value field for the third step.



  17. Input your team's name into the "TeamName" box.



  18. Enter your team's code into the "TeamCode" box.



  19. Save the flow.


    20. Kindly perform the above-mentioned steps for each of the teams listed within your Helpdesk application. After completing these configurations, you can proceed to test the setup by sending an email to the designated shared mailbox ID.

    NOTE:- You are limited to configuring a single flow for each team. To accommodate multiple teams, please import and configure the same flow as needed for the desired number of teams.
