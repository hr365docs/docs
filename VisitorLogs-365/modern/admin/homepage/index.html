  • Home Page

      • The Visitor Logs 365 Home Page offers an intuitive and user-friendly interface to record, manage, and track visitor details securely within your organization. It allows users to easily add visitor information, monitor check-in and check-out times, and maintain a structured log for better visitor management.


      Key features include:

      Add Visitor: Click the + Add button to open the Add Visitor Details form. Enter the visitor’s name, phone number, email, purpose of visit, host name, and check-in/check-out details. This ensures accurate record keeping and streamlined visitor registration.



      View Visitor Records: All visitor entries are displayed in a structured table format showing details such as Title, Phone Number, Email, Purpose, Host Person, Check-In Date/Time, and Check-Out Date/Time. Users can easily filter, sort, and search for specific records.



      Edit Visitor Details: If any information needs to be updated, click on the visitor entry to modify details such as check-out time, purpose, or host information. This helps maintain accuracy in visitor tracking.



      Delete Visitor Entry: Remove outdated or incorrect visitor records by selecting the corresponding entry and clicking the delete option. This keeps the log organized and relevant.



      Settings: Administrators can manage configuration options such as selecting a preferred theme (Light, Dark, or Site), hiding SharePoint default settings for a cleaner interface, and managing user access to the Visitor Logs 365 application.


      Help: The Help section provides access to license information, version details, product activation, and customer support links, ensuring users can quickly find assistance when needed.
