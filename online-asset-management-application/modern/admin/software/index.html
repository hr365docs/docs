  • Software


      By default, the software feature is disabled, the admin can enable it from general settings. Once it is enabled, two software tabs will display on the home page as shown below.


      The first tab is used to sync the O365 License. This sync works on API configuration in general settings. Please find below the link for API configuration. API Permissions

      Click on sync button to sync the O365 license as shown in the below image.



      The second software tab is used to add software, you can add software with quantity and then you can assign it to the users.



      Assign and return software.
      In the action column, there are assign, return icons from where you can assign and return the software., If you want to edit the software, click on the pencil icon and delete from the delete icon.
