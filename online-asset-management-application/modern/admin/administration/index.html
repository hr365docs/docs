      1. New Assets - From here admin can add the new assets. To add the assets follow add asset.

  • Add Status

      From here you can able to add status.

      4. To add multiple status, click on Save and Add more button.

      Edit: Click the Edit Asset status button, this display Edit Asset Type page. Update the entries and click on Update button.

  • Sub Category

      In the Administration section, you will find the Sub-Category option. Click on it to proceed.


      In the Sub-Category section, there are two levels available: Sub-Category Level 1 and Sub-Category Level 2.


      When you click on Sub-Category Level 1, a panel will open where you can fill in the data according to the fields. After filling in the data, click on the Save button to save the information.


      First, you need to select the Category. After selecting the category, the items for Sub-Category Level 1 and Sub-Category Level 2 will be displayed according to the selected category level. You can then select the items from Sub-Category Level 1 and Sub-Category Level 2 as per your requirements.


      In Category Level 1, you can perform bulk imports by uploading a document file while following the provided instruction rules. Additionally, a download button is available, allowing you to download the required template or file for your convenience.

  • Sub Location

    In the Administration section, you will find the option for Sub Locations. Click on this option to manage or view sub-locations within the system.


    After that, click on the Add Sub Location button, enter the Location and Sub Location details, and then click the Save button. Your sub-location will be successfully added.


    This feature allows you to see the Sub Location field while adding an asset, where you need to provide the sub-location value.

  • Pickup Location

      From here, admin can add, edit or delete locations.

      To add pickup location, follow the below steps:

      1. Click on the +Add, this displays Add Pickup Location page.

      2. Enter the Pickup Address, State,City, Zip Code, Country, Email Id(s) and Mobile Number.


      3. Click on the Submit button.

      To edit, select the Action and click on the Edit button. Update the entires and click on Submit button.


      To delete, select the Action and click on the Delete button.
