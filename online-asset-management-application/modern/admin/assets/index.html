  • Add Assets

      Here assets could be added. Enabling the fields that are required or disabling those that are not required in the add assets page.

      To add the asset type, follow the below steps:

      1. Click on the +Asset.


      2. Fill in mandatory fields like Asset name, Category, Asset type, and others are optional fields, upload an image or add the document. Enable the toggle switch to set Asset Expiry Notifications for assets.

        Mandatory fields like Asset name, Asset type, and Category would not be able to be disabled.


      3. Cloning-Item Click on ok button.


      4. Click on the Save button.


      5. By Clicking on Save and Assign Button We can Assign the Asset Immediately.


      6. The created asset will be displayed on the Home Page and assigned asset table.

      To generate a system barcode please enable “System Generated Barcode” from the general setting.

  • Import asset or Bulk Assets

      Clients could add 1000 records in one go using CSV file. Download the sample file and add data in the specified format.


      Follow the below instructions before adding bulk assets:

      1. Please download the sample file and fill in the assets details as per the below instructions, save the file to upload.
      2. Please add a category, subcategory, asset type, model number, and vendor through administration before uploading the bulk asset file.
      3. Remove separators and currency symbols from the cost/value.
      4. Please remove commas from notes or any other fields to avoid conflicts with commas in CSV file.
      5. Date format should be MM/DD/YYYY, this is only during upload the data, while displaying it will follow as per your choice in settings.
      6. Mandatory fields like asset name, asset id, asset type, and category should not be empty.
      7. Please don't change the order of the columns or add new columns, this may lead to feeding values in incorrect columns.
      8. You can add 1000 assets in one go from import assets.


      From here you can update the bulk asset details.

  • Assign Assets

      Here you could assign assets. To add the asset type, follow the below steps:

      1. Click on the Assign Assets from the top navigation panel.



      2. Click on the link under the action column. The panel would be displayed

      3. Enter the Username, select Project, and Return date.


      4. Finally, click on the Assign button.

      5. The assigned assent will be displayed on the Return Assets page.

  • Return Assets

      From here you can able to return assets. To add the asset type, follow the below steps:

      1. Click on the Return Assets from the top navigation panel.


      2. Click on the link under the action column. This would display a panel.


      3. Finally, click on the Return button.

  • Book-Assets

    From here you can able to Book assets. To add the Book type, follow the below steps:

    1. Click on the Book Assets from the top navigation panel.


    2. Click on the link under the action column. This would display a panel.

  • Booking Details

      On the Booking Details page, you can find all the booked assets.

  • Approve/Reject Asset

      By default, Approve/Reject Asset feature is not enabled, the admin could enable it from the Users tab by navigating through the side panel under settings.


      Approve/Reject of assets could be done. Click on the Approve and Reject symbol to approve and reject the assets.
