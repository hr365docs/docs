    User, Roles And Permissions

    From here, admin can assign roles to users and teams.


    Importance of Roles:

    Admin:
    • Admin has full control.
    • Able to do all settings.
    • Can view dashboard.
    • Admin can allow manager to add assets.
    • Admin can allow manager to delete assets.
    • Admin can allow users to select approver.
    • Admin can allow users to book assets.
    • Can generate reports of all assets.


    Asset Manager:
    • Asset Manager Able to add, delete, assign, and return assets, audit assets, report assets, and record assets. book assets can be edited or deleted.


    Asset Location Manager:
    • Asset location Manager can perform the same functions as asset manager do Able to add, delete, assign, and return assets, audit assets, report assets, and record assets. and book assets can be edited or deleted, but only for assets that share the same location.


    No Access:
    • The User cannot Access the Application.


    Power User:
    • Able to see all the Assets Only.
    • Power user can Book the Assets.


    Supervisor:
    • Supervisor role entails the ability to allocate assets, retrieve assets, conduct asset audits, generate asset reports, manage book assets, make modifications to or remove book assets.


    User:
    • User role is limited to viewing assets assigned exclusively to them.

    Users

    Admin can add users to this application.


    Admin can add users to this application. There are different types of roles like the user, admin, asset manager, supervisor, power user, asset location manager and no access as shown below.


    Asset Manager to Add Assets

    Enabling this would allow manager to add Assets.

    Asset Manager to Delete Assets

    Enabling this would allow manager to delete Assets.

    Power User to Book Assets

    If this is enabled, Power Users can Book Assets.

    Allow Power User to Select Approver to Book Assets

    If this is enabled, Power Users can select approver.

    Allow Users to Return Assets

    If this is enabled, Power Users can return asset.
