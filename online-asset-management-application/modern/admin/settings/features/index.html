Asset Management 365
  • Admin Guide

    Features

    From here you can customize the features of AM application.

    Depreciation Methods

    To Add Depreciation Settings Click on Configure.


    Click ok Add Button.



    Complete the columns and then save your changes by clicking the "Save" button.

    Default camera scanner for edit assets

    In the default camera scanner for editing assets, you can choose between QR code, barcode, or both. This flexibility makes it easier to locate and edit assets efficiently.

    Depreciation Settings

    In the depreciation settings, clicking on Configure opens a panel displaying details such as category, depreciation method, period, cost, residual cost, action, and more.


    You can customize depreciation settings by clicking the "Add" button. Fill in all the required fields, then click "Save" to apply your changes.

    Configure Barcode Field(s)

    Here You can add Configure-Barcode-Field(s).

    Click on Configure to process continue.


    Select the field and click on Save icon.

    System Generated Barcode

    When the toggle is enabled, the system automatically generates a barcode each time a new asset is added.

    External User

    If enabled, you can add external or guest users to the application and assign assets to them.

    Retain Asset's Last Location

    When this feature is enabled, the user's location will remain assigned to the assets after they are returned. You can view the retained location in the asset's "User Location" field.

    Assign Assets to Location

    If enabled, the location option will appear in the assignment pop-up, allowing you to assign assets to a specific location.

    Assign Assets to Non M365 Users

    If enabled, the option to assign assets to non-M365 users will appear in the assignment pop-up, allowing you to assign assets to non-M365 users.

    Advance Booking of Assets

    If you disable the Assets Advance Booking feature, it will be removed from the application.

    Enable Software

    If enabled, software tabs will appear on the home page. One tab is used to sync O365 licenses, and the other allows you to manually add software and assign assets to users.

    Hide the Assign and Return Assets in Home Page

    When this toggle is enabled, you can hide the "Assign" and "Return Assets" options on the Home Page.

    Hide/Show Thumbnail

    When you enable this toggle, you can choose to hide or show the thumbnail.

    Quick Action

    Assets can be tracked by scanning them with a barcode handheld device using the quick action feature.


    Click the Quick Action tab ,also Assign and Return the asset respectively

    Show Only User Asset in Asset Acknowledgement

    Toggling it on will display the user's assets in the asset acknowledgment section.


    Hide And Show Dashboard

    When you toggle it on, the dashboard will be displayed. If toggled off, the dashboard will be hidden.


    GCC Tenant

    If your Microsoft 365 tenant is GCC (US Government Community Cloud), please enable this setting for it to function properly.

    Asset's Location

    When you turn on this toggle, it will become mandatory to specify the asset location.

    Camera Scanner

    The Camera Scanner tab appears on the home page, allowing you to scan asset barcodes for searching.

    Make Category and Subcategory Static

    When you turn it on, the category and subcategory sections on the home page will become static.

    Inventory Module

    If this is enabled, Inventory tabs will displays on the home page.


    Allow user to Request Inventory : With this feature, users can submit requests for inventory items.


    Allow Shipping of Inventory :When enabled, users can request the shipping of consumable items.


    Inventory Module Title Text

    Customize your title as desired and then save it to update the title of the inventory module.

    Maintenance Module

    If this is enabled, you can add maintenance frequency and maintenance due date while adding assets.


    Maintenance Module Title Text : From here you can rename module title.


    Advance Booking Filter

    In the advance booking filter, clicking the + icon will display the options for category, subcategory, and status. The attributes you select here will be the only ones available to users for booking.

    Asset Acknowledgement

    If you type the acknowledgement text, user can see the acknowledgement page (the acknowledgement page link is available in the assigned email notification of the user).


    Create PDF of asset acknowledgment: When you enable this and set up power automate, pdf file would be created and stored in SP library moreover send it to user who has acknowledged the assets.
