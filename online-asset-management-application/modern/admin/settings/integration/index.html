      Integration

      From here you can Integrate Asset management application with other application.

      Intune

      To sync Office 365 software and integrate Microsoft Intune with Asset Management 365, please configure the web API as per the instructions. Select Software & MS Intune Integration, then click on the Full Sync button as shown below. You can view synced Intune assets on the Return Assets page or by clicking on the Return tab on the home page.


      Full Sync

      Click the Full Sync button to synchronize the Intune assets. Ensure that the Graph API is approved.
      Note: Microsoft allows syncing Intune assets only for users with the Global Admin or Intune Administrator roles. Therefore, the Asset Management 365 admin must hold one of these roles to utilize this feature.


      Daily Sync

      The app automatically performs a sync every day when the admin opens it. This updates Intune asset details if any changes exist. If needed, you can also trigger a manual sync by clicking the button.

      Intune Integration Tooltips: Added helpful tooltips for Intune Integration modules to improve usability. Hover over the info icon to see how the new tooltips work.



      Update User Details: The system automatically synchronizes user information from Azure Active Directory (Azure AD) to ensure the Users list always reflects the most up-to-date details.


      When viewing an asset, the User Department and User Location are shown as the first two columns, providing quick visibility into key user attributes.


      After completing the integration and sync, devices enrolled in Microsoft Intune are now successfully imported and displayed within Asset Management 365. The examples below illustrate how Intune-managed devices and their details appear once the sync is complete.

      The Intune Admin Center overview shows devices grouped by platform (Windows, Android, iOS/iPadOS, macOS, Linux). This confirms which devices are actively managed in Intune.


      The All Devices view displays individual device details such as ownership, compliance status, operating system, OS version, and last check-in. This information is used by Asset Management 365 during sync.


      The Asset Management 365 module now reflects these synced Intune devices, allowing admins to see both user details and compliance information in one integrated system.

      Intune Exclude Filters

      Asset Management 365 now supports MS Intune Exclude Filters. These filters work only when MS Intune integration is successful. To configure, click on the accordion (+) icon and youâ€™ll see the following options:



      1. Exclude by Platforms: Select from available options and click the checkbox: Windows, iOS/iPad, macOS, Android.


      2. Exclude by Compliance Status: Filter devices based on compliance states: Compliant, Non-compliant, Not Evaluated.


      3. Exclude by Ownership Type: Choose ownership types to exclude: Personal, Corporate, Unknown.


      4. Exclude Virtual Machines (VMs): Toggle this option ON if you want to exclude VMs from syncing.


      MS Intune Filter

      Can sync assets based on the applied filter from the Integration settings.

      By using these filters only filtered data will be pulled from MS-intunes, filter by platform , filter compliance by status, filter ownership by type, also exclude VM's by enabling the toogle.


      Customized Platform Title

      User can Customize the platform title here, then click "Save".


      SCCM Integration
