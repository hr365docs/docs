  • Inventory

      Inventory (also known as Inventory goods, non-durable goods, or soft goods) are goods that are intended to be consumed. Inventory are products that consumers use recurrently, i.e., items which "get used up" or discarded.

      By default, the Inventory feature is disabled, admin can enable it from Features Settings.


      The renaming of the module title, could be done from Features Settings.

  • Add Inventory Item

      From here you can able to add Inventory item.


      To add the Inventory item, follow the below steps:

      1. Click on the + Inventory , this displays the Inventory page.



      2. Enter all the mandatory fields like Product Id, SKU ID, Item Name, Status, Item Description, Category, Sub Category, Quantity and Total Price. You can add multiple rows for category and delete the row by clicking on the delete icon.
      3. Finally, click on the Save button.

      4. The added Inventory item will be displayed on the Inventory Items page.



  • Bulk Inventory

      You can add 1000 Inventory in one go using CSV file. Download the sample file and add data in the specified format.



      Follow the below instructions before adding bulk Inventory items:

      1. Please download the above sample file and fill the Inventory details as per below instructions, save the file and upload it
      2. Please add variants and subvariants through administration before uploading the bulk Inventory file.
      3. Please remove the separators, and currency symbol from the cost column.
      4. Please don't add commas in any columns of the file.
      5. The date format should be MM/DD/YYYY for data upload, while its display format can be adjusted in your date settings.
      6. Please ensure the mandatory columns such as SKU ID, Item Name, Status, Location and Product ID are filled out without any emptiness.
      7. Do not alter the column order or add new columns, as this may cause data to be entered incorrectly.
      8. You can add 1000 items in a single upload.

      To add or upload bulk Inventory follow the below steps:

      1. Click on the +Bulk Inventory tab, this displays Add Multiple Inventory Items.
      2. Choose the data loaded CSV file from the system and upload the file.
      3. Finally click on Save button.
