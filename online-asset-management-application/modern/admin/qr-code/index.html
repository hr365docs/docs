  • QR Code


      QR code for all assets generates automatically in Asset Management, to check or take print of QR code, click on assets, and display asset panel will open as shown below. You can print QR code from any printer and can fix assets.


      You can scan QR code from your mobile device using QR scanner app which gives you asset details. You can also setup fields that is to display when we scan the QR code. Click on setting gear icon in display asset panel, QR code display fields panel will open select the select box and click on save button. You can select maximum 1 field.
