      By default, the software feature is disabled, the admin can enable it from general settings. Once it is enabled, two software tabs will display on the home page as shown below.


      The first tab is used to sync the O365 License. This sync works on API configuration in general settings. Please find below the link for API configuration.
      https://www.hr365.us/wp-content/uploads/2021/08/Azure-API-configuration-for-Asset-Management-application..pdf

      Click on sync icons to sync the O365 license as shown in the below image.


      Further to see the users to whom licenses are assigned, click on the item link, and the license Details panel will display then click on the full sync tab as shown below.


      you cannot assign or return O365 Licenses in Asset Management.

      The second software tab is used to add software, you can add software with quantity and then you can assign it to the users.



      Assign and return software.
      In the action column, there are assign, return icons from where you can assign and return the software., If you want to edit the software, click on the pencil icon and delete from the delete icon.

  • Bulk Software


      By default, the software feature is disabled, the admin can enable it from general settings. Once it is enabled, two software tabs will display on the home page as shown below.


      You can add 1000 Software's in one go from import software. Download the sample file and add data in the specified format.


    • Follow the below instructions before adding bulk assets:

    1. Please download the above sample file and fill the software details as per below instructions, save the file and upload the file.
    2. Remove separators, currency symbol from the cost / value.
    3. Please remove commas from the note or any other fields to avoid conflicts with commas of CSV file.
    4. Date format should be MM/DD/YY, this is upload the data, while aplying it will follow as per your choice in settings.
    5. Mandatory fields like Software Name, quantity, price per unit,license type, recurring, renewal, renewal frequency, status should not be empty.
    6. Please don't change order of the columns or add new columns, this may lead into feeding values in the incorrect columns.
    7. You can add 1000 Software's in one go from import software.

    Choose the data loaded CSV file from the system and upload the file.
