  • Consumable

      Consumables (also known as consumable goods, non-durable goods, or soft goods) are goods that are intended to be consumed.
      Consumables are products that consumers use recurrently, i.e., items which "get used up" or discarded.

      By default, consumable feature is disabled, admin can enable it from General Settings. Once it enabled, four consumable tabs will display in home page as shown in below.


      If you want to rename the module title, you can do it from General Settings.

      First tab is used to add consumable items in bulk. Refer Bulk Consumable.

      Second tab is used to add consumable items in bulk. Refer Add Consumable.

      Third tab is used to place order, edit or delete consumable items in bulk. Refer Consumable Item.

      Fourth tab is used to request order of consumable items. Refer Request Consumable Item.

  • Add Consumable Item

      From here you can able to add consumable item.


      To add the consumable item, follow the below steps:

      1. Click on the +Consumable Item, this displays Consumable page.


      2. Enter all the mandatory fields like Product Id, SKU ID, Item Name, Status, Item Description, Category, Sub Category, Quantity, Total Price. You can add multiple rows for category and delete the row by clicking on the delete icon.
      3. Finally click on the Save button.

      4. The added consumable item will be displayed on the Consumable Item page.

  • Bulk Consumable

      You can add 1000 Consumable in one go using CSV file. Download the sample file and add data as specified format.


      Follow the below instructions before adding bulk consumable items:

      1. Add product ID in specified column and is mandatory.
      2. Remove separators from the cost / value.
      3. Use date format should be MM/DD/YYYY, this is uploading the data, while displaying it will follow as per your choice in settings.
      4. Please do not change order of the columns or add new columns, this may lead into feeding values in incorrect columns.
      5. Mandatory fields like Asset Name, Asset Id, Asset Type and Category should not be empty.
      6. Please do not change order of the columns or add new columns, this may lead into feeding values in incorrect columns.
      7. Barcode should not be more than 8 digits.

      To add the bulk consumable follow the below steps:

      1. Click on +Bulk Consumable tab, this displays Add Multiple Consumable Items.
      2. Choose the data loaded CSV file from system and upload the file.
      3. Finally click on Save button.

  • Consumable Item

      From here you can able to edit, delete, place order of consumable item.


      To place order of consumable item, follow the below steps:

      1. Click on the link of SKU ID, this displays popup.


      2. Select the category, sub category and pickup/ship.


      3. Finally, click on the Place Order button.


      4. The added consumable item will be displayed on the Approve/Reject Consumable page.


      Edit: You can edit the details of consumable item by clicking on edit icon under Action. Update the fileds and click on Update button.


      Delete: You can delete the details of consumable item by clicking on delete icon under Action.

  • Request Consumable

      From here you can able to place order consumable item.


      To request consumable, follow the below steps:

      1. Click on the Request under Action, this displays place order popup.


      2. Select the category, sub category and pickup/ship.

      3. Finally, click on the Place Order button.


      4. The added consumable item will be displayed on the Approve/Reject Consumable page.

  • Approve/Reject Consumable

      By default, Approve/Reject Asset feature is not enabled, admin can enable it from Users from side panel under settings.


      From here you can able to Aprrove/Reject asset. Select the checkbox under Action for booked asset and select Approve or Reject button.
