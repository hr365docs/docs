My Assets

Here users can check the status of all assets assigned by the admin, by clicking the Home tab as shown below image.


By clicking on Assets ID, the user can check all the details of assigning assets by admin as shown in the below image.


QR code for all assets generate automatically in Asset Management, to check or take print of QR code, click on assets, display asset panel will open as shown in below. You can print QR code from any printer and can fix on assets
You can scan QR code from your mobile device using QR scanner app which gives you asset details. You can also setup fields that is to display when we can the QR code. Click on setting gear icon in display asset panel, QR code display fields panel will open select the select box and click on save button. You can select maximum 1 field.


Book Assets

Here Users can check and request new assets.


By default, Book assets feature is enabled, admin can disable it from general settings from side panel under settings. To disabled book assets feature, in general settings disable the toggle switch for “Assets Advance Booking”. To book the assets to the users in advance click on Book Asset tab, click on book assets of action, pop up will appear, select the user, assign date, and return date and finally click on Yes book it button. Asset booking will work on assign date and return date of the assets

Booking Details

In case if you want to cancel the booked asset or to edit the booked assets, you can do it from Booking Details.


To cancel the booked asset just click on the cancel of action column and to edit the booked assets (assign date or return date, click on update under of action column, pop up will open change the date and click on Yes book it button. On the Booking Details page, you can find all the booked assets as shown below image.
