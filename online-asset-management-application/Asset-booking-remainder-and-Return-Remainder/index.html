    1. Assets 365 can be configured using either a shared mailbox or a service account mailbox (standard user mailbox). Note: When using a shared mailbox, ensure the logged-in user has the required access. For a service account, sign in using an account with the appropriate permissions.
    2. The logged-in user must have Send As (delegate) permission for the shared mailbox.

    • 4. After successfully creating the shared mailbox, locate it, select it, and scroll down to access Manage mailbox permissions.



    • 5. Grant the logged-in user access to all three mailbox permissions.
