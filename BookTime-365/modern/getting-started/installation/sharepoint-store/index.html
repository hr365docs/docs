  • Installation of applications from SharePoint Store

    Prerequisites:

    1. The application can only be installed by a Global Admin or SharePoint Admin by following the steps outlined below.
    2. The application’s features rely on APIs, which can be enabled by a Global Admin with the necessary permissions.

    1. Navigate to the app catalog page using the following URL format:
      https://yourtenantname.sharepoint.com/sites/appcatalog/_layouts/15/tenantAppCatalog.aspx/manageApps

      Replace yourtenantname.sharepoint.com with your actual tenant name, for example: cubiclogics.sharepoint.com.
      Once you are on the page, click on the "Classic Experience" options.



    2. Click on setting gear icon and select on Add an app.



    3. Click on New and select on App.



    4. Click on SharePoint Store and type Apps365 in search box. All applications from Apps365 will sort



    5. Click on application which you want to install.



    6. Click on Add to Apps site.


    7. Select the checkbox (Enable this app and add it to all sites ) and click on Add button.

      8. Now you can make this solution or application package available for all sites in the organization, once the application is installed, later you can remove the app from all sites.

      By performing these actions, the selected application package will be downloaded from the Microsoft SharePoint Store into your tenant.


    8. Click on Go to API access page or i will do it later button.


    9. On SharePoint admin center page,From left panel under advanced click on API access option under pending request, there are requests ( Microsoft Graph, Presence.Read.All, User.Read.All, MailboxSettings.Read, Calendars.Read, Group.Read.All, Chat.ReadWrite, Chat.Create, Mail.Read.Shared) select and click on approve one by one for all the requests.



    10. The user can select and approve individual permissions, then proceed further.


    11. Go to Location (i.e., Site collection or subsite) where you want to install the BookTime 365 application.

      12. We have to add BookTime 365 as the web part on the page.


    12. Now click on + icon and then select the BookTime 365 as shown in below.



    13. Finally click on Publish button.


    14. Finally, refresh the page.

    How to remove this solution available to all sites in the organization


    1. Now select the BookTime 365, click on files and then the deploy icon.


    2. Now unselect the check box and finally click on deploy button.

    There are two types of installations:

    1. Installation via package upload in the app catalog site.
    2. Installation from the SharePoint store (AppSource).

    If you delete the package installation and reinstall the app from the SharePoint store, or vice versa, it will not impact or result in data loss. This is because the application's data is stored safely in SharePoint lists, which remain unaffected by the installation method.
