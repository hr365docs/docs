General Settings

The General Settings in BookTime 365 allow administrators to configure key preferences that personalize the user experience and align with organizational standards. These options include branding, formatting, localization, and platform-specific settings.


    Date Format

    Customize the format in which dates appear, such as MM-DD-YY, YY-MM-DD, or other regional styles, for localized display across the platform.



    Enable Custom Time Zone

    Enable the toggle to set a custom time zone for your workspace. By default, the time zone is set to UTC, but you can change it to your preferred time zone to ensure all bookings and availability reflect accurate regional time.




    Theme

    Choose from available visual themes such as light mode, dark mode, or custom site themes to match your organizational branding and preferences.



    GCC Tenant

    Enable compatibility with Microsoft GCC (Government Community Cloud) tenants to ensure compliance with U.S. government cloud standards and environments.



    Favicon

    Upload and enable your custom favicon to appear in browser tabs and supported UI locations, reinforcing brand identity.




    Language

    BookTime 365 supports multiple languages to help users operate the application in their preferred or regional language.

    Default Browser Language

    Enable this toggle to display the application using the browser’s default language setting. This ensures a seamless user experience by automatically applying the user's preferred language.



    Select Language

    Manually select a specific language from the list of supported options. This overrides the browser default and enforces a consistent language for all users regardless of their browser settings.



    Hide SharePoint Page

    Configure this setting to hide the SharePoint page components from both the site navigation and site contents. This helps maintain a clean, focused interface by preventing users from accessing or seeing unnecessary default SharePoint elements.
