    Features

    This section allows you to configure optional client-side functionalities. These features enhance the booking experience and provide flexibility for staff and clients.


    Custom Columns

    Configure this section to add custom columns to your booking form.


    Click the +Add button to create a new custom column. These fields will be visible on the customer form for capturing extra information.


    Allow Customer to Select Services

    Enable this toggle to allow clients to choose services when booking through the portal.


    Allow Customer to Select Staff Member

    Allow clients to select specific staff members during appointment booking from www.cubic.booktime365.com.


    Fetch Staff Calendar

    Enabling this option shows real-time availability for staff members during bookings.


    Setting Audit Logs

    Enable audit logs to record and track system changes and user activity for compliance and security monitoring.
