Settings

The NoticeBoard 365 Settings panel allows administrators to configure key components to customize the web partâ€™s appearance, manage content organization, and control access.

Theme

Select a visual theme for NoticeBoard 365:

  • Light: Bright, clean interface.
  • Dark: Optimized for low-light environments.
  • Site: Matches the SharePoint site theme.

Category

The Category section allows administrators to view, create, and manage post categories to keep content organized. Existing categories are displayed in a list with their associated icons.


To add a new category, click the + Add button and fill in the category name.


Choose an icon to represent the category from the available options.


Click Add. The newly created category will now appear on the NoticeBoard 365 home page.



Categories can be managed after creation. You can Edit the name or icon, or Delete the category if it is no longer needed.


Users

Manage users and permissions in NoticeBoard 365. Key functionalities include:


  • User Grid: Displays Name, Email, and Role (Admin/User) for each user.


  • Add User : Click on + add to add new user.


  • Role Assignment: Assign or update user roles via dropdown.


  • Search & Pagination: Search by Name or Email, and adjust the number of entries per page.


Hide SharePoint Componenets

This option hides the default SharePoint settings menu within the NoticeBoard 365 web part, preventing unauthorized modifications and keeping the interface focused on posts and polls.
