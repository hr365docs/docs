  • Home Page

      • The NoticeBoard 365 Home Page offers a clean, intuitive interface where users can browse, create, and manage posts. Each post can include a title, description, category, attachments, thumbnail, and related links, providing context and easy access to information.


      Key Features:

      • Tab Navigation: Seamlessly switch between All Posts and My Posts to streamline browsing.



      • Create a Post: Click the + Post button to initiate a new post submission.


      • Post Content: Provide a Title and Description to define the post’s purpose and information.


      • Category Selection: Assign an appropriate category to organize posts effectively.


      • Attachments: Upload relevant files or documents to complement the post content.


      • Thumbnail: Enhance post visibility by adding a thumbnail via From Site, Upload, or Link, then click Upload.



      • Related Links: Include supporting links to provide additional context or resources.


      • Publish Post: Once all details are complete, click Add to make the post live.


        • Users can express their feedback on a post by clicking Like or Dislike.


        Clicking the Share icon copies a direct link to the post. This link can then be shared anywhere within the organization for easy access.



        Actions

        Users can manage posts using the following options:


        View Details: Open the full post to see all content, attachments, and related links.


        Edit: Modify the post title, description, attachments, or other details.


        Delete: Permanently remove the post from the board.

      • Search Functionality: Quickly locate posts by title, description, or category using the search bar.


      Settings: Administrators can configure the NoticeBoard by selecting a Theme (Light, Dark, or Site Default), managing Users and their permissions, enabling or disabling Hide SharePoint mode, and defining available Categories for posts.

      Help: The Help panel provides license details, version information, subscription validity, and quick links to Support, Privacy Policy, FAQ, Release Notes, Product Activation, and License Validation.
