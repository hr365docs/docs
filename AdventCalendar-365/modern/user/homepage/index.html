  • Home Page

          • The Advent Calendar 365 Home Page provides a clean, organized, and visually interactive interface where each numbered tile represents a day or event. Users can easily identify which doors are open, upcoming, or yet to unlock based on the configured schedule.


          From the home page, users can:

          • View Calendar Layout: The page displays all calendar doors in a grid format, showing sequential numbers for each day or event.


          • Identify Open Days: Doors that are available to open today or previously are highlighted or visually marked, making them easy to identify.


          • Open a Door: Click on a door that is open for the current date to reveal its assigned content — such as an image, video, or message.


          This homepage serves as the main interactive space for both viewing and managing the calendar’s daily content. It provides an engaging visual experience that encourages users to return each day to reveal new surprises.
