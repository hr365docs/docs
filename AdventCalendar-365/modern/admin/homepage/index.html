  • Home Page

      • The Advent Calendar 365 Home Page provides a visually appealing, grid-based layout where each tile represents a day in the countdown. Each numbered tile acts as a “door” that can be configured to display unique content — such as text messages, images, or videos — that reveal as the event progresses.


      Key features include:

      Create or Edit Calendar: Click the Edit icon on any calendar tile (door) to open the configuration panel. From there, you can customize what content appears when the door opens.



      Configure Door Content: Each calendar door can display custom content like images, videos, or messages. You can easily update the content by selecting Add Content Image.



      Add Content Image: When you click the Add Content Image button, a pop-up panel allows you to insert media using multiple options:

      • Recent: Choose from recently used images within your SharePoint environment.

      • Site: Browse and select images stored in your SharePoint site libraries.

      • Upload: Upload new images directly from your device.

      • From a Link: Insert an image or video via a web URL.


      Content: Please add the relevant content here.


      Button and Link Configuration: Add a custom button label (e.g., “Open Gift,” “View Message,” or “Learn More”) and link it to any internal or external page for additional interactivity.



      Content Visibility: Set the date when each door becomes visible using the Content visible from field. The calendar automatically unlocks each door based on the configured schedule, providing a dynamic day-by-day reveal experience.



      Background Customization: Choose how each door’s background appears by selecting from the following options:

      • Transparent: No background, allowing the site’s color or image to show through.

      • Color: Apply a solid background color using the color picker.

      • Image: Use a custom image as the background for enhanced visual appeal.


      Save Configuration: Once all settings are defined, click the Save button to apply your updates. The door will now display the configured content when opened by users on or after the set date.



      View Calendar: As users access the Advent Calendar, doors that have reached their visibility date will appear active. Clicking a door reveals its hidden content, such as a festive image, quote, or link to a SharePoint resource.




      Manage or Update Calendar: You can revisit and edit any door configuration at any time. Simply click the edit icon to modify content, background, or visibility settings.



      Settings: Administrators can configure global settings such as calendar theme (Light, Dark, or Site), visibility rules, and permission controls for who can view or edit the calendar content.



      Help: The Help panel provides license details, version information, subscription validity, and quick links to Support, Privacy Policy, FAQ, Release Notes, Product Activation, and License Validation.
