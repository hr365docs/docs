Home page

In home page you can see various tabs like status, my tasks, weekly timesheets, approval task status and Log Weekly timesheets. As feature as shown in below image.


  1. Weekly timesheets

    1. Setup

      2. Admin can setup or customize weekly timesheets as per their requirement before going to use weekly timesheet. Please look into below points.

      By default all features are disabled you can enable and setup it based on your requirement.

      You can click on finally save setting buttons after making setup in each tab of setting page.

      Time Tracking Days:Here you can block weekend days: Please go to settings from side panel under advance setting tab, you can uncheck days in Time Tracking Days row then click on save button and finally click on save settings button.

      Restrict daily and weekly hours:Here you can setup max daily hour and max weekly hour due to which user not able to put time beyond this limit in their weekly time sheet. Go to settings under features tab enable the Restrict daily and weekly hours row toggle and put restrict time for daily or weekly and click on save icon and then click on save settings button.

      Activities:If it enabled, activity column in weekly timesheet is displayed from where user can select activity in their weekly timesheet. To enable it go to settings side panel under settings from, under general tab enable “Display Activities” row toggle switch and then click save setting button.

      Work Place:If it enabled, work place column in weekly timesheet is displayed from where user can select work place in their weekly timesheet. This play important role because some project billing is made based on onsite rate. To enable it to go to settings from side panel under settings, under feature tab enable “Work Place” row toggle switch, work place popup will display, you can add some more work places, click on submit button of popup and then save setting button.

      Do not edit or remove on-site word in work place popup otherwise onsite billing would not work.

      Mandatory task selection while tracking time:By default this feature is enabled, if you want to submit timesheet without task you can do it just disable it, go to settings from side panel under settings, under features tab, disable “Mandatory task selection while tracking time” row toggle switch and then click on save setting button.

    2. Weekly Timesheets Submitting

      3. Here you can submit timesheet for past week or on last working day of current week from weekly timesheet tab or from side panel under timesheets. To submit it, first select week number, select project selected project will show in task column dropdown, select task, enter times and add comment in comment icon and finally click on submit button as shown in below image.

    3. Weekly Timesheet Submitting for Multiple Projects or Tasks

      4. You are able to submit timesheets for multiple projects or multiple tasks on which you has been worked using Add button as shown in below image.

    4. Save Timesheets

      5. Here you are able to save timesheet and can submit it later, just click on save button, it will display in drafts as shown in below image (click of refresh icon beside week calendar to display draft timesheet).

    5. Weekly Timesheets Cancellation.

      6. User can allow to cancel their submitted timesheets only not approved. To cancel it just click on cross icon located right side of weekly timesheet table as shown in below image.

  2. Track Time or Daily Timesheets

    3. By default Track Time tab is disabled. To enable it to go to settings from side panel under settings, under feature tab enable “Daily time tracking” row toggle switch, Track time able looks like as shown below image.

    1. Setup

      2. Few of setups of weekly timesheet is applicable for Track time. Admin can setup or customize Track Time as per their requirement before going to use weekly timesheet. Please look into below points.

      Tracker column setup: BY default, three-time tracking is displayed (Track time, log time and duration), you can disable any of it as per your requirement. To disable it to go to settings from side panel under settings, under feature tab, in “Daily time tracking options” row uncheck and click on save button and then save setting button.

      Enable daily timesheet report notification: By default, it is enabled, you can disable it from under feature tab in “Daily time tracking options” row.

      Max limit of hours per day: You can setup maximum limit hour per day for Track Time. Go to settings from side panel under settings, under feature tab first enable “Restrict daily and weekly hours” toggle switch then add max time limit and click on save icon finally click o save settings button.

    2. Track Time or Daily Timesheet submitting

      3. Here you can submit daily track time for current day or past day. To submit it, first select date, select project, tasks display related to selected project in task column dropdown, select task, add memo in memo column, add time in racker column finally click on save button, added details will appears in below table, select the check box and finally click on submit button as shown in below image.

      If you submit daily tack time timesheet, it will map with weekly timesheet table from where you can cancel submitted time if required.


      You are able to submit daily timesheet for multiple projects or multiple tasks on which you have been worked. Just repeat adding daily timesheet by clicking on save button and finally select all check box and click on submit button.


      save as draft for daily timesheet is not available

  4. My Tasks

  5. Approval

    6. Project managers can approve the timesheet from Approval tab, select the week then employee, click on item of the table, timesheet table with approve and reject button appears from where you can take the action accordingly

  6. Log Weekly Timesheet As.

    7. By this feature admin and program manager can able to submit timesheet on behalf of the users and by default this feature is disabled, to enable it, go to the settings from side panel under settings, under advance settings tab enable “Time tracking delegation by Admin / PM” row toggle switch.

    First click on weekly timesheet tab, log weekly timesheet dropdown appears, select the user from it and select the week number, fill the details in timesheets, and finally click on submit button shown in below image.

    same step should follow to submit daily timesheet on behalf of the users.

  7. Task Status

    8. Project manager can check task status of their employees from Task Status Tab of home page and notify users in case of task deadline date is overdue as shown image.

    Here you can filter task status from filters and can exports filtered data in to excel file, just click on export link at right top of the table.

    To notify the users about their task status, select the items in action column, click on notify button, popup will display and add the comments and click on submit button, users will get notified through mail

  8. Self-Assign Task (or Home page task tab).

    9. Users can create their tasks by their own from task tab as shown in below image, by default it is disabled, admin can enable it from settings from side panel, under settings, under feature tab, enable toggle switch for “Self-Assign Task” row.

    Allow self-assign task billable :By default this feature is disabled or billable is No in Add task page as shown in above image, when it is enable users are allowed to create billable tasks, admin can enable it from settings from side panel, under settings, under feature tab, enable toggle switch for “Allow self-assign task billable” row.
