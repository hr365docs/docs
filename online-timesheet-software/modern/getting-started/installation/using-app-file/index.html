Please make sure to download the installation package. which is usually a zip or cab file. Once the installation package is downloaded, follow the instructions in the knowledge base to upload the app to the app catalog.

    1. Navigate to the app catalog page using the following URL format:
      https://yourtenantname.sharepoint.com/sites/appcatalog/_layouts/15/tenantAppCatalog.aspx/manageApps

      Replace yourtenantname.sharepoint.com with your actual tenant name, for example: cubiclogics.sharepoint.com.
      Once you are on the page, click on the "Classic Experience" options.



    2. Click on App for SharePoint as shown in below.


    3. This displays the Apps for SharePoint page.

    4. Click on upload.


    5. Choose and upload the file.


    6. Once file got uploaded, select check box and click deploy.

      7. Now you can make this solution or application package available for all sites in the organization, once the application is installed, later you can remove the app from all sites.

      By performing these actions, the selected application package will be downloaded from the Microsoft SharePoint Store into your tenant.


    7. Open your site collection where you need to install your application.

      8. We have to add Timesheet Management 365 as web part in the page.


    8. Click on New and select Page.




    9. Now click on + icon and then select the Timesheet Management 365 as shown in below.



    10. You can also change the title areas as plain by clicking on edit icon and selecting Plain for better look as shown in below.


    11. Finally click on Publish button.


    12. When users first access TimeSheet 365, a welcome popup appears to introduce the platform and guide them through the initial setup process. Clicking "Get Started" will walk both administrators and users through the key features and initial configuration steps.


    13. The Quick Setup in Timesheet 365 allows users to configure their timesheet system efficiently. The setup includes the following key steps:


    14. Add general settings to Timesheet 365 for enhanced features and customization.



    15. When general settings are successfully configured in Timesheet 365, the status changes to "Completed".


    16. Add users to Timesheet 365 to boost collaboration and streamline tasks.



    17. When a user is successfully added in Timesheet 365, the status changes to "Completed".


    18. Add clients to Timesheet 365 to manage different clients and their respective projects.



    19. When a client is successfully added in Timesheet 365, the status changes to "Completed".


    20. Add projects to Timesheet 365 for managing tasks and tracking progress.



    21. When a project is successfully added in Timesheet 365, the status changes to "Completed".


    22. Add approval workflows to Timesheet 365 to handle different levels of approvers and management.



    23. When an approval workflow is successfully added in Timesheet 365, the status changes to "Completed".


    24. Add tasks to the respective projects in Timesheet 365 for effective tracking and task management.



    25. When a task is successfully added in Timesheet 365, the status changes to "Completed".


    26. Once all steps are completed (100%), the "Proceed" button is enabled, allowing users to enter the Timesheet 365 application.


    27. Finally, refresh the page.

    There are two types of installations:

    1. Installation via package upload in the app catalog site.
    2. Installation from the SharePoint store (AppSource).

    If you delete the package installation and reinstall the app from the SharePoint store, or vice versa, it will not impact or result in data loss. This is because the application's data is stored safely in SharePoint lists, which remain unaffected by the installation method.
