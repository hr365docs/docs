    Dashboard

    On the dashboard, you will find information related to time tracking based on Projects, Tasks, Employees, Activities, and Timesheet statuses.

    The first line chart displays the total hours logged for each week in the selected month, which can be filtered using the dropdown.

    The second bar chart shows a comparison between Billable and Non-Billable hours logged each week.

    The third bar chart provides a weekday-wise breakdown of total hours logged, helping identify productivity patterns by day.

    The below donut charts give an insightful breakdown of time distribution across Projects, Tasks, Activities, and Employees.


    Users can also expand each chart using the expand icon available in the top-right corner of every widget.




    The dashboard can also be globally filtered using the global filter icon for a consistent view across all widgets.



    Select the dropdown options and click on the Apply Filter button to reflect data based on selected parameters.



    Timesheet Summary

    You can view the status of timesheet submissions such as Approved, Pending, and Rejected for the selected period.

    This data is represented in both chart and tabular format to support quick actions and deeper analysis.



    Pending timesheets across the last five weeks are visualized using donut charts to show weekly distribution.


    Dashboard Settings


    This section allows users to control which charts appear in each row of the dashboard layout. You can select up to 3 widgets per row.

    1. Click the Dashboard Settings icon on the top-right corner of the dashboard.


    2. Use the toggle button labeled “Display sample data on dashboard” to preview sample data if needed.


    3.The settings panel will display three rows (Row 1, Row 2, Row 3) for configuration.



    Under each row tab, you can select from the following chart/widget options:

    • Total Hours Logged
    • Billable vs Non Billable Hours
    • Hours by Weekday
    • Hours by Projects
    • Hours by Tasks
    • Hours by Activities
    • Hours by Employee
    • Timesheet Submission Status
    • Pending Timesheet

    Note: You can select a maximum of 3 widgets per row.

    4. After making your selections, click the Save button to apply changes to the dashboard layout.
