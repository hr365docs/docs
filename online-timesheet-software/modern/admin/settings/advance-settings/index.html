    Advance settings


    Advance Settings in the Timesheet allow customization of time tracking days, project name rules, and visibility of finance information in user reports.


    Time Tracking Days
    Select the days for time tracking by checking the boxes for Sunday to Saturday and click Save.


    Allow Same Project Names for Different Clients
    Toggle this option to allow multiple clients to have projects with the same name.



    When the toggle is disabled, attempting to assign the same project name to another client will trigger a dialog box with the message: "Project with the same name already exists."



    Exclude Finance Info from User's Reports
    Toggle this option to hide finance-related information from user reports.
