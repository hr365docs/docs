    Version 24.808.2113.0000        Released 08-Aug-2024

    🚀 What's New

        New features/Improvements:
    1. Full Sync Option: Added a full sync option on the User page to help sync Microsoft 365 users.
    2. Timesheet Customization Settings: You can now hide or show features like "Save as Template," "Copy Previous Week," and "Clone" in the Weekly Timesheet from the Timesheet Customization settings.
    3. Non-Billable Column: Added a non-billable column in report pages.
    4. Date Columns: Included submitted date, approved date, and rejected date columns in report pages.
    5. Month Column: Added a month column in the Daily and Monthly reports.
    6. Allow Self-Assign Task: Enabling this feature in Timesheet Customization settings will allow users to create their own tasks.
    7. Save Icon: Added a save icon in the header of Client and Project pages.
    8. Full Report Filter UI: Improved the user interface of the full report filter.
    9. Disallow Exceeding Allocated Task Hours: Enabling this toggle will prevent users from submitting timesheets that exceed the allocated task hours.
    10. Restrict Daily and Weekly Hours Based on Rules: Introduced in Feature Settings, you can now create rules to restrict daily and weekly hours.
    11. Collapsible Side Navigation Panel: Made the side navigation panel of the Microsoft Teams app collapsible.

      🔧 Bug Fixes

    1. Fixed an issue where activities were not populating in the individual activity rates on the Add Project page.
    2. Corrected the report Excel hours columns to be in integer format.
    3. Resolved a mismatch in total hours in the Daily and Monthly reports.

    Version 24.0704.1851.0000        Released 07-July-2024

    🚀 What's New

        New features/Improvements:
    1. Added a custom field to the weekly timesheet. Admins can add custom fields to the weekly timesheet from the features settings.
    2. Included an email template feature in the Premium plan.
    3. Added a search function for all dropdowns in the weekly timesheet.
    4. Included an activity column in the timesheet status table.
    5. Allowed editing of user details such as people manager, job title, and department.
    6. In the old version, the project manager was a default team member. Now, the project manager must be added as a project team member separately.
    7. Disabled projects' data do not display on the report pages.
    8. Removed the project delete option. Now, the user can disable the project instead of deleting it.
    9. Included billable and remark fields in the Add Task panel on the project page.

      🔧 Bug Fixes

    1. In the approval process, the date and month of the timesheet did not match the selected one. This issue is now fixed.
    2. Total tracked hours were not equal to the total hours of the timesheet in project details reports. This issue is now fixed.
    3. The current date was displaying in the Task Deadline Date column of monthly and full reports. This issue is now fixed.
    4. Fetching monthly and full reports required double-clicking the Apply button. This issue is now fixed, and the reports can be fetched with a single click.
    5. Allowing users to select future timesheets when the date format is dd/mm/yyyy is now fixed.
    6. The filter issue in project details reports is resolved.
    7. The user's name disappeared in the approval timesheet if the user's status was set to inactive. This issue is now fixed.
    8. In the Add and Edit Project page, tasks were not deleting. This issue is now fixed.

    Version 24.0424.1806.0000        Released 25-April-2024

    🚀 What's New

        New features/Improvements:
    1. Approval Workflow Feature: This feature allows us to designate a people manager or any specific person as a timesheet approver and create multiple levels of timesheet approvers.To enable this feature, toggle the "Enable Approval Workflow" setting to the "On" position in the Timesheet customization settings. Once activated, the feature will display in left-side panel in administration.
    2. Save as Template Button In Weekly Timesheet: After submitting a weekly timesheet, it can be saved as a template and utilized for other weeks simply by selecting the template.
    3. Copy Previous Week Button In Weekly Timesheet: The timesheet from the previous week can be copied to the current week and then submitted.
    4. Attachments in Weekly Timesheet: Files can be attached to the timesheet by clicking on the attachment icon located in the right-hand column before submitting the timesheet.
    5. Project Team Member Fields: On the "Add Project" page, project team member fields have been changed from dropdowns to a tabular format for easier management.
    6. User Status: Users can now be set as active or paused for a particular project within the project team member settings. The "paused" status makes the project available to the user in the weekly timesheet.
    7. Add and Assign Tasks: Tasks can now be added and assigned directly from the "Add Project" page for better task management.
    8. Comments Column: A comments column has been added to the status table, where remarks or comments added by the approver during timesheet approval or rejection are displayed.
    9. Month Filter: A month filter has been added to the Project report page.
    10. Department Filters: Filters for departments have been added to the daily report, weekly report, and monthly report for better organization and analysis.
    11. Task Status Toggle: A toggle has been added to the Timesheet field settings to hide or show the task status field in the weekly timesheet, providing users with more control over the displayed information.

      🔧 Bug Fixes

    1. -The issue where the admin receive the submitted timesheet email when sending it on behalf of another user has been fixed.
    2. Additionally, the problem with the admin's manager receiving email notifications when the admin rejects the timesheet has also been resolved.
