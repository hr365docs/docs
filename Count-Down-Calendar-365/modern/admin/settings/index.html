Settings

The Count Down Calendar 365 Settings panel enables administrators to configure calendar behavior, appearance, and interaction options. These settings allow full customization of countdown structure, door style, and display preferences to align with organizational themes.

Theme

Select a visual theme for Count Down Calendar 365:

  • Light: Displays doors on a bright, clean background.


  • Dark: Perfect for low-light displays, highlighting door contrast.


  • Site: Adopts the current SharePoint site theme for seamless integration.

Hide SharePoint Settings

The Hide SharePoint Component option allows administrators to remove default SharePoint web part controls, providing a cleaner experience focused on Spotlight content.



Users

The Users section allows administrators to manage permissions in Count Down Calendar 365 365. The user management grid provides the following functionality:


  • User Grid: Displays Name, Email, and Role (Admin/User) for each entry.


  • Click on + Add to add new user.

  • Role Assignment: Administrators can assign or update roles (Admin/User) via the dropdown.


  • Search & Pagination: Search by Name, Email, or Role, and adjust the number of entries displayed (10, 25, or 50 per page).


Calendar Configuration

Customize how your countdown calendar appears and functions:

  • Start Date: Select when the countdown begins.


  • Number of Doors: Define total number of countdown doors (e.g., 5, 10, 25, 31).


  • Rows & Columns: Arrange doors in grid format to suit your display preferences.


  • Custom Skip Dates: Manually enter holidays or non-working days to exclude.


Door Customization

Modify door appearance, color, and numbering:

  • Door Shape: Choose between Square or Circle designs.


  • Door Background Type: Set to Color or Image.


  • Door Background Color: Pick a custom background color.


  • Door Outline Color: Customize the border color.


  • Door Number Color and Size: Adjust visibility of door numbers.


  • Show Door Numbers: Toggle visibility of numbering on each door.


Door Content Configuration: Click on any specific door number to add personalized content such as an image, video, or a combination of both.


Simply enter the Image URL or Video URL in the provided fields, and the system will automatically display the assigned open date. You can also override this date if needed. Once configured, click the Close button to save and preview your customized door setup.
