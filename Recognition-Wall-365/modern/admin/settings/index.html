Settings

The Recognition Wall 365 Settings panel allows administrators to configure key global settings such as theme preferences, user permissions, and recognition categories. These options help personalize the appearance of the web part, manage access levels, and define recognition types for a more engaging experience.

Theme

Choose a preferred visual theme for Recognition Wall 365 to match your SharePoint environment or organizational branding. The available options allow users to toggle between light, dark, or site-based themes for better visual consistency.

  • Light: Provides a clean and bright interface that complements modern layouts.


  • Dark: Designed for low-light environments, offering an elegant dark background for improved readability.


  • Site: Automatically inherits the current SharePoint site theme to ensure a unified appearance.

Hide SharePoint Settings

The Hide SharePoint Settings feature allows administrators to disable the default SharePoint settings menu inside Recognition Wall 365. This helps maintain a clean interface and prevents unauthorized users from changing SharePoint-related options.



Users

The Users section allows administrators to manage access levels within Recognition Wall 365. Administrators can add, edit, or remove users and assign specific roles to maintain control over web part permissions.


  • Add New User: Click the +Add button to open the user form. Enter the user’s details, select a role (Admin or User), and click Save to complete the process.


  • Actions: Administrators can modify or remove users using the Edit and Delete icons under the Actions column.


  • Search & Pagination: Quickly find users by typing their name or email in the search bar. You can also adjust how many entries are visible per page (10, 25, or 50).


Add Category

The Add Category option allows administrators to define recognition categories, such as “Employee of the Month” or “Best Innovator.” Categories help organize recognitions and make them easier to browse on the recognition wall.

  • Click Configure under Add Category in the settings panel to open the category configuration section.


  • Click +Add to create a new category. Enter the Category Name and optionally upload an image representing the category. Click Save to apply the changes.


  • Administrators can also edit or delete existing categories using the respective icons.
