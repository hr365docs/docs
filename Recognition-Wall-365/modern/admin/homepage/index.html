  • Home Page

      • The Recognition Wall 365 Home Page offers an intuitive interface to manage and display employee recognition entries.


      Add Recognition: Click the + Add button to open the recognition form. Fill in details such as Name, Category, and Description, then toggle Active to enable visibility.



      Detail List View: Displays recognition records in a table format for easier sorting and management.



      Card View: Offers a visually engaging layout highlighting employee achievements with pictures and recognition details.



      Each recognition entry displays key details such as Name, Category, Description, and Status with actions like Edit and Delete for easy management.


      Click on Edit icon to open the update page. Make the necessary changes, then click on the Update button to save them.


      Settings: Administrators can configure key options to tailor Recognition Wall 365 according to organizational needs. They can choose between visual themes (Light, Dark, or Site), manage display preferences, and control who can add or view recognitions. This ensures a consistent and secure recognition experience across the SharePoint site.



      Help: The Help section provides essential information about the product, including version details, license type, and subscription validity. It also includes quick access to resources like Support, FAQ, Release Notes, Privacy Policy, and License Validation. Users can use this panel to check their activation status or reach out for assistance when needed.
