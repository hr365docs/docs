  • Home Page

    • The MS Forms 365 home page displays all available forms in a clean List View, showing key information such as title, form type, category, owner, created date, and response count.


    View Toggle Menu: Users can switch between List View and Card View by clicking the view icon. This menu allows toggling to the preferred visual layout.


    Card View Display: The Card View provides a visual layout where each form appears as an individual card displaying owner, creator, created date, and response information.


    Column Customization Button: In List View, users can customize which columns are displayed by selecting the column configuration icon.


    Column View Panel: The Column View panel allows users to check or uncheck specific columns such as Form Type, Owner, Created Date, and Status to tailor the List View.


    Open Form from List: Clicking a form title in List View opens the form’s detailed view on Microsoft Forms, allowing users to fill out or view the survey.


    Form Preview Page: When users click a form, they are taken to the Microsoft Forms response page where they can read the survey description and submit their answers.


    Settings: Access global configuration options such as theme preferences and admin management.


    Help: Open the Help panel to view license details, product version, and support links.



    The Home Page acts as the central hub where administrators can view, edit, or delete surveys, track user participation, and manage survey visibility. Its streamlined layout ensures a seamless experience for both survey creators and end users.
