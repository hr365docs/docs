Settings

The Subscribe Now 365 Settings panel allows administrators to configure global options such as themes and user permissions, ensuring the web part is both visually consistent and securely managed. In addition to Theme and User settings, administrators can also adjust the Appearance settings to control how Subscribe Now 365 is displayed and behaves within the site.

Theme

Select a visual theme for Subscribe Now 365:

  • Light: Applies a bright, clean interface.


  • Dark: Optimized for low-light environments.


  • Site: Inherits the theme of the SharePoint site.

Users

The Users section allows administrators to manage permissions in Subscribe Now 365. The user management grid provides the following functionality:


  • User Grid: Displays Name, Email, and Role (Admin/User) for each entry.


  • Add User: Click on + Add User to add a new user, enter the user name, assign roles, and finally click on the Save button.


  • Role Assignment: Administrators can assign or update roles (Admin/User) via the dropdown.


  • Search & Pagination: Search by Name, Email, or Role, and adjust the number of entries displayed (10, 25, or 50 per page).


  • User Removal: Use the trash icon to delete users and revoke access.


Hide SharePoint Settings

The Hide SharePoint Settings option allows administrators to disable the default SharePoint settings menu within the Survey 365 web part. This helps prevent unauthorized modifications and keeps the interface focused only on poll-related configurations.



Appearance

The Appearance section allows administrators to customize how the Subscribe Now 365 card is displayed on the site. Options include:

  • Display Content: Edit the main text shown on the subscription card.


  • Button Label: Change the label of the action button (e.g., “Subscribe Now” or “Manage Subscription”).


  • Alignment: Choose how the subscription card content is aligned within the site layout.

    Left Alignment: Positions the content to the left side of the card for a structured, text-first layout.



    Center Alignment: Places the content at the center of the card, ideal for balanced and focused layouts.



    Right Alignment: Aligns the content to the right side of the card, often used when emphasis is on visuals or action buttons on the left.






  • Information Message: Customize the informational text shown at the top of the subscription panel.




After making changes, click Save to apply the updates.
