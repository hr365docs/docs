How can I add the app to the left navigation in Microsoft Teams instead of having it appear as an additional tab in a team's channel?
  • The application must be installed in your SharePoint tenant's root site to appear as an app in Microsoft Teams.

    Custom level Permission
      To set up custom-level permissions, follow these steps:
  • Open the site where the application is deployed and click on the settings icon located in the top right corner.


  • Select "Site permissions" and then click on "Advanced permissions settings."


  • Choose "Permissions Levels" and click on "Add a permission level."


  • Click on “Add a Permission Level” and provide a name for the custom permissions.


  • Select "Add items," "Edit items," "Delete items," and "View items." Then, scroll down and click on "Create."


  • Return to the previous page and create a group.

  • Enter the group's name and select the custom permission created in step 3. Click "create."


  • Once the group is created, add users to this group.

  • Click on "New," and enter the names of the users who will access the application. You can also select "Everyone except external users" to grant access to all users automatically, reducing manual additions and you can also add any SharePoint groups directly if needed.


  • Use the "Check permissions" feature to verify the granted permissions, ensuring only the custom permission created in step 4 is assigned.

  • Navigate to the pages, click on the "Share" button, and enter the name of the group created in step 5. Ensure that only view access is granted to the group.

    • These steps will help you grant custom-level permissions efficiently.

    Can a HR apply leave on behalf of the users?
    1. As a HR executive, this tool gives the ability to apply leave on behalf of the user.
    2. Go to HR Actions tab and click on “Add leave on behalf of user” button.
    3. Once the pop-up opens, enter employee name and email and apply the leave on behalf of that employee and click on submit for approve.
    Where can we define annual casual paid leaves & medical leaves?
    We can configure leave management application settings from the HR365 portal.
    Annual casual paid leaves, medical leaves, number of restricted holidays, annual leave start date and end date can be defined in settings page of HR365. The calendar year also can be defined here.
    From where can a manager see all the requests for leaves?
    Manager can see all the requests for leaves for a particular day in “Manager Action” tab.
    Do we have option of customizing or adding more leaves types?
    1. Yes. We can customize or add more leave types as per requirement.
    2. First go to “Leave Settings” tab and select “Leave Type” and add leave type as per your requirement.
    3. This leave type would be added in New request form while adding new leave request.
    Where can employee see his status of leave requests?
    An employee can see the status of all his previous leave requests where those are in pending, approved or rejected in the home page from this portal.
    Where can we define various email templates?
    1. First go to “System Settings” tab then select Email Settings.
    2. Here you can define email body, email signatures, you can set it up as per your organization needs.
    Can we add additional approvals for approving the leave request?
    Yes, we can add additional approvers in Settings by enabling Additional Approvers from advanced settings. Additional approvals would be added to the application while adding a Leave request (in the additional approvers dropdown all manager's roles person shown).
    Does the manager has the ability for bulk approval?
    1. Yes. Manager has the ability for bulk approval.
    2. First click on “Click here for Bulk Approval”.
    3. Once pop-up opens, select the leave request you want to approve or reject and select Approve or Reject button accordingly.
    Is there any provision for applying Comp-off leave request?
    Employee can raise a comp-off for any previous day he worked on.
    First, go to “Add Comp-off” tab, then click on “Add” button.
    Once the pop-up opens, apply your Comp-off leave request, and as soon as user clicks submit for approve, that would stay and get added into the leave record.

    How can I downgrade to a previous version of the Time off Manager 365 application?

    Follow these steps to safely revert to an earlier version of the Time off Manager 365 application:

    • Access the AppCatalog directly via the following link:
      https://yourtenantname.sharepoint.com/sites/appcatalog/_layouts/15/tenantAppCatalog.aspx/manageApps

    • Navigate to the AppCatalog and search for the Time off Manager 365 application.


    • Select the application and click on More Options.


    • Click on Version History.


    • Select the version you wish to restore, then choose the appropriate time and date.


    • Options such as View, Restore, and Delete will be displayed.


    • Click Restore and confirm the action by selecting OK.


    • Once restored, verify the version number displayed below the selected time and date.


    • After confirmation, select the package, click on Files, and choose Deploy.


    • In the deployment popup, check the required checkbox and click Deploy to finalize the process.
