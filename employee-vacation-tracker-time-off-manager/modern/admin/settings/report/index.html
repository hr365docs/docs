Audit Logs

The Audit Logs feature in the application provides a detailed record of all user activities and system changes within the Settings module. It helps administrators track modifications, identify unauthorized actions, and ensure compliance with organizational policies.

Click on this icon button.



After clicking, a panel will be shown where you can check the history of any setting.



If the user wants to view the previous history of the Date Format, they should click on the Date Format option. Both the previous and current history will be displayed, showing what it was before and what changes have been made.



Then, the Audit Logs panel will be displayed. The user should click on the Date Format option.



Then, the user will get all the details of the Audit Logs.
