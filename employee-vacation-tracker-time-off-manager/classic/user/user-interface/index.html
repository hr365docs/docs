User Interface

Apply leave

From here, users can apply leaves. To apply leaves, follow the below steps:


  1. Click on the New Request on the home page, this displays the Apply Leave page.

  2. Select the type of leave. Enter title, start date, end date and note to approver.

  3. Click on the Submit button. The user will receive a mail for the applied ticket.

Apply Comp off (off in lieu)

To apply Comp off (off in lieu), the user should have prior approvals. To apply follow the below steps:


  1. Click on the + Comp off(off in lieu) from side nagivation on the home page, this displays the Comp off(off in lieu) page.

  2. Click on the + Add, this displays the Add Comp off(off in lieu) page.

  3. Enter title, date and reason.

  4. Click on the Submit for Approval button. The Manger will receive the request on the home page.

  5. Once approved, user can apply leave from New Request on the home page.
