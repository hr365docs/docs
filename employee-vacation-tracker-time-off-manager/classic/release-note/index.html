    Version 11.22.1217.1640        Released 18-Dec-2022

    🚀 What's New

    New + Improvements:
    1. Multiple Language (Can select your preferred Language for the Time off manager application from the advanced settings).
    2. Import leaves from the Leave Settings (Can add balance leaves of last year through the bulk import).
    3. Partial day (Can select the partial day on either an hourly basis or half/quarter-day from the advanced setting).
    4. Show/hide applied leaves on the calendar of apply leave form from the leave settings.

    🔧 Bug Fixes

    1. The maximum Carry forward leaves displayed on the apply Leave form on the next new annual leave start date automatically.
    2. Managers can view the leaves report of only their Employees.
    3. Sync M365 issue is fixed (Now it syncs the user's manager name).
