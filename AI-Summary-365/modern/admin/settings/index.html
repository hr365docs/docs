Settings

This section provides an overview of the configurable global settings within the AI-Summary-365 settings panel, enabling administrators to customize themes, AI integration, and user permissions.

Theme

Select the theme for the web part display:

  • Light: Applies a bright, clean interface.


  • Dark: Applies a dark theme designed for low-light environments.


  • Site: Inherits the theme from the associated SharePoint site.

Users

The Users section allows administrators to manage access and permissions within AI-Summary-365. The user management grid provides the following functionality:


  • Add User: Click on + Add User to add a new user, enter the user name, assign roles, and finally click on the Save button.


  • Role Assignment: Administrators can update roles for each entry, choosing between User and Admin.


  • Automatic Entry: The details of the current logged-in user are automatically added to the grid when accessing the site.


  • User Removal: Administrators can delete users using the trash icon to revoke access.


AI Configure

Click Configure next to AI Configure to open the AI Integration panel on the right side of the page.


Azure OpenAI

Use the On/Off toggle to enable or disable Azure OpenAI integration.


Specify the Endpoint URL for your Azure OpenAI service, provide the API Key to authenticate your instance, and enter the Deployment Name associated with your configuration then click on save button.


Copilot

Toggle On/Off to enable or disable Copilot integration. When enabled, provide the Copilot URL associated with your service then click on save button.




Note : Active AI Selection: The AI service (Azure OpenAI or Copilot) whose toggle is left on when you click Save will be used for generating summaries. Ensure only one AI service is active at a time.

Sharepoint Page Configuration

The Sharepoint Page Configuration option allows administrators to disable the default SharePoint settings menu within the AI Summary 365 web part.
