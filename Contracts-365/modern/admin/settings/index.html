Settings

The contract 365 Settings panel allows administrators to configure global options such as themes, date formats, user permissions, contract types, counterparties, currency, and approver roles. These settings ensure consistency across all contracts and help manage access effectively.

Theme

Select a visual theme for contract 365:

  • Light: Applies a bright, clean interface.


  • Dark: Optimized for low-light environments.


  • Site: Inherits the theme of the SharePoint site.

Date Format

Administrators can define how dates appear across contract 365 (e.g., DD/MM/YYYY or MM/DD/YYYY).

Users

The Users section allows administrators to manage permissions in Request 365.


  • User Grid: Displays Name, Email, and Role for each entry.


  • Role Assignment: Assign or update roles via the dropdown (Admin or User).


  • Search & Pagination: Search by Name, Email, or Role, and control number of entries per page.


  • User Removal: Remove users and revoke access when needed.


Contract Type

Define and manage available Contract Types for requests to maintain consistency across the organization.



Click on + Add to create a new contract type. While adding, enter the following details:

  • Contract Name – e.g., Master Agreement, NDA, Service Contract
  • Contract Code – a short identifier for the contract type
  • Status – toggle between Active/Inactive

After filling the details, click Save to add the contract type.


Counterparty

Manage Counterparties for your contracts by adding organizations or individuals involved.



Click on + Add to create a new counterparty. While adding, provide:

  • Counterparty Name – organization or individual
  • Email – contact email of the counterparty
  • Status – toggle Active/Inactive

Once completed, click Save to add the counterparty to the system.


Currency

The Currency section is used to configure how currency values are displayed in requests.

  • Organization Currency – the primary currency chosen to standardize financial records and transactions within your organization.


  • Number Separator – choose how numbers should appear:

    Once the separator preference is selected, the change applies across all requests.

Approver

Approvers : By default approver will be user manager if you want others click on others to add other approver.
