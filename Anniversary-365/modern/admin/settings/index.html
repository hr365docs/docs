Settings

The Anniversary 365 Settings panel allows administrators to configure global options such as data synchronization, display preferences, themes, and administrator management. These settings ensure a tailored experience for each organization while maintaining control over how work anniversaries are displayed and managed.

Sync Date of Joining

The Sync Date of Joining option allows administrators to fetch employee joining details from connected directories such as Azure AD or SharePoint Online (SPO). This ensures that work anniversary data stays accurate and up to date.


Display Work Anniversaries Based On

The Display Work Anniversaries Based On setting allows administrators to select the period for which anniversaries should be displayed. This makes it easier to highlight upcoming milestones.

  • Current Day: Shows anniversaries occurring today only.

  • Current Week: Displays all anniversaries within the current week.

  • Current Month: Lists anniversaries for the current calendar month.

  • Upcoming Month: Displays anniversaries for the upcoming month for proactive celebration planning.


Theme

Select a visual theme for Anniversary 365:

  • Light Mode: Applies a clean, bright interface suitable for well-lit environments.


  • Dark Mode: Optimized for low-light conditions, providing a comfortable viewing experience.


  • Site Mode: Inherits the current SharePoint site theme for a consistent look and feel.


Hide SharePoint Settings

The Hide SharePoint Settings option lets administrators hide the default SharePoint configuration menu inside the Anniversary 365 web part. This keeps users focused on anniversary-specific options and prevents accidental modifications.



Add Additional Admins

The Add Additional Admins feature enables organizations to assign and manage extra administrators for Anniversary 365. Admins can grant roles, manage access, and control who can edit configuration settings.


  • User Grid: Displays user information such as Name, Email, and Role (Admin/User).


  • Role Assignment: Admins can promote or demote users by selecting roles from a dropdown list.


  • Search & Pagination: Quickly search for users and adjust the number of entries shown (10, 25, 50 per page).


  • User Removal: Delete users and revoke their admin privileges using the trash icon.
