    Installation of Civic 365 from SharePoint Store

        Prerequisites:

      1. A global admin or SharePoint admin is capable of installing the application.
      2. The application's features function through APIs, and global admins have the authority to enable these APIs.

    1. Go to the apps catalog page from the below URL format and click on classic experience options.
      https://yourtenantname.sharepoint.com/sites/appcatalog/_layouts/15/tenantAppCatalog.aspx/manageApps


    2. Click on setting gear icon and select on Add an app.



    3. Click on SharePoint Store and type HR365 in search box. All applications from HR365 will sort.



    4. Click on Civic 365 application.


    5. Click on Add to app Site.


    6. Select the checkbox (Enable this app and add it to all sites ) and click on Add button.

      Now you can make this solution or application available for all sites in the organization, once the application is installed, later you can remove the app from all sites.


    7. A dialog box will open click on the Go to API acess page.


    8. On SharePoint admin center page,From left panel under advanced click on API access option under pending request, there are 3 requests (Presence.Read.All, User.Read.All, and MailboxSettings.Read) select and click on approve one by one for all the 3 requests.



    9. Go to Location (i.e., Site collection or subsite) where you want to install the Civic 365 application.

      10. We have to add Civic 365 as the web part on the page.

    10. Click on New and select Page.



    11. Click on Create page.



    12. Now click on + icon and then select the Civic 365 as shown in below.



    13. You can also change the title areas as plain by clicking on Edit icon and selecting Plain for better look as shown in below.


    14. Finally click on Publish button.



    How to remove this solution available to all sites in the organization


    1. Now select the Civic 365, click on files and then the deploy icon.


    2. Now unselect the check box and finally click on deploy button.
