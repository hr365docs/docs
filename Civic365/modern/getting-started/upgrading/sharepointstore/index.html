    Perform Update

    To update the application, you can follow the below steps:

    Only SharePoint admin can upgrade the application.


    1. Go to  "portal.office.com"



    2. Access the Microsoft admin center through the app launcher.



    3. Access the navigation menu in the upper-left corner and then click on 'SharePoint.



    4. In the SharePoint Admin Center, choose 'More features' and then click 'Open' under 'Apps'.




    5. Click on classic experience as shown below.


    6. Select the Civic 365, click on Files and then Upgrade Store App.



    7. Once you click on the Upgrade Store App button, this will redirect you to the below page then click on the Deploy button.


      8. If you want to make this application available for all sites, you can select the check box and then click on the Deploy button.

    8. You can verify the application version in the App Catalog and Civic 365. In both, the application version must be the same.

      9. How to remove this solution available to all sites in the organization


      1. Now select the Civic, click on files and then the deploy icon.


      2. Now unselect the check box and finally click on deploy button.


      3. After completing the upgrade, please go to the application and click on 'Validate App Update' on the Help page.
