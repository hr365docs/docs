Advanced

    Configure SharePoint API permissions:
    1. SharePoint admin can configure API permissions.
    2. This configuration helps in excluding block sign-in users, excluding hidden users in GAL, excluding shared mailbox users of O365, syncing mobile numbers from azure AD.

    How to configure SharePoint API permission?

    Click on click here link in settings, redirect to SharePoint admin centre access API page. There you will find 3 Microsoft graph under organization- wide select and click on approve one by one for all 3 Microsoft graph

    Sync unlicensed users: Technically users who does not contain work email Id in SharePoint profile is considered as unlicensed user, you can sync those users by enabling toggle switch.

    Sync unlicensed users
    1. Here we have 3 option to sync the images namely SPO, outlook and ED.
    2. In case if you select outlook or ED options, if image is not available for outlook or ED, image from SharePoint user profile if exist.
    3. ED: Change the source of profile images, you can use custom images by placing images in image library. Click on click here, redirect to image library upload the images. Format of image is -username@domain.com.jpg

    Sync and display mobile number from azure: This works on API; it sync’s mobile numbers from azure. If mobile number is not available in azure, it displays mobile number if exist in SharePoint user profile.

    User presence This works on API, it shows MS teams status on profiles in home page

    Auto load instead load more button: If you enable this, users load automatically as you scroll down vertically in home page, no need to click on load more button to load the users in home page.

    Hide side navigation panel in SharePoint page: If you enable this, side navigation panel of SharePoint page will hide

    Show extended fields in list view: If this is enabled, the right-side hidden column in lists view will displays, you can check it by scrolling horizontal to right side.

    Records to load: Load number of profiles on refresh or first-time page load, you can select 25 or 50 or 100. Higher numbers come with additional page load time, please select wisely.

    Custom fields: You can add custom fields and sync details from AD, Azure AD and SPO location.

    Birthday animation: On birthday of user, birthday notification will send to the users. This works on Power Automate setup. Here we can change the mail format from template which notified to users on their birthday
