Comp off(Off in lieu)

If user worked on weekend or non-working can avail that leave from add comp off. once approved they can utilize it later. Comp off leave can be enable or disable from leave settings



Once you click on Add button, add comp off panel opens from where user submit the comp off request.

Note: For add comp off request there is only one level approver. Manager will approve the request.
