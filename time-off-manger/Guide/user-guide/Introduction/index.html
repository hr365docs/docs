Introduction

What is Time off Manager?

Time off Manager is a Sharepoint based solution designed to help organization automate the processing of requests for leave.This system allows employees to easily request for leave and can determine their time off balance.

What is in the User Guide?

The Time off Manager User Guide has been written for the end user. This guide covers the Time off Manager User Journey from end-to-end.
