Home Page

On Home Page user can request for leave and can see the status of applied leaves. Wheather it is being Approved, Cancelled, Rejected or Pending.



  • New Request tab: Click on it will open apply leave from where we can request the leave.

  • Leave Status: Leave status show remaining leaves. Admin or HR can enable leave status for particular leaves in Leave settings for navigation panel.

  • All: Here all status leaves display

  • Draft: In these saved leaves will shows, users can later edit and submit.

  • Pending: Here user can see all pending leaves display.

  • Rejected: Here user can see their rejected leaves.

  • Approved: Here user can see their approved leaves.

  • Cancelled: Here user can see their cancelled leaves.
