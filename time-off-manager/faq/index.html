Frequently Asked Questions

From where the employees can apply leave?
  1. An employee can apply his leave from their self service portal or from the application.
  2. First click on New Request button.
  3. Once pop-up is opened, apply your leave request and as soon as user clicks submit for approve, it would get an email to the manager. As well as that leave would be added in manager action.
Can a HR apply leave on behalf of the users?
  1. As a HR executive, this tool gives the ability to apply leave on behalf of the user.
  2. Go to HR Actions tab and click on “Add leave on behalf of user” button.
  3. Once the pop-up opens, enter employee name and email and apply the leave on behalf of that employee and click on submit for approve.
Where can we define annual casual paid leaves & medical leaves?
We can configure leave management application settings from the HR365 portal.
Annual casual paid leaves, medical leaves, number of restricted holidays, annual leave start date and end date can be defined in settings page of HR365. The calendar year also can be defined here.
From where can a manager see all the requests for leaves?
Manager can see all the requests for leaves for a particular day in “Manager Action” tab.
Do we have option of customizing or adding more leaves types?
  1. Yes. We can customize or add more leave types as per requirement.
  2. First go to “Leave Settings” tab and select “Leave Type” and add leave type as per your requirement.
  3. This leave type would be added in New request form while adding new leave request.
Where can employee see his status of leave requests?
An employee can see the status of all his previous leave requests where those are in pending, approved or rejected in the home page from this portal.
Where can we define various email templates?
  1. First go to “System Settings” tab then select Email Settings.
  2. Here you can define email body, email signatures, you can set it up as per your organization needs.
Can we add additional approvals for approving the leave request?
Yes, we can add additional approvers in Settings by selecting the “Second level approval for LAMS App” and two level approvals would be added in the LAMS application while adding Leave request.
Does the manager has the ability for bulk approval?
  1. Yes. Manager has the ability for bulk approval.
  2. First click on “Click here for Bulk Approval”.
  3. Once pop-up opens, select the leave request you want to approve or reject and select Approve or Reject button accordingly.
Is there any provision for applying Comp-off leave request?
Employee can raise a comp-off for any previous day he worked on.
First, go to “Add Comp-off” tab, then click on “Add” button.
Once the pop-up opens, apply your Comp-off leave request, and as soon as user clicks submit for approve, that would stay and get added into the leave record.
