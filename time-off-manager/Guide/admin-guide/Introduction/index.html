    Introduction


    After the completion of installation once open the application after the installation initially, pop up appears from where you can fill your organization details, and if click on save next button, time off popup appears from where you can add your types of leaves and its number.

    If you want to do this later, it can be done from system settings in side navigation panel.





    Importance of Roles:
      Following are the different types of roles in Time off Manger.
    1. Admin:
      1. Admins have full control, they can add users, organizations, and office location.
      2. Can add Public/Government holidays, Restricted holidays.
      3. Can be able to add leave types and number of leave for different leaves.
      4. Can be able to change settings of the application.
      5. Admin cannot be able to approve other manager employeeâ€™s leave request.

    2. HR:
      1. Can able to approve leaves when second level approval is HR through email and HR action option in portal.
      2. Can apply leaves on behalf of other users.
      3. Can approve their employees leaves from Manager action in home page and from email notification.
      4. HR can approve leaves on behalf of managers incase he/she delegated from both email notification and Time off manager portal from HR action option.
      5. Can able to add public holidays.
      6. Can able to view and generates leave reports.

    3. Manager:
      1. Manager able to approve leaves request of their employees only.
      2. Can be able to approve leaves request for delegated managers from email notification in case if Admin or HR delegated
      4. Can be able to approve add comp off requests.

    4. User
      1. Can only send the leave request.

    5. No Access:
      1. The person will not able to access the application with no access role. User can request the access; admin will receive the email notification and can provide access.
