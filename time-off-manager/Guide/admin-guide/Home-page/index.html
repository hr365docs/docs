Home Page



  • New Request tab:  Click on it will open apply leave from where we can request the leave.

  • Leave Status:  Leave status show remaining leaves. Admin or HR can enable leave status for particular leaves in Leave settings for navigation panel.

  • All:  Here all status leaves display

  • Draft:  In these saved leaves will shows, users can later edit and submit.

  • Pending:  Here user can see all pending leaves display.

  • Rejected:  Here user can see their rejected leaves

  • Approved:  Here user can see their approved leaves.

  • Cancelled:  Here user can see their cancelled leaves.

  • Manager Actions:  This section is visible only for admins and managers, from where leave can approve or rejected.

  • Bulk Approval:  Bulk approval shows in Manager action section from where manager can reject or approve multiple leave requests at a time.

  • Leave Status:   On top, you can see the number of leaves that are remaining.

Bulk Approve page.



Click on up arrow icon located on extreme right top corner for the table will redirect to bulk approval page.



Leave Request:
All the leaves that are pending will be displayed here until you approve them. Manager can also take action for multiple type.




Comp Off Request:
In this section, all add comp off leave requests will be displayed here to Manager where their user request the add comp off, from here you can bulk approve/reject them.
Note: Once add comp leave request appproves, user will gain the comp off leave and later they can utilise it. For add comp leave is first level approval.



New Request:
One can apply for a leave by clicking on new request button.


Calendar:
In this section, you can see public holiday in orange color applied holidays in blue color. In below calendar you can see the leaves of the other of same department.




  1. Leave tabs: Select the leave type your applying for and fill the required details like title of the leave, start date, end date and notes.
  2. You can also the see information like total leaves this year, leave entitlement, leaves used, leaves remaining and number of days your applying leave for.
  3. Additional Approvers: This will display when admin enable it from advance settings. User can add additional approver (in additional approvers dropdown all managers roles persons show)
