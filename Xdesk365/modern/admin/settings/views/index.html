    Views

    In Xdesk 365 you will be presented with various views. These views are structured to present you with key information.


      Align employee basic information


      In this, you have the option to align basic information to the left, right, or center.

      If the user has selected left alignment


      Then the user details will be displayed on the left side.


      If the user has selected center alignment


      Then the user details will be displayed on the Center.


      If the user has selected right alignment


      Then the user details will be displayed on the right side.


      Clear alphabet selection with reset filter


      This configuration will reset any alphabet selection to 'All' when you click on the reset filter icon on the home page.


      Dashboard feature

        There are two features in the dashboard:

      1. Available for All Users: This indicates that the dashboard is accessible to all users
      2. Available for Selected Users: This means the dashboard will only be accessible to specific users who are added to the include user list.



      Grid Width


      In this location, you can expand or reduce the grid width in the grid view. Furthermore, you'll find two buttons: "Reset to Default" and "Save." Selecting the "Reset to Default" button will revert the width to its initial default setting.


      If the user has added a 300px width and then clicked Save, the width will be set to 300px.


      And then, the card will be displayed with a width of 300px, according to the specified size.


      Hide manager of specific user


      In this location, you can expand or reduce the grid width in the grid view. Furthermore, you'll find two buttons: "Reset to Default" and "Save." Selecting the "Reset to Default" button will revert the width to its initial default setting.


      If you want to hide a user's manager, enter their name.


      So, then the manager will not be visible in the user's profile card.


      And if you want to revert to the previous view, select the user in the delete section and delete it. Then, it will show as it was before.



      Hide mobile number of specific user


      Utilize this configuration to conceal the mobile number of specific users within the application. Simply add the manager and click the "Add" button.

      If you want to hide a user's mobile number, enter their name.


      So, then the mobile number will not be visible in the user's profile card.


      And if you want to revert to the previous view, select the user in the delete section and delete it. Then, it will show as it was before.


      Hide Sharepoint Page's Default Components


      From here you can hide sharepoint page default Hide side navigation panel in SharePoint page: To Enable this toggle you can hide the navigation.

      Hide topbar in SharePoint page: To Enable this toggle you can hide the topbar in Sharepoint Page.

      Hide command bar in SharePoint page: To Enable this toggle you can hide Command bar from sharepoint page.

      Hide sharepoint comments wrapper: Enable this toggle will hide comments wrapper in sharepoint page where app is running.

      Hide webpart title in SharePoint page: Enable this toggle will remove the Webpart title.

      Custom css for webpart to remove spaces: Enable this toggle will remove space outside of Webpart.

      Make SharePoint page layout in full width: Make your SharePoint page width in full screen for larger resolution where app is running.



      Labels


      Within this option, you can update the label "Executive" to "Executive(s)" and "Non M365" to "Non-M365 users."


      If you want to change a label, such as changing 'About Me' to 'About For Me,' type 'About For Me' in the field under 'About Me' and then click Save.


      And then you will see that the 'About Me' label in the Profile Card has now been changed to 'About For Me'.

      You can do the same for 'Non M365' and 'Executive' as well.


      Profile Image Width

      This setting allows you to adjust the width of profile images. It includes a "Reset to Default" option and a "Save" button to apply changes.



      Show or Hide modules


      Within this panel, there are many features, each accessible via a toggle switch:


      In ED365, users can come from Azure, categorized as external, internal, or unlicensed. When the external toggle is turned on, only external users will be synced.


      You can view the total users, which will include all external users.


      In the Guest and External users differentiation, you get an option to add a border to the user's profile image for identification.


      The first option is the default mode, where the border color will appear in the default orange shade.


      The second option is the theme mode; selecting this will apply the border color based on the selected theme.


      The profile image border will display something like this, highlighting the user based on the selected theme mode.


      The last option is Custom Mode, where you can set a custom color for the profile image border according to your preference. Once saved, the border will display in your selected color.


      Same as the tree shown in the image below.


      When the Sync unlicensed users toggle is turned on.


      All users without a license will then be displayed on your site.


      When the "Free/Busy Information" toggle is on,


      The profile information card will display the user's status along with their image, indicating whether the user is busy or free.


      When the "Use 24H Format" toggle is turned on, you will see the 24-hour format in the user's profile details.


      When the "Show Member Of" toggle is turned on.


      After that, you will be able to see which group the user is a member of.


      Hide labels for empty profile fields:

      When disabled: Labels are displayed for all profile fields, regardless of whether data is available.



    When enabled: Labels for profile fields without data are hidden, providing a cleaner and more streamlined profile view.



If the Organizational Chart toggle is on.


This icon will appear within the home tab.


If you click on it, a chart will open in front of you.


If the User presence toggle is on.


Then, you can see the user's status as Active or Inactive.


If the Show dashboard toggle is on.


This icon will appear within the home tab.


And you can view the dashboard data by clicking on the icon at the top.


If the "Enable MS Teams live chat integration" toggle is on.


You will be able to chat live on MS Teams, allowing real-time communication with your team members directly through the platform.


If the "Enable WFH" toggle is turned on.


After that, an edit icon will appear next to the user's name on the profile card.


Click on the icon, and a popup will appear where you will need to input the WFH (Work From Home) details.


And then, the Profile card with the WFH icon will be displayed on the home page.


When the "Display Weekly Work Status on Profile Card" toggle is on.


Then the weekly work status will appear on the profile card, showing the user's WFO or WFH details along with the date and day for the entire week.


If you want to edit, click on the edit icon.


Select "Default Always Work From Home." If you choose this option, then click on save.


Select "Default Always Work From Offer." If you choose this option, then click on save.


If you want to change the status for the current week, you can do so. After making the changes, click on the save button to save them.


Similarly, if you want to make changes for the next week, you can do so as well.


A "Next" icon is provided; clicking on it will display the weekly report for the next week.


Enable the "Allow users to update their pronouns" option.


Then the current user can update their pronouns (they/them, she/her, he/him) and save the changes.


When the "Allow users to export directory" toggle is on.


Click on the CSV file icon on the home page.


Then, a popup will appear, and in the popup, you will have the option to download. If you want to download the displayed records, click on the download button to save them.


When the "Allow users to print pdf" toggle is on.


Click on the Print file icon on the home page.


After that, a popup will appear with two buttons: "Portrait" and "Landscape." You can choose either option based on the type of file you wish to download.


When the "Report incorrect info" toggle is on.


In the "Report Incorrect Info" section, if there is incorrect information or details you need to add, you must provide the required field values. After entering the information, click "Submit." This request will then be sent to the Admin for review. This option is only available to the current user.


When the "Show extended fields in list view" toggle is on.


In this, you will only see a vertical scroll (up and down) within the list, and the horizontal scroll on the side will not be visible.


If the "Show filters as dropdown" toggle is on.


Then, the dropdown will be displayed at the top of the home page, allowing users to select filters easily from the dropdown menu.


If the favicon toggle is off.


If the favicon toggle is off, the default favicon for SharePoint will be displayed.


If the favicon toggle is on, you will see an option to update the favicon at the bottom, allowing you to set your custom favicon.


After cropping the favicon, click on the "Save" button to apply your custom favicon.


Then, your custom favicon will be displayed.


If the "Show Groups" toggle is on, then you can select the group you need.


And then, a dropdown will appear with the group dropdown. You can search for the group that was selected after turning on the toggle.


    Sort by


    To overcome the Microsoft Graph API search query limitations, the option for sorting by last name is exclusively accessible when SharePoint Online and Graph API integration is chosen within the Advanced settings. You have the choice between two sorting options: UPN and display name.


    If the user selects 'Sync Employee Profile Information from Graph API.


    And then, two options will be shown: first, 'Sorted by UPN' and second, 'Sorted by Display Name.' The user can select the first option Sorted by UPN.


    Then the cards will be sorted by UPN.


    The user can select the second option Sorted by Display Name


    Then the cards will be sorted by Display Name.


    If the user selects 'Sync Employee Profile Information from SharePoint Online and Graph API.


    The user can select the first option Sorted by First Name.


    Then the cards will be sorted by First Name.


    The user can select the second option Sorted by Last Name.


    Then the cards will be sorted by Last Name.


    If the user wants to sort by last name but display the first name, they should select 'Last Name' for sorting and toggle on the 'Display Label as First Name' option.


    Then, at the bottom, you will see the cards sorted by last name, with the display showing the first name.


    If you select the 'Display Name' option


    And then, the cards will be displayed sorted by display name..


    Theme


    From here you can select the theme for the application. There are three themes namely dark mode, light mode, and site theme color.


    If you have selected the Dark Theme option.


    Then the theme of your application will change to dark.


    If you have selected the Light Theme option.


    Then the theme of your application will change to Light color.


    Work email ID/UPN ID


    The default setting for pulling user details based on their UPN or work email ID is currently disabled. If you wish to enable pulling records based on work email, you can do so here.
