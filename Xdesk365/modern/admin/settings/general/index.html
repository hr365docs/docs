General

From here you can update date format, logo, theme etc.

Date Format

From here admin can change the date format. The date format settings apply everywhere in the application.

Time Format

From here, the admin has the capability to adjust the time format of the helpdesk application.


Theme

From here, the admin can change the theme of the helpdesk application.


Admins can customize the theme of the helpdesk application by selecting a custom color and saving the changes in the settings.


Show resolved ticket in selected tabs

If you enable Show Resolved Tabs in Home Page from General, you can view the resolved tabs on the home page.

GCC Tenant

GCC (US Government Community Cloud), if your Microsoft 365 tenant is GCC, please enable this setting to make it work

Show resolve tab in user home

The Resolve tab in the user's home page allows users to view and manage tickets that have been marked as resolved or are ready for resolution.

Disable CC notifications

The Disable email notifications to all CC users option prevents email updates from being sent to users listed in the CC field for ticket-related activities.

Disable Popup Alert

If the user enables the "Disable New Ticket Popup Notification" option, the notification popup will not appear after creating a ticket.

Reopen tickets without adding comment

This option allows agents or users to reopen a closed ticket without needing to add a new comment. It helps when a ticket was closed by mistake or if the issue persists, avoiding unnecessary repetitive comments and saving time.

New ticket creation for users

Toggle this option to allow or restrict end-users from creating new tickets. When enabled, users can log new tickets from the portal. When disabled, the option is hidden and only admins or agents can create tickets. This is useful for controlling ticket inflow or pausing user submissions temporarily.

Show Latest Comment with Ticket Title

When enabled, the latest comment on each ticket will be displayed alongside the ticket title. This gives agents a quick overview of the most recent updates without opening the ticket. However, enabling this feature may impact initial loading performance if you have a large number of tickets.

Disable default approval email to Admins

Disable approval notifications for default admin. admin will no longer recieve ticket approval alerts.

Display Help Page

Admin could enable the help page from the general settings, and users would be able to see the help option on the side navigation panel.

Single line ticket form

The single-line ticket form option allows users to quickly create a ticket by entering only the essential details, like the ticket name or short.

Language Translation

To enable language translation, please create a Translator resource in the Azure Portal. Once the resource is set up, enter the endpoint API key, and region details below. This service uses Microsoft’s Translator API to provide accurate and reliable multilingual translation capabilities

    Use Sharepoint URL for MS Teams installed application

    The option Use SharePoint URL for MS Teams installed application enables the integration of SharePoint links within the MS Teams application, allowing users to access SharePoint resources directly from Teams.


    Enable the “Use SharePoint URL for MS Teams installed application” toggle, then enter your SharePoint site URL in the following format:
    https://yourtenantname.sharepoint.com/sites/SiteName/SitePages/PageName.aspx
    After entering the URL, click the Save button to apply your changes.


Language

This section allows you to manage translations for all supported languages in the helpdesk system. You can view system text such as labels, messages, and menu items in multiple languages (e.g., English, French, German, Arabic, Chinese, etc.). Each row shows the equivalent phrase in different languages, and you can edit them to customize wording as needed. Use the search option to quickly find a phrase and update its translation across languages.

    Naming Convention

    From here, you can rename the display title, company name, and application title. Click the Configure button of the Naming Convention to open the Naming Convention page.


    Enter the desired name and click on the save icon.


    Use Teams Code as Ticket Suffix: You can use the team code as a suffix for ticket IDs. Toggle this option on or off as needed. When enabled, the team code will be added at the end of the Ticket ID.



    Teams Name in New Ticket Page: Enable this option to display the team name instead of the team code on the New Ticket panel or ticket form.




    Customizable Sub-Service Names: This feature allows users to rename sub-services such as L1, L2, and L3 according to their preferred naming conventions. It provides greater flexibility and alignment with organizational terminology, making the system more intuitive and user-friendly.

    Hide Sharepoint Page's Default Components

    From here you can hide sharepoint page default Hide side navigation panel in SharePoint page: To Enable this toggle you can hide the navigation.

    Hide topbar in SharePoint page: To Enable this toggle you can hide the topbar in Sharepoint Page.

    Make SharePoint page layout in full width: Make your SharePoint page width in full screen for larger resolution where app is running.

Recycle Bin

The deleted tickets will display in recycle bin, the admin can restore them or delete them permanently.


Default Date Range of Tickets on homepage

The Default Date Range of Tickets on Homepage setting allows users to view tickets from the current month up to one year, streamlining access to relevant ticket data.


Enable logs collection

The Enable logs collection option allows the system to gather logs for tracking and troubleshooting purposes; if not needed, you can turn off the toggle to disable log collection.


    Brand Logo

    From here admin can add or update the company logo. The updated logo is displayed at the top of the left-side navigation panel. The recommended size of the logo is 150 X 60 with the same aspect ratio along with a transparent or white background is recommended for a better visibility and experience.

    favicon

    Toggle on to display the favicon. The favicon appears on the left side of the browser tab, helping users easily recognize the website.

    Customize the favicon

    The Customize the favicon option allows users to change the small icon displayed in the browser tab for the helpdesk application, enhancing branding and personalization.
