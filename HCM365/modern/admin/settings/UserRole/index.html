    User, Roles And Permissions

    Add/Edit Users: Allows administrators to add or modify user accounts within the system. Administrators can manage access levels, assign roles, and define permissions for each user.


    In the "Add/Edit User" form, the "Leaves Per Year" section allows you to configure annual leave entitlements, including PTO (Paid Time Off), Sick Leave, and Comp Off (compensatory off for extra work). The "Work Week" section defines the employee’s standard working hours for each day of the week, typically set to 8 hours on weekdays and 0 on weekends. Below this, you can assign the user specific roles across multiple applications (like HR365, ED365, TM365, etc.), ensuring they have appropriate permissions based on their responsibilities in each system.


    A user can be assigned different roles for different applications.
    These roles define the level of access and permissions granted within each application.

    • Admin – Manages system configurations, user roles, and overall platform settings.
    • Approver – Reviews and approves requests, tasks, or documents assigned for verification.
    • Coordinator – Handles the coordination of team activities, schedules, and task progress.
    • Finance – Manages budget, expense tracking, and other financial operations.
    • Finance Head – Oversees the entire finance department and approves financial strategies.
    • Global Admin – Has top-level access across all modules and settings in the system.
    • Hiring Manager – Responsible for interviewing and making hiring decisions for open roles.
    • HR – Manages employee records, policies, and general human resource tasks.
    • HR Executive – Supports HR operations including onboarding and employee engagement.
    • HR Head – Leads the HR department and sets strategic direction for people management.
    • Interviewer – Conducts interviews and submits evaluations of potential candidates.
    • Invoice Approver – Reviews and approves submitted invoices for further processing.
    • Invoice Manager – Manages invoice creation, submission, and financial reconciliation.
    • Manager – Oversees team activities, monitors performance, and ensures task completion.
    • Program Manager – Handles multiple related projects and ensures program alignment.
    • Project Manager – Leads individual projects and ensures timely delivery and reporting.
    • Project Observer – Views project progress and reports without editing access.
    • Recruiter – Sources, screens, and shortlists candidates for various roles.
    • Recruiters Manager – Manages recruiting team performance and overall hiring strategy.
    • Screener – Conducts initial resume reviews and pre-qualifies candidates.
    • User – Has limited access for task execution, assigned workflows, or viewing data.



    Full Sync for Users: This feature allows administrators to synchronize all user data from an external directory or source (such as Microsoft 365) into the system. It ensures that user information like names, emails, and roles are up to date and accurately reflected in the application.


    Update from M365: This option allows administrators to refresh or update existing user details from Microsoft 365. It syncs only the latest changes (such as name, email, or role updates) for already added users without performing a full sync.



    Click on Setting Icon.

    • Overwrite user details: If enabled, it updates existing user details during sync.
    • Exclude Options / Sync M365 Groups: Choose to either exclude specific user types or sync users based on M365 groups.
    • Exclude Sign in Blocked Users: Prevents blocked users from being synced.
    • Exclude Shared Mailboxes: Shared mailboxes will not be included in the sync.
    • Exclude Unlicensed Users: Filters out users without any license.
    • Exclude Licensed Users: Excludes users with licensed or free license and a valid subscription.
    • Exclude Free Licensed Users: Removes users who only have a free license.
    • Exclude User by Department: Allows filtering users by department.
    • Exclude User by Location: Lets you exclude users based on their location.
    • Exclude User by Title: Filters users by their job title.
    • Excluded Domain: Prevents users from specific domains from being included.
    • Exclude Users by Name: Specific user names can be excluded from syncing.


    Apps Settings

    All applications integrated into the system will be listed under the "App Settings" section. Users can access the settings of any specific application by simply clicking on the corresponding app name. This will redirect the user to the configuration page of the selected application, where various settings and preferences can be managed.
