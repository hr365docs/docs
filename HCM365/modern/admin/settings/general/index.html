    General

    The General Settings section in HCM365 provides fundamental configuration options for system customization. It includes settings related to appearance, localization, and branding. Users can adjust preferences such as themes, date formats, and currency. Additionally, it allows customization of SharePoint components and favorite icons. These settings help personalize the user experience and maintain consistency across the platform.


    Theme

    HRMS Application has the theme option where the user can select Dark Mode, Light Mode, or Site Mode. The theme will then change accordingly. Users can also switch between modes based on their preference for better visibility and comfort.

    Within HRMS, Users will have the ability to customize the theme of each application individually.



    Date Format

    Users have the ability to configure the date format globally through this option. Should the user choose the MM-DD-YYYY format, it will be universally applied across all applications within the HRMS ecosystem.


    After selecting, the user can click the Save button to save the changes.


    Default Currency

    Similar to the date format, the user can select a default currency. For example, if the user selects USD as the default currency, it will be applied consistently across all applications within HRMS for transactions, reports, and any financial data.



    Favorite icon


    The favicon is displayed on the left side of the browser tab, allowing users to easily identify the website. If the toggle is turned on, the icon will appear; otherwise, it will not be shown.

    Brand logo

    Here you can add your company logo, which will be displayed at the top of the navigation bar. From here, the admin can add or update the company logo. The updated logo will appear at the top of the navigation bar. The recommended size for the logo is 150 X 60, with the same aspect ratio, and a transparent or white background is recommended for better visibility and user experience.


    Click on the 'Upload Logo' button, then select the logo file. After selecting, the user will have the option to crop the logo, allowing them to adjust the image as needed.


    After cropping, the user can click on the 'Save Logo' button. Once saved, the logo will be visible on the top bar.



    If the user wants to remove the logo they added and upload a new one, they can click the 'Remove' button. A confirmation prompt will appear asking 'Yes' or 'No.' If the user selects 'Yes,' the logo will be removed.
