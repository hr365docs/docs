  • Home Page

      • The JoiningCard 365 Home Page provides a comprehensive and user-friendly interface to manage and display new employee introductions within SharePoint. It offers two distinct view modes—Detail List View and Card View—enabling users to switch seamlessly between structured data management and engaging visual presentations.


      Detail List View: Displays all employee records in a tabular format for easy sorting, filtering, and management. Each entry includes essential details such as Name, Email, Role, Joining Date, Skill, Bio, and Active status.



      Add Member: Click the + Add button to open the Add Member form. Enter details such as Name, Email, Role, Joining Date, Skills, and Bio. You can also upload a profile image and toggle the Active option to display the employee on the welcome board.



      Upload and Crop Image: The platform supports profile image uploads, allowing users to crop and preview the image before saving. This ensures a clean, consistent appearance across all employee cards.



      Actions: You can edit user details or delete a user if they are no longer required.


      Card View: Presents new joiners in an elegant and welcoming layout. Each card displays the employee’s name, role, bio, and profile photo, along with a personalized welcome message. This visually engaging format promotes a positive first impression and strengthens team connection.



      Comments Section: Team members can share their warm wishes and messages of congratulations to welcome the new employee. This feature helps create a collaborative and inclusive company culture.



      Settings: Administrators can configure the display preferences, theme (Light, Dark, or Site), and permission controls to tailor the JoiningCard 365 experience according to organizational policies.



      Help: The Help section provides key product information including version details, licensing, and support links. It ensures users have access to resources like FAQ, Support, and Release Notes for smooth usage and maintenance.
