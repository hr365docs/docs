• How to connect SharePoint list to Power BI


    1. Firstly, Download and install Power BI Desktop.
    2. Open Power BI Desktop and in the Home tab of the ribbon, select Get data


    3. After clicking on Get data, click on More.


    4. Select Online Services, then select SharePoint Online List.


    5. Find the address (also known as a URL) of your SharePoint Online site that contains your list.

    6. In Power BI Desktop, paste the address into the Site URL field in the open dialog box and click OK.


    7. You may or may not be connected to Microsoft account. To connect your Microsoft account, Select Microsoft account on the left side of the page.


    8. Select Sign In and enter the user name and password you use to sign in to Microsoft 365.


    9. When you finish signing in, select Connect.


    10. Select Load. Power BI loads your list data into a new report.

    11. On the left side, select the Data icon to see that your SharePoint list data was loaded.


    12. Select the report icon and start visualizing your report

    13. Select columns you want to visualize by selecting the checkbox beside them in the Fields pane on the right.
