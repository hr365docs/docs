  • Home Page

    • The Request 365 Home Page provides a simple and user-friendly view of all requests. Users can easily create, edit, or delete requests, and each request clearly shows key details such as Title, Contract Type, Counterparty, Currency, and Status.


    Create a Request

    To begin, click on the + New Request button. A request panel opens where you enter information such as contract type, counterparty, currency, and related details. Once submitted, the request will appear under In Progress.



    Review & Approval

    When a user submits a new request, it first appears in the In Progress section. This means the request has been created but is still waiting for review and approval.


    The request is then sent to the manager or designated approver. The approver reviews the details and can either Approve, Reject, or Send Back for More Information. Until the approver takes action, the request will remain in In Progress.



    Once the manager or approver approves the request, its status changes to Active. This means the request is confirmed and the contract or process can now move forward. Active requests are visible under the Active section for tracking and further actions.


    With every active request, tasks may be assigned to team members. These tasks are displayed under My Tasks, helping users focus on their individual responsibilities. Tasks are grouped by Due, Overdue, and Completed for easy tracking.



    Step 5: Edit or Delete Requests

    Users can edit details of a request when updates are required, or delete requests that are no longer needed. These actions ensure that the request list always reflects accurate and current information.




    The Request 365 Settings panel allows administrators to configure global options for appearance, permissions, and request management. These settings ensure consistency across the organization and provide flexibility in how requests are displayed and processed.
