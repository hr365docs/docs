            Custom Settings

              From here you can theme color, date format, rename display title for teams and company name, save link validation, Integrate Asset Management Plus with Helpdesk Plus, Graph API Integration, and company or brand logo.

            • Date format: If you change the date format, it will update in entire application.

            • Display title for teams: If you change the title, it will update in entire application. You can find update display title just below the user’s option from where you can create departments or groups. The created one will display in new ticket panel which can be selected during ticket submission.

            • Company name: If you update company name, it will update in closed and resolved emails notification.

            • Save link validation: Here you can add the number of the days to the survey link expire.

            • Integrate Asset Management Plus with Helpdesk Plus: You can integrate HR365 Asset Management application with helpdesk plus application, make sure both applications should be installed in same site location. Once you integrate you will find assigned asset to user in comment page as shown in below. you can click on ticket id in home page to view the comment page.


            • Graph API integration: If you configure graph API, Ticket mailbox works with help desk email ID.

            • Use teams code as ticket suffix: Just enable the toggle switch for Use Teams code as ticket suffix.

            • Display resolved tabs in home page:Here you can enable or disable all resolved related tabs in home page.
