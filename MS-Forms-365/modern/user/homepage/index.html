  • Home Page

    • The MS Forms 365 Home Page displays all forms available for the user. These include personal forms shared with the user as well as group forms published within organizational teams.


    Users can switch between List View and Card View by selecting the view toggle icon. This helps users choose the layout they find most convenient for browsing forms.


    In Card View, each form appears as an easy-to-read card showing key information. This view helps users quickly identify forms based on title, creator, created date, and description.


    Users can click on any form title to open the survey directly in Microsoft Forms, where they can fill in and submit their responses.


    When a form is opened, users are taken to the Microsoft Forms response page, where they can read the description, complete questions, and submit feedback.


    The user home page includes access to Help, where users can find information related to usage, license, product version, and support links.


    Overall, the user Home Page provides a simple, intuitive space to browse, open, and respond to forms efficiently.
