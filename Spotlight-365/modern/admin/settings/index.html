Settings

The Spotlight 365 Settings panel allows administrators to configure global options such as themes, banner content, user permissions, dismissal behavior, and SharePoint components. These settings help customize the web part’s appearance and control access effectively.

Theme

Select a visual theme for Spotlight 365:

  • Light: Applies a bright, clean interface.


  • Dark: Optimized for low-light environments.


  • Site: Inherits the theme of the SharePoint site.

Banner Configuration

The Banner Configuration section allows administrators to customize the banner displayed on the home page. Options include:


  • Title & Description: Add a heading and supporting text to communicate key information.


  • Button (Learn More): When the Show Call to Action toggle is turned ON, you can configure the button text and add a destination link for the call-to-action.


    • The Layout option in Spotlight 365 allows administrators to choose how the spotlight content is displayed. Two layout styles are available:


    • Banner Layout: Displays the spotlight as a large banner with title, description, and call-to-action button for maximum visibility.


    • Standard Layout: Displays the spotlight in a more compact, card-like format, suitable for balanced page designs.


    • Text Color: Customize the font color of the banner title and description to match your site’s branding.


    • Background Type : Choose how the banner background should appear. Two options are available:

      • Background Color: Select a solid color to apply as the banner background.


      • Background Image: Select the image radio button and then click + Upload Image to add a custom background image for the banner.




    • Alignment: Choose left, center, or right alignment for text and button placement.


    The Audience Targeting option allows administrators to display the Spotlight banner only to specific users or groups. This helps in delivering targeted communication to the right audience.


Users

The Users section allows administrators to manage permissions in Spotlight 365. The user management grid provides the following functionality:


  • User Grid: Displays Name, Email, and Role (Admin/User) for each entry.


  • Click on + Add to add new user.

  • Role Assignment: Administrators can assign or update roles (Admin/User) via the dropdown.


  • Search & Pagination: Search by Name, Email, or Role, and adjust the number of entries displayed (10, 25, or 50 per page).


Dismissal Settings

Here where administrators can reset or delete saved banner content and configurations.


Hide SharePoint Component

The Hide SharePoint Component option allows administrators to remove default SharePoint web part controls, providing a cleaner experience focused on Spotlight content.
