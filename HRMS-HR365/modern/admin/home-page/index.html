    Home Page

    This is the HRMS homepage, where all applications integrated within HRMS, such as ED365, TM365, EO365, TS365, ET365, and RM365, are displayed on the top bar with their respective icons and names. Additionally, these applications are also visible below the top bar for easy access. Beneath this section, the user dashboard is displayed, providing a comprehensive overview and quick access to key information.


    Task and Approvals

    In the Task and Approval section, the current user will be able to view their assigned tasks along with any pending approvals that require their attention.


    The current user will be able to view the count of their assigned tasks.


    The current user will be able to view the count of their pending approvals.


    The user will see an action button in the Task and Approval section. When the user clicks on it, a left-side panel will appear.


    Inside the panel, the user will have access to tabs for Tasks and Pending Approvals.

    Click on the Tasks tab, the user can view details of their tasks, including the task name, type, and source.


    Click on the Approvals tab, the user can view details of their approvals, including the approval name, type, and source.

    Out Today

    In the "Out Today" section, users will be displayed who have their leave approved for today and are currently on leave. Users whose leave approval is still pending will not be shown in this section.

    Leave balances

    In the Leave Balances section, users can view their available leaves, used leaves, and remaining leaves.


    The user will see the count of their PTO (Paid Time Off), showing the total available, used, and remaining balance.


    The user will be able to see the count of their Comp Off Leaves, including the total available, used, and remaining balance.


    The user will be able to see the count of their Sick Leave, including the total available, used, and remaining balance.


    Next Holiday

    In the **Holiday** section, the user can view their next holiday with details, including the date and holiday name.


    My Courses

    In the **My Course** section, the user can view their courses. By clicking on a course, the user will be navigated to the same course for further details.


    Quick Links

    In the **Quick Links** section, the user can see a list of important links for easy and fast navigation to frequently used pages or features.


    Organization

    In the Organization section, users can create posts such as announcements, praises, and polls to engage and communicate with others within the organization. A select field is available to indicate that the post is being made to the organization.


    By clicking on the Post tab, users can create a post where they can add a title, description, emoji, and imagebefore publishing.


    After adding all the details, the user can click on the Upload button. A confirmation prompt will appear, and upon clicking the Confirm button, the post will be successfully uploaded.


    The uploaded post will be displayed below for the user to view.


    In the Announcement section, the user can add a title, followed by detailed information about the announcement. The user can select checkboxes to notify employees and to require acknowledgment, ensuring recipients confirm they’ve read the announcement. Additionally, there's an option to hide the post after a specified date by marking the corresponding checkbox.


    After completing all the necessary details, the user can click the Upload button. A confirmation prompt will appear, and upon clicking the Confirm button, the announcement will be successfully uploaded.


    The uploaded announcement will be displayed below for the user to view.


    In the Praise section, the user must first enter the recipient's name, followed by details explaining the reason for the praise. After that, the user needs to select a badge and can also add attachments if required.


    After adding all the details, the user can click the Send button. A confirmation prompt will appear, and upon clicking the Confirm button, the praise will be successfully sent.


    The uploaded praise post will be displayed below for the user to view.


    In the Poll section, the user can first enter a question and provide some additional details. Then, the user can add options for the poll. By default, there will be two option fields available. If the user wants to add more options, they can click the Add Option button to add additional fields for options.


    The user will also see a delete option icon next to each poll option. If the user wants to remove any option, they can click on this icon to delete it.


    The user can also set an expiration date for the poll. Additionally, they can check the Notify Employees checkbox to notify employees about the poll, and the Anonymous Poll checkbox if they wish to make the poll anonymous.


    After filling in the poll details, the user can click on the Submit button. A confirmation will appear, and if the user clicks Confirm, the poll will be successfully created and shared with the selected recipients.


    After the poll is uploaded, the user will be able to select their option. Additionally, a Clear Vote button will be visible, allowing the user to remove their selection and vote again if needed.
