Users, Roles and Permissions

In this section, administrators can allocate roles to users and create teams


    • Onboarded Users
    • Add/Edit Teams
    • Teams Permissions
    • Side Navigation User Permissions
    • Enable My Profile Access for ESS

    • Onboarded Users

    • Click on the + icon to add a new user.


    • Fill in the required user details in the form.


    • Once the details are entered, click on the Save button to complete the onboarding process.


    Add/Edit Teams

    • Select Configure to continue.


    • Add the required teams and assign multiple members to each team as needed. The respective teams will appear as Pre-Onboarding Approvers.


    • Click on the Save button to finalize the configuration.


    Teams Permissions

      In Teams Permissions, you can grant other teams access to view only the selected tabs during employee onboarding.

    • Select Configure to continue.


    • As shown in the image, when all tabs are selected in the Basic Information section, only the IT form will be visible to those teams.


      • The same applies to other tabs such as Job Information, Group Membership, and more.

    Side Navigation User Permissions

    • Allows administrators to control which menu items are visible to specific user groups in the side navigation panel.


    Enable My Profile Access for ESS

    • By enabling the My Profile feature in Employee Self-Service (ESS), users gain access to MyDelve, allowing them to view and manage personalized profile information.
