      Reports

        We have four options within the report: All Reports, Project Reports, My Teams, and Invoice Report.


        The Daily Report offers a detailed breakdown of data for a single day. This report is ideal for monitoring daily activities, performance, and issues that may arise.


        The daily report will display all date columns and show all records for the month, filtered by user, client, Department, Status, project and task.


        A User Status column has been added to the report pages, displaying whether a user is active or inactive. The status information is directly fetched from the user page.


        The Weekly Report provides a summary of data and key metrics for the week. It highlights trends, changes, and performance indicators on a weekly basis, enabling users to track progress and make comparisons between weeks.


        Weekly report will display all date columns and show all records for the week, filtered by Week, Client, Project, Task, User, Project Manager, Department, and Status.


        The Monthly Report provides a summary of data and performance metrics over the course of a month. This report is useful for identifying long-term trends and understanding broader performance changes.


        Monthly report will display all date columns and show all records for the month, filtered by Month, Client, Project, Task, User, Project Manager, Department, Status, and Assigned To.

        .

        The Full Report is a comprehensive view of data across all time frames (daily, weekly, monthly). It allows users to analyze a more complete dataset for in-depth analysis and decision-making.


        Full report will display all date columns and show all records for the selected date range, filtered by Date Range, User, Client, Project, Task, Assigned To, Department, and Status.


        Project Reports

        The Project Report Page provides a centralized view of all ongoing and completed projects. Users can view a list of all projects, and by clicking on a specific project, they can access a detailed summary of that project, including key information and relevant data.


        My Teams

        The Team Page provides an overview of the team’s activities, enabling users to track and monitor team performance and engagement. It is designed to give a snapshot of the tasks, goals, and progress of each team member or the team as a whole.


        The Project section provides a comprehensive overview of all ongoing and completed projects, including key details such as project status, timelines, and objectives. This helps stakeholders monitor progress and make informed decisions based on project performance.


        The Team Members section lists all individuals involved in various projects, detailing their roles and responsibilities within the team. This facilitates better collaboration and accountability by showcasing each member's contributions and the time dedicated to project tasks.


        The Invoice Module provides a comprehensive view of all invoices within the application. It allows users to view, manage, and track invoices efficiently, helping streamline billing and payment processes.
