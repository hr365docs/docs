Installation of HRMS-HR365 from Microsoft Teams

    Prerequisites:

  1. A global admin or SharePoint admin is capable of installing the application.
  2. The application's features function through APIs, and global admins have the authority to enable these APIs.

Please read this Article before Installing or adding the App in Microsoft Teams.

Only SharePoint Administrator can install the applications. Administrator can make a member an admin.


  1. Once the application has been deployed on the sharepoint store, choose the app and select the Sync to Teams option from the files menu.


  • Teams app permission policies


      1. Sign in to Teams admin center with the below URL.

        https://admin.teams.microsoft.com/


      2. Under Teams apps select the Setup policies.


      3. From the list of policies click on the Global policy to open it.


      4. Toggle on upload custom apps and User pinning


      5. Click on Add Apps to install an app.


      6. Search for HRMS-HR365 that you want to add.


      7. Click on the Add button.


      8. Choose that application, then click the Save button.


      9. When the pop-up appears, click on the Confirm button.



      10. To pin the app, select the desired app and click the "Save" button.
