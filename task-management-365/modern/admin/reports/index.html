    Reports


    The Reports section provides a centralized view of all Tasks and Approvals within the system. Users with Admin or Supervisor roles can filter, view, and export detailed information directly from this interface.


    1. Task Report

    The Task Report provides a complete overview of all tasks within the system. Each task entry is interactive—clicking on a task name opens its detail panel, where you can update progress, assign users, change priority, or add comments.



    Use advanced filters such as date range, priority, progress status, or custom labels to tailor the report view according to your workflow needs.


    2. Approval Report

    The Approval Report lists all approval requests submitted and received across your workspace. This includes pending, approved, rejected, and reassigned approvals, giving complete visibility into the decision pipeline.


    Clicking on an approval name opens a detail view, where authorized users can review the context, add comments, and take actions such as Approve, Reject, Hold, or Reassign.
