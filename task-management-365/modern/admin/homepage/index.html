Task Management 365
  • Admin Guide

  • Home Page

      • Task Management 365 provides a centralized home page that allows users to easily manage personal and team workloads, monitor projects, and handle approval workflows effectively.


      My Tasks

      Personalized list of all tasks assigned to the logged-in user, categorized for better clarity.

      All: View all assigned tasks in one place.


      Current: Tasks that are actively in progress or due soon.


      Upcoming: Tasks scheduled to begin in the future.


      Overdue: Tasks that have missed their deadlines.


      Completed: Tasks that have been finished and marked as complete.


      Click on the + Add button to create a new task.


      Fill in the required details, assign the task to the appropriate user, and then click the Add button to save it.


      Team tasks

      A consolidated view of tasks assigned to various team members. Useful for leads and managers.

      • All: All tasks across the team.
      • Current: Ongoing team tasks.
      • Upcoming: Tasks scheduled for future dates.
      • Overdue: Team tasks that are delayed or past due.
      • Completed: Tasks that have been finished by team members.

      Projects

      Projects are containers for multiple tasks. Tasks can be grouped under personal or team projects.


      My Projects: Projects created or owned by the logged-in user.


      Click on the Project Name to view all the tasks associated with that specific project.


      Use the List View to access detailed information about each task in a structured, easy-to-read format. You can update the task details directly from this view and click the Save button to apply changes.


      Click on any task to view its details. You can update the task's progress, add comments, and mark it as Completed when done.


      Switch to the Board View to manage tasks using a drag-and-drop interface for better workflow visualization.


      Click on the project options menu to rename, move, or delete the project as needed.


      To add a new task under a project, select the + Add Task button within the desired project, fill in the necessary details, and click Add to create the task.


      Access Charts to analyze progress, task distribution, and team performance insights.



      Open the Schedule View to visualize all tasks on a calendar and manage time effectively by tracking due dates and workload.


      Team Projects: Projects shared among team members or managed by a group.


      Approvals

      Tracks all approval requests and their current status.



      Received: Requests received for approval from other users.


      You can directly select the request and Approve or Reject it, or click on the request title to view more details.


      Once opened, you'll see options to Reply, Approve, Reject, Hold, or Reassign.


      The panel also displays Properties. You can update any required details and then click the Update button to save the changes.


      You can connect with the requester through various contact options provided in the panel.


      Requested: Requests initiated by the user that require others' approval.


      Click on + Requested to create a new approval request. A form will open where you can specify the details then click on save button.



      Navigate to the Template section to manage approval workflows efficiently.


      To create a new template, click the + Template button.


      You can either choose from existing templates for reuse or build a new one from scratch to suit your specific approval process.
