    General Administration

    Configure buckets, set task priorities, and manage clients and projects under General Administration.


    Priority

    Set up task priorities to help users categorize and manage tasks effectively based on urgency and importance.


    To create a new priority, click the + Add button.



    Enter the priority name, icon name, and optionally select a color. If the priority should be active, enable the toggle and click the Save button.



    To edit an existing priority, click the Edit icon under the Action column.



    Update the desired fields, then click the Update button to save changes.



    The configured priorities will be available when creating or modifying tasks.


    Buckets

    Buckets help organize tasks into categories for better tracking and workflow management.



    To create a new bucket, click the + Add button.



    Enter the bucket name, Enable the toggle if it should be active, then click Save button.



    To modify a bucket, click the Edit icon under the Action column.



    Update the required fields and click the Update button to save your changes.



    Buckets will be available when assigning or grouping tasks.


    Clients

    Clients represent the businesses or individuals for whom projects and tasks are managed.



    To create a new client profile, click the + Add button.



    General: Enter the client's name, Tax ID, registration number, and other relevant business information. Click Save and Continue to proceed.



    Contact: Provide the Primary Contact details including name, email, phone number, job title, and contact type. Optionally, add a Secondary Contact. Once completed, click Save and Continue.



    Documents: Upload any necessary documentation related to the client. Click Save to finalize and create the client profile.



    To edit an existing client, click the client’s name from the list, make the necessary changes, and click the Update button to save your changes.



    Clients are used to associate projects and tasks for billing, reporting, and organization purposes.


    Projects

    Projects help you group related tasks under a single initiative for better visibility and planning.



    To create a new project, click the + Add button. If you wish to add multiple projects at once, use the Bulk Add option.



    General: Provide the project name and link it to the respective client. Configure essential settings such as project start/end dates, description, and any custom preferences. Once all details are filled, click Save to continue.



    People: Assign key project roles such as Program Manager, Project Manager, and Project Observer. Add the relevant team members who will be working on the project. After assigning roles, click Add Project to proceed.



    Billing: When the Billing toggle is enabled in the General section, the following options become available to configure how the project is billed.



    Mandatory Billing: Enable this option to ensure users cannot log time without associating it with a billable item. This helps maintain billing compliance and accuracy.



    • Select a Billing Method: Only one of the following billing options can be enabled at a time:
      1. Activity/Role-Based Rate: Applies different hourly rates based on either the type of activity or the user’s role in the project. Useful for role-specific billing.
      2. Individual User Hourly Rate: Assigns a custom hourly rate to each user participating in the project. Ideal for personalized billing agreements.
      3. Project-Based Hourly Rate: Sets a flat hourly rate for all users and tasks in the project. Suitable for simplified billing scenarios.
    • Specify a total budget for the project to help manage and track financial limits.
    • Once the appropriate billing option is selected then click Add Project button



    To edit a project click on Project Name then Update the project information as needed, then click Update.



    Click the three dot on a project card to view available actions: Edit, Archive, Disable, and Delete.



    To make changes to a project’s details, click on the Edit option.



    Click Archive and give necessary reason to move the project to the Archived Projects section.



    To restore an archived project, go to the Archived section, select Unarchive to return it to active status.



    The Disable option temporarily deactivates the project. It will not appear in active project selections until re-enabled.





    Click Delete to permanently remove the project. Use this with caution as deleted projects cannot be recovered.



    Projects will appear during task creation and in reports for filtering and grouping tasks.



    Delegation

      The Delegation feature in Task Management 365 365 allows administrators and users to temporarily assign their responsibilities to another user. This ensures continuity of the onboarding process when the original approver or responsible user is unavailable.

    • Open the Delegation panel to view or create delegation records.



    • Click on + Add to create a new delegation entry.



    • Administrators can create delegations for any user, while standard users can only delegate their own responsibilities.



    • From the list, choose the delegatee who will temporarily manage tasks or approvals.



    • Define the start date and end date for the delegation period.



    • Use the Active toggle to enable or disable the delegation. Once all details are filled in, click Submit to finalize.
