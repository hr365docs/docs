    General

    The General Settings section in task-management-365 provides fundamental configuration options for system customization. It includes settings related to appearance, localization, and branding. Users can adjust preferences such as themes, date formats, and time formats. Additionally, it allows customization of SharePoint components. These settings help personalize the user experience and maintain consistency across the platform.


    Date Format

    Users have the ability to configure the date format globally through this option. Should the user choose the MM-DD-YYYY format, it will be consistently applied throughout the Task Management 365 application.


    Time Format

    Users can choose between 12-hour and 24-hour time formats. This preference will be applied across the Task Management 365 application wherever time is displayed.



    Theme

    Task Management 365 Application has the theme option where the user can select Dark Mode, Light Mode, or Site Mode. The theme will then change accordingly. Users can also switch between modes based on their preference for better visibility and comfort.

    GCC Tenant

    Click the toggle to enable GCC support. After toggling, click Save to apply the changes.


    If your Microsoft 365 tenant is part of the GCC (US Government Community Cloud), please enable this setting to ensure full compatibility and functionality within the Task Management 365 application.



    Show Fav Icon

    Use the toggle switch to show or hide the favicon. After making your selection, click Save to apply the change. The favicon appears on the left side of the browser tab, helping users easily identify the website.



    Language

    Language Selection as a Browser Default Language: When the user enables the "Browser Default Language" option, the application will automatically use the browser's default language.


    The default language is English, but you can change it to your preferred language.


    Hide SharePoint Page's Default Components



    Remove Custom Webpart CSS: Clears any applied custom styles on web parts.

    Hide WebPart Title: Hides the title section of each web part.

    Hide Side Navigation: Removes the left navigation panel for a cleaner layout.

    Hide Top Command Bar: Hides the action bar with options like New, Edit, etc.

    Hide Site Header: Removes the site name and logo from the top section.

    Hide Comments Wrapper: Disables the comment section on the page.

    Hide Top Nav Header: Hides the global SharePoint navigation bar.

    Hide Suggested Item: Removes suggested content from appearing on the page.



    Brand logo

    Here you can add your company logo, which will be displayed at the top of the navigation bar. From here, the admin can add or update the company logo. The updated logo will appear at the top of the navigation bar. The recommended size for the logo is 150 X 60, with the same aspect ratio, and a transparent or white background is recommended for better visibility and user experience.


    Click on the 'Upload Logo' button, then select the logo file. After selecting, the user will have the option to crop the logo, allowing them to adjust the image as needed.
