    Email Template

    Task Management email templates automate communication during key actions such as task creation, assignment, status updates, reminders, and approvals. These templates ensure all stakeholders — including requesters, assignees, project owners, admins, and approvers — are promptly notified when a task is created, assigned, updated, or requires action. The notifications also cover application requests, approvals, rejections, and task reminders, enhancing transparency and accountability across the workflow.


    From here, you can update the email template name, subject, body, and placeholders.


    Notification Settings

    Notification Settings help ensure key users stay informed about task activities through automated emails and reminders. You can configure recipients, timing, and message delivery to improve task accountability and timely execution.


    Task Update Emails: Enable the checkbox for each recipient role — Requester, Assignee, and Project Manager — to receive email notifications whenever a task is updated.


    Task Reminder Notification 1: Set the number of days before the due date when the first reminder should be sent. Turn the toggle ON to activate this reminder.


    Task Reminder Notification 2: Configure the timing for the second reminder and enable it via the toggle to notify assignees in advance.


    Task Reminder Notification 3: Use this setting to send a final reminder close to the task deadline. Adjust the days and activate it as needed.


    Shared Mailbox ID: This setting uses Power Automate to fetch and validate the configured shared mailbox. Simply click Validate
