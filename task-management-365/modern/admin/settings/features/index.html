    Features

    The Features section provides flexible customization options to enhance usability across different modules. It includes the ability to define custom columns that allow organizations to capture additional data specific to their workflows.


    Custom Columns

    Approvals: Custom columns in the Approvals module allow administrators to capture additional approval-specific information, such as department tags or internal reference IDs.



    Users: Custom columns for users can be used to track additional user details such as employee ID, region, or team name, helping in better classification and reporting.



    To add a new custom column, click the + Add button. Enter the column name and select the column type (e.g., Single Line, Number). Once done, click Save to apply the changes.



    Actions: Use the Edit icon to modify an existing column’s name or type. If a column is no longer needed, click the Delete icon to permanently remove it from the list. Make sure to review dependencies before deleting any column.
