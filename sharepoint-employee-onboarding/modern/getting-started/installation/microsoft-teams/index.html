Installation of Employee On-boarding 365 from Microsoft Teams

Only SharePoint Administrator can install the applications. Administrator can make a member an admin.

  1. Go to https://your tenant name-admin.sharepoint.com/_layouts/15/online/TenantAdminApps.aspx.

  2. This displays the Apps page.

  3. Click on App Catalog as shown in below.


  4. Click on Apps for SharePoint as shown in below.


  5. This displays the Apps for SharePoint page.

  6. Click on upload, choose your file and upload the file.



  7. Once file got uploaded, select check box and click deploy.


  8. Once the application has been deployed, choose it and select the Sync to Teams option from the files menu.


  9. Go to the Sharepoint site's side navigation and select Teams.


  10. Teams will open and now click on Apps from side navigation. Click on the Employee Onboarding 365 application as shown below in the image.


  11. A dialog box will appear, select Add to a team from the dropdown.


  12. Employee Onboarding 365 will be available for the entire team, but start by setting it up in a channel.


  13. Choose the channel for the application you wish to setup. After choosing a channel, click on Set Up a Tab button.


  14. Now click on the Save button.


  15. Now click on the + icon as shown below in the image. There will be a search bar where you can type Employee Onboarding for searching it. Select the Employee Onboarding application.



  16. Employee Onboarding 365 application will be added as a tab on the channel.


  17. Click on the three dots shown in the left pane. From here you can also add Employee Onboarding 365 as a Teams Personal App.
