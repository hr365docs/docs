Task Administration

This section allows you to create and manage task categories, task templates, and define specific activity types.

    • Task Category
    • Task
    • Task Template

    • Task Categories


    • Select Task Categories to begin the setup process.


    • Click +Add to dynamically create new categories.


    • Define the Process Type as required.


    • Click the Save button to store your changes.


    • To delete multiple categories, use the Delete option.


    Task


    • Select Task to continue setup.


    • Click +Task to dynamically create new tasks.


    • Enter task details as required.


    • Click the Save button to store your changes.


    • To delete multiple tasks, use the Delete option.


    Task Template


    • Select Task Template to continue setup.



    • Click +Add to dynamically create a new task template.


    • Enter the Template Name and select the appropriate Process Type.


    • Click the Save button to store your changes.


    • To delete multiple task templates, use the Delete option.
