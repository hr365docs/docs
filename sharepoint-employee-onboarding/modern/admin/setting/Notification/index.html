Notifications

The Notifications section enables administrators to manage automated communications throughout the onboarding process. Here, you can configure email templates, set up reminder notifications, integrate task notifications through Power Automate Flow, and define recipients for onboarding attachments and shared mailboxes.



    • Email Templates
    • Reminder Notifications

    • Define Onboarding Attachments Recipient

    • Email Templates

      The Email Templates feature allows administrators to design and manage customized email formats used during the onboarding workflow. These templates ensure consistent communication with employees and teams.


    • Click on this Email Templates to proceed.


    • You can add, edit, or delete dynamic email templates that suit your organization’s onboarding communication standards.


    Reminder Notifications

      The Reminder Notifications feature helps ensure timely completion of onboarding tasks by sending automated reminders to responsible users.


    • Click on this Reminder Notifications to proceed.


    • Select the appropriate reminder options and intervals based on task due dates or statuses.


    • Click on this option to save the configuration.


    Enable Task Notification with Power Automate Flow

      The Enable Task Notification feature integrates with Microsoft Power Automate to send real-time alerts for assigned onboarding tasks. This automation ensures that users receive timely reminders based on task status or due dates, improving task tracking and compliance.


    • Click on this Enable Task Notification to proceed.


    • Once enabled, automated email alerts will be sent to assigned users when a task is created, updated, or nearing its deadline.

    Define Onboarding Attachments Recipient

      This option allows administrators to define specific users or groups who should receive onboarding-related attachments via email. These may include employee documents, agreements, or compliance forms.


    • Click on this option to proceed.


    • Once saved, all onboarding attachments will automatically be sent to the defined recipients.

    Shared Mail Box ID

      The Shared Mail Box ID setting enables centralized communication by using a common mailbox for all onboarding-related emails. This ensures continuity and visibility for HR or admin teams handling employee onboarding.


    • Click on this option to proceed.


    • All notifications and reminders will now be sent using the specified shared mailbox address.
