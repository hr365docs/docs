Courses

The Courses section enables administrators to design and manage structured learning programs for employees — composed of sections and chapters, supported by surveys and certificates, and governed by role-based access.

    • Sections

      Sections are top-level groupings for course content. Use sections to split a course into logical areas (for example: Company Overview, Policies, IT Setup).

    • Click the + Section button to create a new section. This opens a panel where you provide a title, author, and a short description.


    • In the Add Section panel, fill the required fields:
      • Title — Enter a clear section name.
      • Section Author — Select the content owner.
      • Description — Provide a short summary for learners.
      • Active toggle — Enable to make the section visible.


    • Click Save or Save and add more to continue adding more sections.

    Chapters

      Chapters are the individual content items inside a section. Each chapter holds the learning material — text, attachments, video links, or embedded media.

    • Click the + Chapter button to add a new chapter.


    • In the Add Chapter form, complete the fields below:
      • Title — Enter the chapter title.
      • Section — Select the parent section for this chapter.
      • Description — Add text, images or embed links using the rich text editor.
      • Attachment Type — Choose if the chapter requires learners to download or view attached files.
      • Timing — Enter an estimated time (in minutes) to complete the chapter.
      • Active — Toggle to publish the chapter.


    • Click Save or Save and add more to create additional chapters.

    Survey

      Surveys collect feedback from learners after course completion. Use surveys to measure satisfaction, comprehension, and training effectiveness.

    • Click + Survey to create a new survey.


    • When adding a survey question, provide these details:
      • Question Title — Short, clear prompt for the learner.
      • Question Type — Choose from Multiple Choice, Rating, Text, etc.
      • Instructions — Explain how to respond to the question.
      • Required — Toggle if response is mandatory.
      • Active — Toggle to enable the question.


    • After creation, preview responses and export results for analysis.

    Courses

      The Courses area is where you assemble a full learning program by linking sections, chapters and surveys, assigning roles, and adding visual assets (thumbnail and cover images).

    • Click + Add Course to start creating a course. The course creation is a multi-step wizard: Step 1: Basic details, Step 2: Content & Chapters, Step 3: Settings & Publish.


    • Step-by-step course creation:
      1. Step 1 — Basics: Enter Course Title, Course Description, choose Activity Type, select Roles who can access, upload Thumbnail and Cover Image.

      2. Step 2 — Content: Attach sections/chapters to the course, reorder chapters, define required/optional chapters, and set passing criteria if applicable.

      3. Step 3 — Settings & Publish: Configure availability dates, enrollment options (manual/auto), completion certificates, and enable/disable the course. Save and publish to make it visible in ESS.

    • After publishing, the course appears in the Courses catalog and is visible to learners with assigned roles.

    Certificate Templates

      Create certificate templates that are automatically issued when learners complete a course. Certificates can include placeholders (Learner Name, Course, Completion Date).

    • Click + Add Template to design a new certificate. Upload background images, add text placeholders, and preview the result.


    • Assign a template to a course in the course settings so learners receive the certificate automatically upon successful completion.

    Roles

      Roles determine who can create courses, manage content, and who can view or complete courses. Typical roles include Admin, Instructor, and Learner.

    • Click + Role to create or modify roles and set permissions for course access and editing.


    • Assign roles to courses during course creation to control visibility in the Employee Self-Service portal.
