Courses

The Courses section enables administrators to design and manage structured learning programs for employees — composed of sections and chapters, supported by surveys and certificates, and governed by role-based access.

    • Sections

      Sections are top-level groupings for course content. Use sections to split a course into logical areas (for example: Company Overview, Policies, IT Setup).

    • Click the + Section button to create a new section. This opens a panel where you provide a title, author, and a short description.


    • In the Add Section panel, fill the required fields:
      • Title — Enter a clear section name.
      • Section Author — Select the content owner.
      • Description — Provide a short summary for learners.
      • Active toggle — Enable to make the section visible.


    • Click Save or Save and add more to continue adding more sections.

    Chapters

      Chapters are the individual content items inside a section. Each chapter holds the learning material — text, attachments, video links, or embedded media.

    • Click the + Chapter button to add a new chapter.


    • In the Add Chapter form, complete the fields below:
      • Title — Enter the chapter title.
      • Section — Select the parent section for this chapter.
      • Description — Add text, images or embed links using the rich text editor.
      • Attachment Type — Choose if the chapter requires learners to download or view attached files.
      • Timing — Enter an estimated time (in minutes) to complete the chapter.
      • Active — Toggle to publish the chapter.


    • Click Save or Save and add more to create additional chapters.

    Survey

      Surveys collect feedback from learners after course completion. Use surveys to measure satisfaction, comprehension, and training effectiveness.

    • Click + Survey to create a new survey.


    • When adding a survey question, provide these details:
      • Question Title — Short, clear prompt for the learner.
      • Question Type — Choose from Multiple Choice, Rating, Text, etc.
      • Instructions — Explain how to respond to the question.
      • Required — Toggle if response is mandatory.
      • Active — Toggle to enable the question.


    • After creation, preview responses and export results for analysis.

    Courses

      The Courses module allows administrators and instructors to create, manage, and publish structured learning programs. Use the guided wizard to define course details, add content, and configure publishing options.

    • Click + Add Course to start creating a new course. The setup includes three simple steps: Step 1 – Basic Details, Step 2 – Content & Chapters, and Step 3 – Settings & Publish.


    • Step-by-Step Course Creation
      1. Step 1 – Basic Details: Enter key information such as course title, description, activity type, roles, and thumbnail. Click Next to continue.


      2. Step 2 – Content & Chapters: Add course chapters, set timing, and include assessments or surveys as needed. Click Next to proceed.


      3. Step 3 – Settings & Publish: Review and configure course settings such as skills, tags, enrollment, approvals, and ratings. Once finalized, click Create to publish the course.


    • After publishing, the course appears in the Courses Catalog and is accessible to assigned user roles.

      • The Enrollment option allows administrators to add team members or learners to a course. It provides tools to manage participation and monitor individual progress through different learning stages.

    • To begin, click the Enrollment icon to access the course enrollment panel.


    • Click + User to add new members to the course.



    • After enrollment, you can view user progress categorized under Not Started, In Progress, and Completed statuses for easy tracking.


    Certificate Templates

      The Certificate Templates feature allows administrators to select a predefined certificate design that will be automatically issued to learners upon course completion. Each template includes dynamic fields such as Learner Name, Course Title, and Completion Date.



    Roles

      The Roles section defines user permissions within the Courses module — determining who can create, manage, or access course content. Common roles include Teacher and Learner.



    • Roles can be configured and assigned directly while creating a course. In the course creation panel, you will see a Role option where you can specify which users can view, edit, or enroll in the course.


    • Assigning appropriate roles ensures that only authorized users have access to manage or complete courses within the Employee Self-Service portal.
