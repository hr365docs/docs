General Administration

Use this section to synchronize Office 365 users and manage organisational metadata such as positions, departments, locations, devices, and related settings.

Positions


    Here you can add the Positions, which are reflected in the Pre-Onboarding, Onboarding and Offboarding workflows. Positions added here are used as dropdown options and suggestions when creating employee profiles.

  • Select Position to proceed.




  • Click + Add to create a new position.



  • Use this option to import position names from the Office 365 user profiles.



  • Click the Delete button to remove multiple positions at once.


Department


    Use this area to add Departments. Departments are available across the Pre-Onboarding, Onboarding, and Offboarding workflows.

  • Select Department to continue.



To add a department, click + Departments and enter the department name.

Click the Save button to persist the department.




You may import department names from Office 365 user profiles using the import option.


Use the Delete option to remove multiple departments at once.


Locations

  • Select Location to continue.



    • After opening the Locations panel, enter the location details.

    • Click the Save button to persist the location.

    • You can import location names from Office 365 user profiles.


    • Click Import to upload locations from a file.


    • Attach the file and then click the Upload button.

      • To remove multiple locations, use the Delete option.

Software

  • Select Software to continue.



    • Click + Software to add a new software entry to the system.


    • Click the Save button to store the software details.


  • Import


  • To remove multiple software entries, use the Delete option.


Devices


    • Click + Device to add a new device type.


    • Click the Save button to store the device entry.


  • Use the import option to upload devices in bulk from a sample file.

  • To remove multiple devices, use the Delete option.


Status


    • Click + Status to add a new status value.


    • Enter the status name and click the Save button to persist it.


  • To remove multiple status entries, use the Delete option.


Employee-Type


  • Click + Type to add a new employee type (e.g., Full-Time, Contractor).


  • Click the Save button to persist the employee type.


  • To remove multiple employee types, use the Delete option.

Additional Fields


  • Click + Additional Fields to create custom fields for employee profiles.


  • Click the Save button to persist the new fields.


  • To remove multiple additional fields, use the Delete option.

Reason-for-Offboard


  • Click + Add to enter a new reason for offboarding.


  • Click the Save button to persist the reason.

  • To remove multiple reasons, use the Delete option.

Notice-Period


  • Click + Add to create a notice period entry (in days).


  • Click the Save button to persist the notice period.


  • To remove multiple notice periods, use the Delete option.

Document Settings

  • Click + Add to create a new document type or template.


  • Click the Save button to persist the document settings.

  • To remove multiple documents, use the Delete option.

Audit Logs

  • Use the Audit Logs to review changes and activity within the application.


  • Click the Save button where applicable to confirm actions.


  • To remove multiple entries, use the Delete option.

Groups

  • Include or exclude specific groups from application flows. Only included groups will be visible within the application.


Custom Forms

  • After clicking + Add, you can create Dynamic Custom Forms.


  • To add a custom form, complete the required fields. If you would like users to upload files during onboarding, enable the Add Attachments toggle and then click the save icon.


  • When enabled, the Attachments option appears within the selected custom form section and allows users to upload supporting files.



  • To remove numerous form , use the Delete option.
