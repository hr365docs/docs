Users, Roles and Permissions

The Users, Roles and Permissions section allows administrators to manage user access, roles, and team configurations within the application. This includes adding users, defining permissions, configuring teams, and enabling self-service access for employees.

  • App Users
  • Onboarded Users
  • Add/Edit Teams
  • Side Navigation User Permission
  • Enable My Profile Access for ESS

    App Users

    Administrators can add, manage, and assign roles to application users.


    Roles such as Admin, HR Head, HR Executive, Power User, Finance, Manager, and others can be assigned as shown below.


    Full Sync for Users: Synchronizes all user data from external directories (e.g., Microsoft 365) to ensure names, emails, and roles remain up to date.


    Update from M365: Updates existing user details such as names or roles without a full sync. Administrators can include or exclude groups before syncing via the Settings icon.


    Click on Settings Icon to configure sync preferences.


    • Overwrite User Details: Updates existing user data during sync.
    • Exclude Options / Sync M365 Groups: Exclude specific users or sync users based on Microsoft 365 groups.
    • Exclude Sign-in Blocked Users: Prevents blocked users from being synchronized.
    • Exclude Shared Mailboxes: Skips shared mailboxes from sync.
    • Exclude Unlicensed or Free Licensed Users: Filters users without or with free licenses.
    • Exclude by Department, Location, Title, Domain, or Name: Apply custom filters to exclude users based on these attributes.

    Onboarded Users

    The Onboarded Users section displays employees who have completed the onboarding process. Admins can review details such as onboarding status, joining date, and assigned teams.


    Administrators can filter or search for specific users and view their onboarding details for reporting and tracking purposes.

Add/Edit Teams

  • Add multiple members under each team. These teams appear as approvers during the pre-onboarding process.


  • Click Save after configuring the teams.


Side Navigation User Permission

    The Side Navigation User Permission feature allows administrators to control which sections or tabs each team can access during the employee onboarding process.

  • Select Configure to customize permissions.


  • As shown below, only the selected tabs under each section (e.g., Basic Information, Job Information, Group Membership) will be visible to the corresponding teams.



Enable My Profile Access for ESS

    The Enable My Profile Access for ESS (Employee Self-Service) option allows employees to access and view their own profile information directly from the system. When enabled, users can view limited personal details such as contact information, department, and reporting manager.
