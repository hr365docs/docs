  • Update Details

    On the update details page, there are three tabs employee details, employee confidential details and work details.
    1. Employee Details

      From here user can update the details like Date of birth, address, nationality and alternate number etc.

    2. Employee Confidential Details

      From here user can update the confidential details like father name, personal email, blood group etc.

    3. Work Details

      From here user can update work details like tax ID, social ID etc.
