Installation of applications from app catalog

Only SharePoint Administrator can install the appliications. Administrator can make a member an admin.

  1. Go to https://your tenant name-admin.sharepoint.com/_layouts/15/online/TenantAdminApps.aspx.

  2. This displays the Apps page.

  3. Click on App Catalog as shown in below.


  4. Click on App for SharePoint as shown in below.


  5. This displays the Apps for SharePoint page.

  6. Click on upload, choose your file and upload the file.



    7. Installation of App file


  7. First open your site collection where you need to install your app, click on site content and then click on Settings gear icon.


  8. You can either click on Add an app or App under New.



  9. This displays the SharePoint apps page.



  10. You will navigate to the page where ADD is visible as shown in below, click on it.


  11. And finally click on Trust it, application starts installing.


  12. Once application got installed you can check in Site contents.
