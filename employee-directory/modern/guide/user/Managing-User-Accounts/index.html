Managing User Profile

    There are many configuration options available to users once you open the link of Employee Directory application.
    You can find your profile either by search option or scrolling to the application.


    You can view your profile by clicking on your profile picture or in the top right section of the profile.



    Update Profile

    Once you have clicked to edit icon, you will be taken to the 'User Profile' which gives an overview of your profile such as date of birth, skills, hobbies etc.

    System Administrators can enable/disable the edit icon.
