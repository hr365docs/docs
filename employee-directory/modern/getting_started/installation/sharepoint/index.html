    Installation of application from SharePoint Store

    Only SharePoint admin can install the appliications. Admin can make a member an admin.

    1. Go to the tenant admin apps page from the below URL format and click on app catalog options https://YourTenantName-admin.sharepoint.com/_layouts/15/online/TenantAdminApps.aspx.


    2. Click on setting gear icon located at right top side and then Add an app.

    3. Click on New and select App.


    4. Click on SharePoint Store and type HR365 in search box. All applications from HR365 will sort



    5. Click on application which you want to install.

      6. We have to add Employee Directory Modern as web part in the page.


    6. Click on Request.



    7. Go to App Requests page and click on Employee Directory Modern .This displays app details page.


    8. Click on click here to view app details as shown below.


    9. Click on ADD IT button.


    10. Select the check box and click on Deploy.



    11. Click on New and select Page.



    12. Click on Create page.



    13. Now click on + icon and then select the Employee Directory Modern as shown in below.

    14. You can also change the title areas as plain by clicking on Edit icon and selecting Plain for better look as shown in below.


    15. Finally click on Publish button and refresh the page.
