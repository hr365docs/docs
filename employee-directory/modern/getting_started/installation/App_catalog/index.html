  • Installation of application from App Catalog

    • Only SharePoint Administrator can install the appliications. Administrator can make a member an admin.

      1. Go to https://your tenant name-admin.sharepoint.com/_layouts/15/online/TenantAdminApps.aspx.

      2. This displays the Apps page.

      3. Click on App Catalog as shown in below.


      4. Click on App for SharePoint as shown in below.


      5. This displays the Apps for SharePoint page.

      6. Click on upload.


      7. Choose and upload the file.


      8. Once file got uploaded, select check box and click deploy.



      9. Open your site collection where you need to install your application.

        10. We have to add Employee Directory Modern as web part in the page.


      10. Click on New and select Page.



      11. Click on Create page.



      12. Now click on + icon and then select the Employee Directory Modern as shown in below.

      13. You can also change the title areas as plain by clicking on edit icon and selecting Plain for better look as shown in below.


      14. Finally click on Publish button and refresh the page.
