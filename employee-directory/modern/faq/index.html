Frequently Asked Questions

What is the trial period of the Employee Directory Modern?
The trial period is 30 days.
What would happen after the trial period?
After the trial period, users limit to 50 users and all features are disabled.
What should be done to display all the users and enable the features?
You can purchase the application.
How do we purchase the Employee Directory Application?
To buy the application. Refer to Sharepoint Employee Directory
How to activate the Product?
In Employee Directory Modern please click on the help (?) icon and then on product activation put the license key and click on validate button.
From where do I get the license key?
Once you purchase the product, a purchase confirmation email notification will send to you which contains the license key. You can activate the product after 24 hours from the time of purchase.
How to add new users in the Employee Directory Modern?
You can add users from the O365 admin page or azure and users sync or appears in Employee Directory Modern after 24 hours automatically.
Can I add non-office365 users In the Employee Directory?
Yes, we can add non-office365 users. To add non-office365 follow below steps:
  • Go to the Settings.
  • In Advance, click on the import button, the panel open and download the sample file.
  • Fill non-O365 user details as per the specified format in the file. Choose the file, and click on the Save button.
    • How to edit the user profiles in Employee Directory Modern?
  • Admin can edit the user’s details from the O365 or Azure. The user’s details like birth date, joining date, skills, hobbies, projects, and about me can be updated from the SharePoint User profile. SharePoint admin can visit the SharePoint user profile from the below link URL format.

    • https://Your tenant name-admin.sharepoint.com/_layouts/15/TenantProfileAdmin/ProfMngr.aspx?
    Can I install Employee Directory Modern on different sites multiple times?
    Yes. You can install Employee Directory Modern on different sites multiple times.
    Why does the outlook image do not appear or update in the Employee Directory Modern?
    Due to CORS, the outlook image does not appear or update. You have to open outlook or delve into the same browser for one time.
    How often does Employee Directory Modern sync?
    Employee Directory modern sync unlimited times.
    Will the application stores any data?
    Yes. The application stores the data in a special list.
    How to update the application?
    To update application, see Upgrading.
    How to exclude multiple users/bulk users from CSV files?
    To exclude multiple users/bulk users from CSV files. You can follow the below steps:
  • Go to the settings, click on the Exclude Options tab.
  • Click on Upload and download the sample file.
  • In the CSV file, add the user’s name.
  • Upload the CSV file and click on the Save button.
    • How to sync mobile number using Azure API?
    To sync mobile number using Azure API, You can follow the below steps:
  • In the Settings, go to Advanced tab.
  • Enable the Mobile Sync using Azure API.
    • How to configure the flow for birthday and anniversary templates?
    To configure the flow for birthday and anniversary templates, You can follow the below steps:
  • In the Settings, go to Views tab.
  • Enable the Show joyful animations and Date of Birth.
    • What are the fields that can be edited by users?
    Users can edit the following fields:
  • Mobile Number
  • Work Phone
  • Date Of Birth
  • About Me
  • Birthday
