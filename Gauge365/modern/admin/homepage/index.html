  • Home Page

      • The Gauge 365 Home Page acts as the central workspace where users can view performance indicators and visualize data. It provides a clean and intuitive interface that highlights KPIs using different chart types like Gauge, Dial, Column, Doughnut, and Progress.


    Selecting a Chart: The Home Page allows users to choose from multiple visualization formats including Gauge, Dial, Column, Doughnut, and Progress. This flexibility ensures the right display for different data needs.


    Gauge Chart: Displays progress against defined KPIs with clear minimum, maximum, and actual value indicators.


    Column Chart: Uses vertical bars to represent data ranges, enabling quick comparisons across KPIs.


    Dial Chart: Provides a speedometer-style visualization, making performance metrics instantly recognizable.


    Doughnut Chart: Presents KPIs in a circular format, with segments representing proportions or thresholds.


    Progress Bar: Shows goal completion in a linear bar format, making it easy to track percentage-based progress toward defined targets.


    Settings: Configure global options such as theme, layout, and value ranges to tailor Gauge 365 to your reporting needs.


    Help: Access step-by-step guidance and product activation support directly within Gauge 365 for a seamless experience.
