Settings

The Gauge365 Settings panel allows administrators to customize the web part by configuring global options such as themes, layout, values, and display preferences. These settings ensure the gauge charts align with organizational reporting requirements.

Theme

Select a visual theme for Gauge365:

  • Light: Bright, clean interface.


  • Dark: Optimized for low-light environments.


  • Site: Inherits the theme of the SharePoint site.

Users

The Users section allows administrators to manage access and permissions within Gauge 365. The user management grid provides the following functionality:


  • Role Assignment: Administrators can update roles for each entry, choosing between User and Admin.


  • Automatic Entry: The details of the current logged-in user are automatically added to the grid when accessing the site.


  • User Removal: Administrators can delete users using the trash icon to revoke access.


Layout

The Layout option in Gauge365 defines how charts are presented, allowing administrators to switch between a compact view for quick insights and a detailed view for in-depth analysis.

Values

Administrators can configure minimum, maximum, and actual values to accurately represent KPIs in Gauge365. Additionally, they can specify units and enable count labels to enhance clarity and context in gauge visualizations.

When the Unit is set to Count, you can add a descriptive label to clarify what the number represents.



When the Unit is set to Currency, you can specify a currency symbol (e.g., USD, EUR).



When the Unit is set to Percentage, the gauge automatically adds the % symbol.



Next, enter Minimum, Maximum, and Actual values in the respective fields to define the gauge range and current KPI.



Finally, click the Save button to apply the configuration.






Options


In the Options section, you can configure additional settings to enhance data clarity and visualization.




Display Axis Label: Toggle this option to show or hide axis labels on the gauge.




When this toggle is ON, a Shorten Axis Label option appears, which automatically shortens large values (above 999) for better readability.



Number of Tiers: Select the number of tiers to represent different value segments on the gauge.finally click on save button
