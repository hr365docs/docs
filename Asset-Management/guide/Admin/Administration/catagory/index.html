      From here you can able to category. Here you can categorize the assets as Consumable by selecting Consumable Category checkbox.


      To add the asset type, follow the below steps:

      1. Click on the +Add.

      2. Enter the Categoryand select checkbox as per your requirements.

        Default Sub Category can be for one catagory only.


      3. Finally click on the Save button.

      4. To add multiple status, click on Save and Add more button.

      5. The Category dropdown will be displayed on the + Asset or +New asset page.


      Edit: Click the Edit button, this display Edit Category page. Update the entries and click on Update button.

      Delete:Click the delete button, this displays a dialog box. Click Yes button.
