  • Add Asset


      You can enable fields those which are required and disable those which are not required of in add assets page. Click on asset tab, add asset page will open then click on settings gear icon at right top, column display page will open. Uncheck the check boxes for disable and check the check boxes for enable fields and finally click on save button. You are not able to disable mandatory fields like asset name, asset type and category.


      For add asset page, you can setup default category, subcategory, asset type and status from side navigation panel under administration.
      We can add assets from two places.
      1. Asset tab
      2. Assets from side navigation panel under Administration

      1. Asset Tab

        Fill mandatory fields like asset name, category and asset type and others are optional fields, upload image or add document. You can also set asset expiry notification for asset just enable it and click on submit button or save icon as shown in below. Created asset fall under home and assign asset table.

        To generate system barcode please enable “System Generated Barcode” from general setting.


        Adding an Asset as Multiple Click on Add Multiple buttons, fill mandatory field and type number value to create number of asset and click on Submit button as shown in below.

        Do not type serial number for creating multiple assets.



        Import asset or Bulk Assets You can add bulk assets about 200+ in one go using CSV file. Download the sample file and add data as specified format.

        Care should be taken while adding data in CSV file.

        1. Add Asset ID in specified column and is mandatory.
        2. Remove separators from the cost / value.
        3. Use date format should be MM/DD/YYYY, this is uploading the data, while displaying it will follow as per your choice in settings.
        4. Please do not change order of the columns or add new columns, this may lead into feeding values in incorrect columns.
        5. Mandatory fields like Asset Name, Asset Id, Asset Type and Category should not be empty.
        6. Barcode should not be more than 8 digits.
        7. Please do not change order of the columns or add new columns, this may lead into feeding values in incorrect columns.
        8. Barcode should not be more than 8 digits.

        Click on asset tab then import asset, add multiple assets page will display then choose the data loaded CSV file from system and click on save as shown in below. You can add multiple assets from Bulk Asset tab of home page.



      2. Assets from side navigation panel under administration


        1. New Assets - From side navigation panel click on Asset under Administration, click on New Assets tab and follow the above step mentioned under Asset tab

        2. Delete Assets - From side navigation panel click on Asset under Administration, select the asset and click on Delete Asset tab. You can delete 25 assets at a time.

        3. Clone Asset - If you want to create new asset with minor change with the existing one or same copy of existing one you can go for clone asset. From side navigation panel click on Asset under Administration, select the asset and click on Clone Asset tab, either edit data or remain same as per your requirement and finally click on Submit button.

        4. Edit Assets - From side navigation panel click on Asset under Administration, Administration page will open, click on Edit Asset tab, edit pop page will display then click on Edit icon , you will enter in edit mode, edit the asset and finally click on Update button as or Click on hyper link of asset under Home or Assign Assets table, edit pop page will display then click on Edit icon, you will enter in edit mode, edit the asset and finally click on Update button as shown in below image.
