Users


Admin can add user to this application. There are different types of roles like, user, admin, asset manager, supervisor and no access as shown in below image.



  • User: Only able to view assigned assets to them.
  • Power user: Able to see all the assets only.
  • Asset Manager: Able to add assets, delete assets, assign assets and return assets, audit assets,report assets, book assets, edit or delete book assets and depreciation.
  • Supervisor: Able to assign assets, return assets, audit assets, report assets book assets, edit or delete book assets and depreciation.
  • No Access: User cannot access the application.

  1. Add
    To add users, click on users under settings from side panel, page will open, click on add tab, add users page will open, First select the user type (M365 users or Non M365 users). For M365 users details fill fetch from M365 automatically if exists, you can edit the details further and click on submit button. For non M365 users, need to add all details manually.
    Note: Assign & return assets notification for non M365 users works on Power Automate setup


  2. Bulk Add
    Download the sample file and fill the email Ids and role as per specified format in the sample and save it. Choose the same file and click on save button
    Note: Add only email(s) and role for M365 users, no need to add details if user’s details are available in O365. If user is M365 user, update “No” in IsNonM365User column. You can add both M365 and non M365 users from bulk add.


  3. Edit User Role
    We can edit user role for either individual or multiple users as shown in below image. Select multiple users to assign same role.
  4. Delete User
    Select the users and click on delete users.
  5. Sync from M365
    If you click on sync from M365 tab, the user's detail in table will update from O365
