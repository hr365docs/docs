General Settings


    Here you can set date format, currency, asset accept notification, system generated barcode, theme color, brand logo etc.


    1. Assets accept notification: If it is enabled, when asset is assigned to users, they will receive email notification from where use can accept/reject assets. If it is disable when asset assigned to users, email notification we not send and those assets will show in acknowledgement status page later we can initiate

    2. System generated barcode: If this is enabled, barcode generate automatically on adding assets

    3. Show projects while assigning assets: If this is enabled, project option will display while assigning assets in pop, you can assign assets to the users along with project

    4. Camera Scanner: If this is enabled, camera scanner tab will display in home page from where you can scan barcode of the assets to search the assets.

    5. External User: If you enabled it, you could add external users or guest users in the application and can assign assets to them. Note: Make sure guests or external users SharePoint site permissions should be provided where the Asset management application is hosted

    6. Assign Assets to Location: If this is enabled, location option will display while assigning assets in pop, from where you can assign assets to the location.

    7. Retain Asset's Last Location: If this feature is enabled, the user’s location will retain on available assets after returning the assets. You can see retained location in user location field of the assets.

    8. Asset Acknowledgement Text: if you type acknowledgement text, user can see in acknowledgement page (acknowledgement page link is available in assigned email notification of the user).

    9. Create PDF of asset acknowledgement: When we enable this switch and setup Power Automate, pdf file will create and stored in SP library and sent to user who has acknowledged the assets.
      https://YourTenantName.sharepoint.com/sites/SiteName/HR365AMPAssetdoc This is the URL where we can find pdf files

    10. MS Intune Integration: If you integrate it, Intune assets will sync in the application.

    11. MS group Integration: If you integrate it, distribution type groups will sync in the application

    12. Assets Advance Booking: if you disabled this asset booking feature disable from the application.

    13. Depreciation Method: You can enable depreciation feature, just selecting either Straight Line or Diminishing Balance method as per your requirement and finally clicking on save button.

    14. Assign Assets to Non M365 Users: If this is enabled, assign assets to non M365 users’ option will display while assigning assets in pop, from where you can assign assets to non O365.

    15. Enable Software: If this this is enabled, software tabs in home page will displays, one tab is used to sync O365 license another is used to add software manually and can assign to users.

    16. Display Assign Assets and Return Assets options in side Navigation Panel: You can hide assigned and return assets in navigation panel just disabling it
