General Settings


    Here you can set date format, currency, asset accept notification, system generated barcode, theme color, brand logo etc.


    1. Depreciation Method: You can enable depreciation feature, just selecting either Straight Line or Diminishing Balance method as per your requirement and finally click on Save button.

    2. System Generated Barcode: If this is enabled, barcode generate automatically on adding assets.

    3. External User: If you enabled it, you could add external users or guest users in the application and can assign assets to them.

      Make sure guests or external users SharePoint site permissions should be provided where the Asset management application is hosted.

    4. Internal Datalist Path: If this is enabled, App data can b stored in SharePoint list within the app or externally at the site/subsite collection level where app is installed.

    5. Show projects while assigning assets: If this is enabled, project option will display while assigning assets in pop, you can assign assets to the users along with project

    6. Camera Scanner: If this is enabled, camera scanner tab will display in home page from where you can scan barcode of the assets to search the assets.

    7. Retain Asset's Last Location: If this feature is enabled, the user’s location will retain on available assets after returning the assets. You can see retained location in user location field of the assets.

    8. Assets accept notification: If it is enabled, when asset is assigned to users, they will receive email notification from where use can accept/reject assets. If it is disable when asset assigned to users, email notification we not send and those assets will show in acknowledgement status page later we can initiate

    9. Assign Assets to Location: If this is enabled, location option will display while assigning assets in pop, from where you can assign assets to the location.

    10. Assign Assets to Non M365 Users: If this is enabled, assign assets to non M365 users’ option will display while assigning assets in pop, from where you can assign assets to non O365.

    11. Assets Advance Booking: if you disabled this asset booking feature disable from the application.

    12. Enable Software: If this this is enabled, software tabs in home page will displays, one tab is used to sync O365 license another is used to add software manually and can assign to users.

    13. Asset Location Mandatory: If this this is enabled, you can get the asset location.

    14. CC to Location Email Id: If this this is enabled, email notification will be sent to location.

    15. Allow Asset Manager to Add Assets: If this this is enabled, Manager can add Assets.

    16. Allow Asset Manager to Delete Assets: If this this is enabled, Manager can delete Assets.

    17. Assign & Return Assets Option in Side Navigation Panel: You can hide assigned and return assets in navigation panel just disabling it.

    18. Allow User to Book Assets: If this this is enabled, Users can Book Assets.

    19. Select Approvar to Book Assets: If this this is enabled, Users can select approver.

    20. Availability of Assets During Maintenance: If this this is enabled, Users can book asset whcih is in maintenance.

    21. Maintenance Module: If this this is enabled, you can add Maintenance Frequency and Maintenance Due Date while adding assets.

    22. Hide Asset ID Prefix: If this this is enabled, You can hide Asset ID prefix.

    23. Asset Acknowledgement Text: if you type acknowledgement text, user can see in acknowledgement page (acknowledgement page link is available in assigned email notification of the user).

    24. Barcode Customization: From here you can customize barcodr by adding Barcode Title Text, Organization Name, Asset Field Attribute and enabling Set Asset Field Attribute for Barcode.

    25. SCCM Integration:

    26. Software & MS Intune Integration: If you integrate it, Intune assets will sync in the application.

    27. MS group Integration: If you integrate it, distribution type groups will sync in the application.

    28. Enable Theme Color: You can select color as per your requirement and click on Save button.

    29. Brand Logo: You can upload logo by clicking on Upload logo. You can crop the logo and finally click on Save Logo button.
