Assign-Asset

    You can assign assets either from Assign Assets tab or from side navigation panel. Here you can see assign link under action column, click on it, popup will display enter the name, select project, return date, and finally click on Assign button as shown in below.

    If you want to assign assets along with project, first enable toggle switch for “Show Projects While Assigning Assets” from general settings page under settings from side panel.
    Note: you can add projects, from project tab in home page or click on project under administration from side panel, project page will open from where you can add projects.
    If you want to assign assets for location, first enable toggle switch for “Assign Assets to Location” from general settings page under settings from side panel.
    Note

    1. First add locations from location options under administration from side panel, location panel open from where you can add location. Click on Sync location from M365 buttons to sync location from O365.
    2. Location option under administration is used to assign assets to location, user location field is different.

    When you enable above features, you can see changes in assign pop

    If asset accept notification is enable in General setting, user will receive email notification with link from where users can take the action (accept/reject) accordingly.

    The assigned one asset gets shifted from assign asset table to return asset table. When you assign it, assigned date for the asset will bind in assigned date column of home table.
