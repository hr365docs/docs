  • Home Page

      • The Survey 365 Home Page provides a clean, modern, and intuitive interface for creating and managing surveys within SharePoint. It enables administrators to easily configure and monitor surveys, collect user responses, and manage participation in real time. The layout is designed for simplicity, with an organized table and toolbar that enhance user efficiency.


      Key features include:

      Add Survey: Click the + Survey button to create a new survey. This opens the “Create a Survey” panel, where you can define the survey’s details such as title, question, options, and active period.



      Configure Survey Settings: After creating your survey, manage participant permissions and appearance using the toggles below:


      • Allow users to change the submitted answer: Enables participants to update their response after submission.


      • Allow users to see the responses of the survey: Lets participants view survey results after submission.


      • Active: Toggle to make the survey live for participants.


      • Background Image: Upload a custom background image for the survey interface.


      Action Toolbar: Located at the top-right corner of the web part, the toolbar provides quick access to key features:

      • View Toggle: Switch between List View and Card View layouts.

      • Settings: Access global configuration options such as theme preferences and admin management.

      • Help: Open the Help panel to view license details, product version, and support links.


      The Home Page acts as the central hub where administrators can view, edit, or delete surveys, track user participation, and manage survey visibility. Its streamlined layout ensures a seamless experience for both survey creators and end users.
